# 1. 这篇文章给谁看

写给已经读过前两篇 [八大超市地图](/life/us-grocery-stores) 和 [亚洲超市食材对照表](/life/us-asian-grocery) 的留学生 / 学者——**前两篇是知识地图，这一篇是行动手册**。

地图懂了不等于会走——**关键是把不同超市拼成自己的采购节奏**。读完这一篇你会知道：

- **每月去几次超市 / 每次买什么 / 怎么排周历**——按你的身份（单身 / 情侣 / 家庭 / 健身党）有具体配置
- **5 招省钱实战**——weekly ad / coupons / cashback apps / generic 替代 / sale cycle
- **4 种配送服务**对比——Instacart / Walmart+ / Whole Foods+Prime / Target Drive Up，哪个适合你
- **信用卡 + 超市返现的进阶玩法**——月伙食 $400 通过 5%-6% cashback 卡多省 $20-$30

这是**美国超市三部曲的收尾**——三篇连起来你以后采购**每月省 $150-$200**，从此不为吃饭头疼。

# 2. 结论先行

**核心策略一句话**：**一周不超过 1 次大型采购**——Costco 月 1 次大宗 + 亚超每 2-3 周 + 美超 / Aldi 每周补——**每周不超过 2 次进店**，省时间也省冲动消费。

**省钱叠加效应**（按月伙食 $400 估算）：

| 招数 | 月省 |
|---|---|
| 去亚超买中餐食材 | $150-$200 |
| Generic 替代 name brand | $30-$50 |
| Weekly ad + 主食特价时囤 | $20-$40 |
| Coupons + Cashback apps | $15-$30 |
| 5% Cashback 卡（如 Discover It / Chase Freedom）| $20-$30 |
| **合计**：从 $600 月预算降到 | **$300-$350** |

**月省 $250-$300 = 年省 $3K-$3.6K** —— 这笔钱可以加一次回国机票、或买一台 MacBook、或多攒几次旅行基金。

下面是 7 种典型场景的具体配置：

## 3.1 单身宿舍党（不开车 + 食堂为主）

**画像**：本科生 / 研一住宿舍 + 校园食堂为主 + 想偶尔自己做点东西。

**采购配置**：

| 频率 | 去哪 | 一次买什么 | 每次预算 |
|---|---|---|---|
| 月 2 次 | **校园 / 校外 Walmart 步行可达** | 零食 + 速冻饺子 + 调料 + 简易调味料 | $30-$50 |
| 偶尔 | **亚超 Weee! 在线** | 老干妈 + 速冻饺子 + 中式零食 | $40-$60 / 单 |
| 偶尔 | **CVS / Walgreens** | 急用洗漱 / 药品 | $15-$30 |
| **月度合计** | | | **$120-$200** |

**关键点**：
- **不办 Costco 会员**——没车没法囤
- **食堂为主、自己做为辅**——不要追求“完整厨房”
- **Weee! 是宿舍党的救命**——可以送到校园 mailroom
- **微波炉 + 电饭煲**就够大多数中餐基础（水煮饺子 / 蒸米饭 / 微波炉热菜）

## 3.2 单身公寓党（开车 + 自己做饭）

**画像**：研究生 / OPT 工作族 / PhD 住公寓 + 有车 + 一周做饭 4-5 顿。

**采购配置**：

| 频率 | 去哪 | 一次买什么 | 每次预算 |
|---|---|---|---|
| 月 1-2 次 | **Costco**（合租 Executive 与朋友共享）| 大宗肉类 + 鸡蛋 + 牛奶 + 米 + 卫生纸 + Kirkland 红酒 | $100-$150 |
| 每 2 周 | **亚超**（H Mart / 99 Ranch） | 调料 + 干货 + 中式蔬菜 + 五花肉 + 速冻饺子 | $50-$80 |
| 周 1 次 | **Walmart 或 Aldi** | 牛奶补 + 水果 + 偶尔生鲜补缺 | $30-$50 |
| 月 1 次 | **Trader Joe's** | 零食 + 冷冻 meals + 周末 brunch 食材 | $30-$50 |
| **月度合计** | | | **$280-$400** |

**关键点**：
- **Costco 合作账户**：和 1-2 个朋友合用 Executive Membership 平摊会员费
- **冰柜设备**：投资一台小冰柜（$200，IKEA 或 Walmart）—— **大幅扩容**囤货能力
- **每周 30-40 分钟 meal prep**：周日做 4-5 顿菜放冷藏 / 冷冻 —— **省周中时间**

## 3.3 两人小家 / 情侣 / 同居

**画像**：两人住公寓 + 都做饭 + 月预算 $500-700。

**采购配置**：

| 频率 | 去哪 | 一次买什么 | 每次预算 |
|---|---|---|---|
| 月 1 次 | **Costco** | 大宗肉类 + 蛋奶 + 米 + 大瓶调味料 + Kirkland 食材 | $150-$200 |
| 每 2 周 | **亚超** | 中餐食材完整套 | $80-$120 |
| 周 1 次 | **Walmart 或本地 Kroger 系** | 生鲜补 + 周末特价品 | $40-$60 |
| 月 1-2 次 | **Trader Joe's** | 零食 + 简餐 + 招待朋友食材 | $40-$60 |
| **月度合计** | | | **$430-$580** |

**关键点**：
- **Costco 升 Executive**：两人月消费 ~$200 + 加油 ≈ 年 $3,000+，Executive 2% 返现回本
- **协调买菜清单**：用共享 Notion / Google Keep 列表，避免重复买
- **周末“双采”**：周六上午一起去 Costco，周日上午一起去亚超——半天搞定一周

## 3.4 重度中餐家庭（两人或全家 + 完全中餐）

**画像**：每天三顿都是中餐 + 月预算 $600-1000 + 经常请朋友来吃饭。

|---|---|---|---|

| 月 1 次 | **Costco** | 大宗肉蛋奶 + 米 + Kirkland 调味料 | $150-$200 |
| 周 1 次 | **Walmart** | 美式杂项（牛奶 / 卷纸 / 洗漱）| $30-$50 |
| 偶尔 | **Whole Foods 365** | 高品质海鲜 / 调料 / 招待朋友食材 | $50-$100 |
| **月度合计** | | | **$650-$900** |

**关键点**：
- **亚超是主战场**：60% 以上预算 + 每周必去
- **囤主食 + 调料**：50 lb 大米 / 整箱啤酒 / 整桶老干妈
- **周末请朋友**：“Asian American friends 聚餐”已经成为美国留学生圈最重要的社交场景，**这部分支出不要省**——文化和友情的投资

# 4. 5 招省钱实战

下面五招按“投入时间 vs 省钱效应”性价比排序。**第 1 招省最多**：

## 4.1 Generic 替代 name brand（无脑省 30-40%）

这是**最便宜的省钱招数**——**0 时间投入、月省 $30-$50**：

**80% 的“基础品”用 generic 替代 name brand 几乎无差**：

| Name brand | Generic 替代 | 价差 |
|---|---|---|
| **牛奶**（Horizon 等）| Walmart Great Value / Costco Kirkland | 30-50% |
| **鸡蛋**（Eggland's Best）| Walmart / Aldi 普通鸡蛋 | 30-40% |
| **燕麦**（Quaker）| Walmart / Costco Kirkland | 40-50% |
| **大米**（Mahatma）| Walmart / 亚超 Three Ladies | 30% |
| **维他命**（Centrum）| Costco Kirkland / Equate | 40-60% |
| **阿司匹林 / Tylenol** | Walmart Equate / CVS Health | 50-60% |
| **卫生纸 / 厨房纸** | Costco Kirkland / Walmart Great Value | 30-50% |
| **洗衣液 / 洗洁精** | Costco Kirkland | 30-40% |

**例外**（generic 不如 name brand）：
- **可口可乐 / 百事**——口味差异明显
- **巧克力 / 高端零食**（Lindt / Ghirardelli）—— generic 通常较差
- **咖啡 / 红酒**——风味决定一切
- **过敏药**（Zyrtec / Claritin）——多数 generic 一样但部分人对辅料敏感

## 4.2 Weekly Ad + 主食特价时囤货（月省 $20-$40）

每个超市有**周一 / 周三更新的 weekly ad**——上面是该周特价商品。**主食 / 调味料 / 罐头**特价时**整箱囤**：

**典型 sale cycle（什么时候买什么便宜）**：

| 时段 | 什么便宜 |
|---|---|
| **1 月**（New Year）| 健康食品 / 减肥餐 / 蛋白粉 |
| **2 月**（Super Bowl）| 零食 / 啤酒 / 鸡翅 / pizza |
| **3-4 月**（春季）| 烘焙原料 / 鸡蛋 |
| **5 月**（Memorial Day / BBQ 季）| 牛肉 / 猪肉 / 烧烤酱 / 调味料 |
| **7 月**（July 4th）| 同上，BBQ 食材最便宜 |
| **9-10 月**（秋季 / Halloween）| 苹果 / 南瓜 / 糖果 |
| **11 月**（Thanksgiving）| 火鸡 / 烘焙原料 / 罐头蔬菜 |
| **12 月**（Christmas）| 烘焙 / 高端食材 / 礼盒 |

**实战**：**Memorial Day / July 4th 是囤肉的最佳时机**——Costco 牛肉 / 猪肉常打 20-30% 折扣，囤一冰柜够吃 3 个月。

## 4.3 Coupons（实体 + 数字，月省 $15-$30）

**Coupons 文化在美国仍然活跃**——但比 10 年前精简很多：

- **Sunday Newspaper Inserts**（Smart Source / RetailMeNot）—— 实体 coupons，每周日报纸夹页
- **Manufacturer Coupons**：品牌官网 / app
- **App-based**：Walmart / Target / Kroger / CVS / Walgreens 都有 in-app coupons
- **Honey / Rakuten**：浏览器插件 + 在线 coupons

**最值得用的 3 个 app**：

1. **Target Circle**（**免费**）：app 内 personalized 自动 clip coupons + 5% 返现部分商品
2. **Kroger Plus / Safeway Just for U**（**免费**）：weekly digital coupons + fuel points
3. **CVS ExtraCare**（**免费**）：药品 + 化妆品的“$5 off $20" coupons

**不推荐**：
- **过度 couponing**（小红书上”我用 coupons 把 $300 购物买成 $30"）—— 投入时间 / 收益比低，**多数美国人都不这么玩**

## 4.4 Cashback Apps（被动累积，月省 $10-$20）

**装一个 cashback app 在手机里**——购物时拍 receipt 上传 / 在 app 内打勾 promos——**累积现金返还**：

| App | 主打 | 怎么用 |
|---|---|---|
| **Ibotta** | 杂货返现 | 购物前在 app 中“激活”特定商品，购物后拍 receipt 上传，自动累积 |
| **Fetch Rewards** | 杂货 / 万能返现 | 拍任何 receipt，累积 points 兑 gift card |
| **Rakuten** | 在线购物返现 | 浏览器插件 + app，在 Walmart / Target / Whole Foods 在线购物时自动激活 |
| **Upside** | 加油 + 餐厅 | 加油站特定商家几 cent / gal 返现 |

**实战建议**：
- **Fetch + Ibotta** 装着——每次购物 30 秒拍 receipt
- 月累积 $5-$20 现金 = 一年 $60-$240
- **不要为了 cashback 买你不需要的东西**（这是 cashback app 的最大陷阱）

## 4.5 信用卡 5% Cashback 卡（月省 $20-$30）

**这是最后一招但收益不容小觑**——一张专门针对 grocery 的高返现卡：

| 卡 | 杂货返现 | 年费 |
|---|---|---|
| **Discover It Cash Back** | 5%（季度旋转，含 grocery 季度）| 免年费 |
| **Chase Freedom Flex** | 5%（季度旋转）| 免年费 |
| **Citi Custom Cash** | 5%（自动选你消费最多分类）| 免年费 |
| **Amex Gold** | 4%（超市 + 餐厅，每年最多 $25K 消费）| $325（含 $120 餐饮 credit）|
| **Amex Blue Cash Preferred** | 6%（超市，每年最多 $6K）| $95 |

**实战**：
- **月伙食 $400 × 5% = $20/月** —— 全年 $240 cashback
- **季度旋转 5% 卡**：每季度激活，避免 5% 上限
- **Amex BCP 6%**：年伙食 > $5K 的家庭，回本年费 $95 后还净赚 $250+

# 5. 配送服务全对比

不开车 / 工作日忙的留学生——配送是省时间救命方案。下面是 4 大主流 + 1 个亚超线上：

## 5.1 Instacart（基本所有超市）

- **覆盖**：Costco / Walmart / Target / Whole Foods / Kroger 系 / Aldi / Publix / 多数亚超 / CVS / Walgreens
- **价格**：商品价 + service fee（5-10%）+ delivery fee（$3-$8）+ 小费（建议 15-20%）
- **会员 Instacart+**：$99/年 / $9.99/月——**$35+ 单免运费** + 5% 部分商品 cashback + 降低 service fee
- **典型加价**：原价 $100 的购物，Instacart 总账单 $125-$135
- **优势**：**一个 app 能下任何超市**——最大灵活性
- **劣势**：**最贵**——加价 25-35%

**适合**：偶尔需要 / 不想自己配多个 app / 临时 / 雨天

## 5.2 Walmart+

- **会员**：$98/年 或 $12.95/月
- **覆盖**：所有 Walmart 门店
- **关键福利**：
    - **$35+ 单免费配送**（unlimited，不是 same-day 也不收 fee）
    - **加油折扣 $0.10/gal**（部分加油站）
    - **Paramount+ 流媒体含**
- **劣势**：仅 Walmart——不能配 Costco / 亚超 / Whole Foods
- **vs Instacart**：Walmart+ 配送的是 Walmart 真人配送 / 自动 + Walmart Pay 价格——通常**更便宜 + 商品价不加价**

**适合**：Walmart 是你主要采购来源 + 一年配送 > 8 次 = 回本

## 5.3 Whole Foods + Amazon Prime

- **会员**：Prime $139/年（同时享 Whole Foods 福利）
- **关键福利**：
    - **$35+ 单免费 same-day delivery**（部分城市）
    - **Whole Foods 在店额外 5-10% 折扣**（部分商品）
    - **365 by Whole Foods Market** 自有品牌打折最大
- **劣势**：仅 Whole Foods + Amazon Fresh
- **价值**：**Prime 会员本身就值 $139**（送货 + Prime Video + Music），Whole Foods 是 free 加成

**适合**：已经是 Prime 会员 + 喜欢有机 + 在 Whole Foods 服务区

## 5.4 Target Drive Up + Shipt

- **Target Drive Up**：免费——**app 下单 + 开车到门店 + 5 分钟员工放后备箱**——这是**最适合留学生**的“半配送”方案
- **Target Circle 360 ($99/年)**：免费 same-day delivery + 5% 部分返现
- **Shipt**：Target 旗下，类似 Instacart，但仅 Target

**适合**：Target 是你常去的 + 不想进店 / 不想等配送 + 想要 Target 调性 + 有车

## 5.5 亚超线上：Weee! / Yamibuy / 各家本地配送

- **Weee!（湾区总部，全美主要城市）**：
    - **$35+ 免运**
    - 多数地区**次日达 + 冷链**
    - 中餐食材最全
    - 偶尔促销 $5-$10 off 给新用户
- **Yamibuy**：
    - 中国零食 + 调料 + 美妆
    - **$69 免运**（USPS）/ $99（FedEx）
    - 全美 1-3 天到货
- **本地亚超 H Mart / 99 Ranch 在自家 app**：
    - 部分大店有 same-day delivery
    - 用 Instacart 整合居多

**适合**：所在地没有线下亚超 / 想买中餐零食 / 偶尔送大宗到家

## 5.6 哪种适合你？

| 你的情况 | 推荐 |
|---|---|
| **不开车 / 不想出门 / 配多种超市** | **Instacart** + Instacart+ 会员（$99/年）|
| **Walmart 是主要采购 + 经常 8+ 次/月** | **Walmart+** ($98/年）|
| **已经是 Prime 会员 + 喜欢 Whole Foods** | 无需额外，自然 Prime + Whole Foods |
| **常去 Target + 想要“开车 5 分钟取”** | **Target Drive Up**（免费）+ 偶尔 Circle 360 |
| **没有线下亚超** | **Weee! + Yamibuy** 必装 |
| **多人 / 家庭 / 重度采购** | **多个会员组合** —— Costco + Walmart+ + Prime |

# 7. 几个进阶玩法

## 7.1 Costco Executive 升级时机

- 单身 / 偶尔去：Gold $65 即可
- 月消费 > $270 / 季度 > $810：**升 Executive $130，2% 返现回本**
- Executive 还有：**额外免费 Roadside Assistance + Executive Member 专属 promo**
- **怎么升级**：会员卡当天去服务台，秒升

## 7.2 Costco Hot Dog Combo

**$1.50 终身不涨价**的 Costco hot dog（一个 hot dog + 22oz 饮料） —— 留学生省钱党的传统：周末去 Costco 大宗 + 顺便吃个 hot dog 当午饭。**这不是“穷”，是 Costco 文化**。

## 7.3 信用卡 + 超市 + Cashback App 三层叠加

最高 cashback 路径：

```
1. Discover It Cash Back（季度 5% on grocery）= 5%
2. + Ibotta promo on specific item = +$0.50-$2/item
3. + Walmart+ 加油折扣 = +$0.10/gal
4. + Costco Executive 2% = 2% on Costco
========================================
合计有效 cashback 5-8% 不算夸张
```

## 7.4 季度 / 半年大宗囤货

**Costco 卫生纸 / 洗衣液 / 大米 / 食用油 / 调料**——这些**保质期长**的东西**每季度大宗囤一次**：

- 一次买够 3 个月 / 6 个月用量
- **避开“急买”**——急买价高
- 配合 **Costco app 的“Member Savings”**——每月有的物品打折时囤

## 7.5 周日下午的“价格降临”

- **Whole Foods / Trader Joe's / Wegmans 在周日下午经常降价处理快到期商品**
- 烘焙 / 熟食 / 海鲜 / cheese 大幅打折
- **周日 3-5 PM 到店“赶降价”**——是高品质党的甜区

## 7.6 节日大促预算规划

**主要超市 promo 节日**：

- **Black Friday**（11 月最后一周五）：电器 / 家居为主 + 部分 grocery
- **Christmas Eve / 当周**：烘焙 / 高端食材便宜
- **New Year**：保健品大降价
- **Memorial Day / July 4th / Labor Day**：肉 / BBQ 食材
- **Thanksgiving**：火鸡 + 烘焙

**实战**：每月查一次“接下来的 promo”——根据 promo 提前调整采购清单。
