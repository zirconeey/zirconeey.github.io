# 1. 这篇文章给谁看

写给三类人：

- **刚到美国第一周要开银行账户**——选 Chase 还是 BoA？要 SSN 吗？怎么把国内汇过来的钱存进去？
- **已经有了 checking 账户但 savings 利率才 0.01%**——错失高息储蓄账户（HYSA）的 4-5%
- **要从美国汇钱给中国父母** / 父母汇美国——**Wise 和 Western Union 哪个划算**？

中国银行：**任何柜台 / app 全国通用 + 一个支付宝走天下**。
美国银行：**checking + savings 分开 + 几十家银行各有强项 + 跨境汇款单独考虑 + Zelle / Venmo 各有用法**。

这篇文章把“开户 → 选银行 → 高息 savings → 汇款 → 投资 → 毕业 / 回国处理”全流程讲清楚。**主线以 Penn State 留学生为例**——刚到 State College 第一周开 Chase 账户、研究生津贴 direct deposit、月底转一部分到 HYSA、暑假实习把钱存银行投资——这是典型留学生的银行使用路径。

> ⚠️ 银行政策 / 利率 / 开户奖金会**频繁变动**。文中价格 / 利率以 [Doctor of Credit](https://www.doctorofcredit.com/) / 各银行官网最新为准。

# 2. 结论先行

**留学生最优组合**（“3 张卡走天下”）：

| 角色 | 推荐 | 月费 | 利率 |
|---|---|---|---|
| **主 Checking**（日常）| **Chase College Checking** | $0（学生免）| 0% |
| **HYSA**（存钱省钱）| **Marcus by Goldman** 或 **Ally** | $0 | **4-5%** |
| **国际 ATM**（出国 / 回国）| **Schwab Investor Checking** | $0 | 0%，**全球 ATM 免费 + 退手续费** |

**月伙食 $400 + 月房租 $1,500 + RA 工资 $2,500 的典型 Penn State PhD**——三张卡可以**省掉所有月费 + 多挣 $50-$100 / 月利息**。

**3 个最大坑**：

1. **没 SSN / Itin 时去 Chase 开户被拒**——选错银行会跑空趟
2. **保持 $0 利率 savings**——错失年息 $200-$1,000
3. **用 Western Union 汇款**——比 Wise 多收 5-10% 手续费

# 3. 美国 5 大类银行

## 3.1 大型连锁银行（Chase / BoA / Citi / Wells Fargo）

**特点**：

- 全美**几千家网点 + 几万台 ATM**——不愁找
- **大学城基本都有**（Penn State 的 State College 有 Chase / Wells / PNC 等）
- 服务标准 + 手机 app 好
- **月费高**：通常 $12-$15（满足条件可 waive）
- 利率低：savings 通常 0.01%（鸡肋）

**主要银行简介**：

- **Chase**：最大 + UR 积分体系强 + 免费 Zelle / 数字银行体验最好
- **Bank of America**：覆盖大 + 信用卡组合好
- **Citi**：国际化 + 服务全球用户
- **Wells Fargo**：传统老牌 + 中部强

**适合**：第一年留学生用作主 checking + 偶尔 walk-in 存现金 + ATM 网络。

## 3.2 区域银行（PNC / US Bank / Fifth Third / TD Bank 等）

**特点**：

- 覆盖**特定地区**（PNC 主要在中部 + 东岸；US Bank 主要在西部）
- 服务质量与大型连锁类似
- 部分有特殊学生福利

**State College / Penn State 区域**：**PNC** 是主流——本地存在感强 + 校园 ATM 多。

**适合**：所在地区主要银行 + 已经有大型 checking 的人开个 backup。

## 3.3 网上银行（Schwab / Ally / SoFi / Marcus / Discover）

**特点**：

- **没实体网点**（或网点很少）
- **HYSA 利率高**（4-5%）
- **月费 $0**
- 移动 app 优秀
- ATM 网络通过卡的 logo 找（多数走 Allpoint / MoneyPass）

**主要网上银行**：

- **Charles Schwab Investor Checking**：**全球 ATM fee 退**——出国神器
- **Ally Bank**：HYSA + checking 一站，体验好
- **SoFi**：HYSA 4%+ + checking 0.5%
- **Marcus by Goldman Sachs**：纯 HYSA，利率高
- **Discover**：HYSA + 信用卡组合

**适合**：**第二张账户**（不是第一张）——已经有大型连锁 checking 的人开网银存高息。

## 3.4 信用社（Credit Union）

**特点**：

- 非盈利组织 + member-owned
- 利率通常**比商业银行好**（loan 便宜 + savings 利息高）
- 客户服务通常更亲切
- 多数有“membership requirement”（特定职业 / 地区 / 学校）

**Penn State Federal Credit Union**：PSU 学生 / 教职工可以加入——大学城 ATM 多 + 利率比 Chase 好。

**适合**：所在城市 / 学校有特定 credit union 的人。

## 3.5 留学生友好银行

**几家“专门接受国际学生”**的银行：

- **HSBC Premier**：全球银行，从国内 HSBC Premier 可以“全球转移到美国”——有钱家庭的最佳选择
- **Citi International**：类似
- **Bank of China USA**：纽约 / LA 有分行——熟悉中国背景客户
- **East West Bank**：加州为主——华人圈

**适合**：**国内已有银行账户**（HSBC Premier / Citi）+ 想“无缝转过来”。**普通留学生用 Chase / BoA 即可**。

# 4. 4 种账户类型

## 4.1 Checking Account（支票账户）

**日常使用账户** —— 收入打这里 + 付账单 + 刷 debit card。

- **月费**：$0-$15（通常 minimum balance 或 direct deposit 可 waive）
- **利率**：通常 0%（除部分 high-yield checking 0.25-0.5%）
- **特征**：可写支票 / debit card / Zelle / mobile deposit

**留学生主要用**：所有日常支出 + 工资 direct deposit + 房租支付。

## 4.2 Savings Account（储蓄账户）

**存钱账户** —— 不日常动 + 利息高于 checking。

- **传统 savings**（Chase / BoA）：利率 0.01%（**几乎是 0**）
- **HYSA**（High-Yield Savings Account）：利率 **4-5%**（2026 年）
- **限制**：每月最多 6 次 withdrawal（联邦 Reg D，部分银行已废）

**关键**：**所有不日常用的钱放 HYSA** —— 不是传统 savings。

## 4.3 Money Market Account（货币市场账户）

**Savings + checking 的混合**：

- 利率**接近 HYSA**
- 可写支票（少数次数 / 月）
- 通常需要更高 minimum balance（$2,500+）

**实战**：**多数留学生不需要**——HYSA + checking 已经够用。

## 4.4 Certificate of Deposit (CD，定期存款)

**锁定存款一段时间换更高利率**：

- 3 个月 / 6 个月 / 1 年 / 5 年期
- 利率通常**比 HYSA 高 0.5-1%**
- **提前取出有罚款**（损失部分利息）

**实战**：

- 你**确定 1 年内不用**的钱 → CD
- 灵活 / 短期不确定 → HYSA
- 多数留学生用 HYSA 就够（CD 略高的利率不值得失去流动性）

# 5. 留学生开户 6 步（以 Penn State + Chase 为例）

## 步骤 1：选银行

**Penn State 留学生第一年推荐 Chase College Checking**：

- State College 有分行（W. College Ave）
- 学生免月费
- ATM 全美最多
- **开户奖金 $200-$300**（Chase 经常推 promo）

## 步骤 2：在线 / 现场预约

- **网上提交 application**（[Chase.com](https://www.chase.com/personal/college-banking)）—— 30 秒
- 选 Chase College Checking
- **预约现场 appointment** 或直接 walk-in（State College 那家通常 walk-in 即可）

## 步骤 3：准备文件

带上：

- **护照**
- **I-20 / DS-2019**
- **SSN 卡**（如有）/ **SSA letter**（没 SSN 的）
- **学校录取信** / **Penn State ID**
- **美国地址证明**（lease / utility bill / 学校宿舍证明）

## 步骤 4：到银行 + 开户

- 银行员工帮你 open account（30-60 分钟）
- 选 debit card 设计（Chase 给几种）
- 设 4 位 PIN
- 设 username / password 网银

## 步骤 5：First Deposit

- 现金 / 国内电汇过来的钱 / 学校工资 direct deposit
- **最低开户金额**：Chase $25 / BoA $25
- **没 minimum balance** for Chase College Checking

## 步骤 6：等正式 debit card + Checks

- **临时 debit card**：当场拿（直接能用）
- **正式 debit card**：7-10 天邮寄到家
- **支票本**（要的话）：单独申请，$15-$25 / 一沓

# 6. 没 SSN 怎么开户

**很多刚到美国的 F-1 学生**还没拿到 SSN（要拿到 on-campus job offer 才能申请）。

## 6.1 接受护照 + 学校 ID 的银行

| 银行 | 接受文件 | 备注 |
|---|---|---|
| **Chase** | 护照 + 学校 ID + 地址证明 + ITIN（部分情况）| 多数情况接受，不要 SSN |
| **Bank of America** | 类似 | |
| **Citi** | 类似 | 国际背景客户友好 |
| **Wells Fargo** | 类似 | |
| **HSBC** | 全球账户 transfer | 已有 HSBC 国内账户首选 |
| **Schwab** | **必须 SSN / ITIN** | ❌ 没 SSN 不能开 |
| **Ally / SoFi / Marcus** | **必须 SSN / ITIN** | ❌ 同上 |

**关键**：**网上银行多数要求 SSN**——刚到美国第一周开**大型连锁实体银行更稳**。

## 6.2 后续拿到 SSN 后怎么办

- 把 SSN 加到现有账户：
    1. 打银行客服 / 走进分行
    2. 提交 SSN 卡复印
    3. 银行**链接 SSN 到账户**——你的信用记录 / 报税开始关联
- **重要**：**SSN 链接到账户后才会发 1099-INT**（年度利息税表）

# 8. HYSA 高息储蓄账户对比

**这是留学生最被低估的省钱机会**。把不日常用的钱从 Chase savings（0.01%）转到 HYSA（4-5%）—— **每月多挣的利息抵几顿外卖**。

## 8.1 主流 HYSA 利率（2026 年）

| HYSA | 利率（APY）| 最低金额 | 强项 |
|---|---|---|---|
| **Marcus by Goldman Sachs** | 4.4% | $0 | 老字号 + 稳定 + 简单 |
| **Ally Bank Online Savings** | 4.2% | $0 | UI 好 + 同公司 checking 体验顺 |
| **SoFi Savings** | 4.0%+ | $0 | + checking 一体 + 加保险更高 |
| **Discover Online Savings** | 4.0% | $0 | 信用卡用户额外 promo |
| **Capital One 360 Performance** | 3.9% | $0 | + 实体银行混合 |
| **CIT Bank Platinum** | 4.5% | $5,000 | 高余额最高利率 |

## 8.2 怎么用

1. **打开 HYSA app + 注册**（Marcus / Ally / SoFi）—— 5 分钟
2. **link 到你 Chase / BoA 主 checking**——通过 routing + account number
3. **每月把“不日常用”的钱 transfer 到 HYSA**——比如工资 $2,500 中的 $1,500 转过去
4. **每月底自动 interest 入账**

**算账**：留学生平均存款 $5,000-$15,000——HYSA 4% 年利息 = $200-$600 / 年——**完全免费的小额收入**。

## 8.3 HYSA vs CD vs Money Market

| 维度 | HYSA | CD | Money Market |
|---|---|---|---|
| 利率 | 4-5% | 4-5%+ | 4-5% |
| 流动性 | **任何时间取** | 锁定 3-12 月 | 中等 |
| 最低金额 | $0 | $500-$1,000 | $2,500+ |
| 推荐 | **多数留学生用这个** | 1 年不用的钱 | 进阶用户 |

# 9. 中美汇款（最实用的一节）

**留学生 / 工作族常需要**：父母汇美国（学费 / 生活费）/ 自己汇回国。

## 9.1 Wise（推荐）

**[Wise.com](https://wise.com/)**（前 TransferWise）—— **业内最便宜 + 最透明**：

- **手续费**：0.5-1.5%（透明显示）
- **汇率**：mid-market 真实汇率（无加价）
- **速度**：1-3 天
- **支持**：USD ↔ CNY ↔ EUR ↔ 50+ 货币
- **限额**：单笔 $50K（high-volume verify 后更高）

**举例**：父母从中国汇 $5,000 美元来 → Wise 总成本 ~$50（1%）→ 1-2 天到你 Chase 账户。

**vs 银行电汇**：传统银行电汇 $30-$50 fee + 不透明汇率（隐藏 2-3% 加价）= $150-$200 总成本。**Wise 省 70%**。

## 9.2 Remitly（中国汇款专门）

**[Remitly.com](https://www.remitly.com/)** —— 专做中国 / 印度 / 拉美的汇款：

- 手续费：0.5-2%
- 速度：**Express 5 分钟到** / Economy 1-3 天
- 支持人民币 / 微信 / 支付宝直接收款
- 限额：单日 / 单月有限制

**适合**：紧急汇款回国（5 分钟到家人微信）。

## 9.3 Western Union / MoneyGram

**传统国际汇款** —— 现金 / 网上都行：

- 手续费：高（**3-10%**）
- 速度：分钟级 / 当日
- 全球门店多

**实战**：**多数情况下不推荐**——Wise / Remitly 便宜很多。**唯一例外**：汇到没银行的偏远地区 / 紧急现金需求。

## 9.4 父母从国内汇美国

**国内外汇限额**：

- **个人年度结汇 $50,000 / 人**——这是中国国家外汇管理局规定
- 父母帮你汇 = 用爸爸 + 妈妈各 $50K = 总 $100K / 年
- **超过 $50K 需要“特别申请”**——通常需要在职证明 / 学费证明 / I-20

**实战路径**：

1. **Wise** 是父母最简单的选择——国内绑卡 + 网上转账 + 1-3 天到你 Chase
2. **传统银行电汇** ICBC / BOC → Chase—— 慢 + 贵但稳
3. **找能用的换汇渠道**——非正规但常见，**有风险**

## 9.5 你汇回国

- 留学生工作 / 实习存钱 → 汇父母 / 自己国内账户
- **Wise 是首选**——便宜、透明、快
- 限额：Wise 一般 $50K / 笔，年度无限制

## 9.6 跨境用 Zelle / Venmo / Cash App **不行**

- **Zelle**：仅美国境内银行
- **Venmo / Cash App**：仅美国境内
- **跨境必用 Wise / Remitly / Western Union / 银行电汇**

- **PayPal**：在线购物 + eBay

# 11. ATM 网络与 fees

## 11.1 In-Network 免费

每家银行有自己的 ATM 网络：

- **Chase ATM**：全美最多（16,000+）
- **Bank of America ATM**：~16,000
- **Allpoint Network**（多数网上银行用）：55,000+
- **MoneyPass**（信用社用）：32,000+

**用自家 / 自家网络 ATM = 免费**。

## 11.2 Out-of-Network ATM Fee

- **银行 fee**：$2.50-$3.50（你的银行收）
- **ATM owner fee**：$2-$5（ATM 所在银行收）
- **总单次成本**：$5-$8

**实战**：尽量用自家 ATM——大学城 / 大城市基本都有几家。

## 11.3 国际 ATM Fee

普通银行 + 国外 ATM = **手续费 + 外汇 fee 总 5-7%**。

**Schwab Investor Checking 的“必杀技”**：

- **全球任何 ATM 免费**（Schwab 自己不收）
- **ATM owner 收的 fee 月底全额退**
- **0% foreign transaction fee**

**留学生回国 / 出国旅行 = 永远带一张 Schwab card**——这一项就值得开账户。

# 12. 投资账户入门

**F-1 / OPT 都能开美国投资账户**——这是留学生在美国“被动收入”的起点。

## 12.1 主流 broker

| Broker | 特点 | 适合 |
|---|---|---|
| **Charles Schwab** | 全能 + 客服好 + 免佣 + 投资 + ATM | **首选 + 一站式** |
| **Fidelity** | 类似 Schwab + 老字号 | 替代 Schwab |
| **Vanguard** | Index fund 创始 + 长期投资 | 长期持有党 |
| **Robinhood** | UI 现代 + Crypto + Options | 散户 / 喜欢炒 |
| **E*Trade** | 老牌 + Morgan Stanley 旗下 | 进阶交易者 |

## 12.2 留学生投资基础

**F-1 NRA 投资税务**（前 5 年）：

- **股票资本利得**：通常**免联邦税**（NRA + < 183 天 in US）
- **股票分红**：30% 预扣（中美 tax treaty 降到 10%）
- **银行利息**：通常免联邦税
- **报 1040NR 时要申报**——Sprintax 自动处理

**RA 投资**（第 6 年起）：

- 全部按公民税率算
- Long-term capital gains 0/15/20%
- Short-term 按工资税率

## 12.3 401(k) / Roth IRA（OPT 工作族）

- **401(k)**：雇主提供 + 税前存 + 雇主 match（“免费的钱”）
- **Roth IRA**（年 $7,000）：税后存 + 退休时免税取
- F-1 / OPT 都能开 —— 但**回国后取出有 10% penalty + 联邦 / 州税**

**实战**：

- OPT 工作 → 立刻开 401(k)（即便回国，钱仍在 + 复利）
- Roth IRA：如果确定**长期留美**就开

# 13. 毕业 / 回国时的银行处理

## 13.1 关账户 vs 保留

**保留**（推荐）：

- **大型 checking 保留** —— Chase / BoA 支持中国地址 + 美国手机号继续登录
- 月费豁免条件可能改变 —— **每年检查一次**
- 投资账户保留 + 远程操作

**关账户**：

- **彻底切断关系**时关
- 关之前**取空 + 转账**
- **要 close confirmation letter**——保留作为证据

## 13.2 保留账户的最低余额

- **Chase 大学生免月费**：毕业后变成普通账户，需要 $1,500 minimum balance 或 $500 direct deposit 免月费
- **BoA**：类似
- **Schwab / Ally / SoFi**：**无最低余额**——回国后保留最稳

**留学生策略**：**留 Schwab + Ally**（无 minimum）+ 关 Chase（毕业后月费）。

## 13.3 远程操作

**回中国后能远程做的**：

- **登录网银 / app**：可以
- **转账 / 付账单**：可以
- **mobile deposit 支票**：可以（手机拍照存）
- **ATM 取现**：用 Schwab 卡在中国 ATM 即可

**不能做的**：

- 现金存款（中国没有美国银行 ATM 接受现金）
- 拿新支票 / 支票本——美国地址送达
- 投资账户取出超 $5K（多数 broker 要电话验证）

## 13.4 投资账户特别处理

- **F-1 / OPT 开的 401(k)**：保留——10 年后取出更划算
- **Roth IRA**：保留——一辈子都能用
- **brokerage account (taxable)**：保留 —— 远程操作 OK
- **关键**：**保留账户 + 报美国税**（即便回国）—— 投资分红 / 利息要报

# 14. 几个反直觉常识

## 14.1 Overdraft Fee 是个大坑

- 账户余额 $0 + 你刷卡 / 付账单 → **overdraft**
- 银行**自动垫付** + 收 fee $35
- 一天**多次** overdraft = 多次收 fee（$100+ 一天）

**预防**：

- **关掉 overdraft protection**（多数银行可以选 opt out）
- **link checking 到 savings 自动 transfer**
- **设置 low balance alert**

## 14.2 月费 Waiver 条件

每家银行有“满足条件就免月费”：

- **Chase Total Checking**：$1,500 minimum 或 $500 direct deposit 月免 $12 月费
- **BoA Advantage Plus**：$2,000 minimum 或 $250 direct deposit
- **Wells Fargo Everyday**：$500 minimum 或 $500 direct deposit

**学生账户 / Schwab / Ally**：**无月费 + 无 minimum**。

## 14.3 Mobile Deposit Limits

- 拍支票存银行 = **mobile deposit**
- **每日 / 每月限额**：Chase $5,000 / BoA $5,000
- 大额支票 → 走 walk-in 或 ATM

## 14.4 Safe Deposit Box

- 银行**保险箱**：$50-$300 / 年
- 存放：护照 / 重要文件 / 贵重物品
- **不保险公司保赔**——理论上银行不负责
- **多数留学生用不上**——家里小保险柜更便宜

## 14.5 银行倒闭 FDIC 保 $250K

- **FDIC**（Federal Deposit Insurance Corporation）保所有 checking + savings + CD
- **每家银行每个用户 $250,000**
- 投资账户 SIPC 保 $500,000

**实战**：你存 $300K 在一家银行 → $250K 保 + $50K 风险 → **应分散到 2 家银行**。

## 14.6 信用卡 Cashback ≠ 钱

- 大多数 cashback 卡 1-2% 是常态
- 看似 1-2% 不多 —— 但年消费 $20K = $200-$400
- **配合 HYSA 4% 利率 = 复利效应**
- 详见 [信用卡指南](/life/student-credit-cards-roadmap)
