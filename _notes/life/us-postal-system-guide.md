# 1. 这篇文章给谁看

写给三类人：

- **第一次寄件 / 收件**——刚到美国，要买的东西全靠 Amazon、要寄回中国不知道走哪家
- **包裹被偷 / 丢了**——porch pirate / 公寓 mailroom 拿不到——怎么办？
- **跨境寄件**——从中国寄一些东西过来 / 寄回中国——海关怎么处理？哪些不能寄？

中国快递：**顺丰 / 京东 / 圆通**——价格低 + 当日 / 次日达 + 一个 app 走天下。
美国快递：**USPS / UPS / FedEx / DHL** 四大并行 + 各有强项 + 价格档差很大。

这篇文章把“4 大快递 → 寄信寄包 → 收件 → 跨境 → 防诈骗”全流程讲清楚。文章以 **State College / Penn State 留学生**为典型场景——大学城邮局 + 学校 mailroom + Amazon Locker + 公寓 package room——这些是日常常用的渠道。

> ⚠️ 各快递价格 / 服务每年小变。具体以**官网 + 实时报价**为准。

# 2. 结论先行

**4 大快递选哪个**：

| 场景 | 选哪家 | 原因 |
|---|---|---|
| **国内寄信 / 小包裹** | **USPS** | 最便宜（First Class $0.73 / Priority $10+）|
| **国内寄商业大件 / 跨州** | **UPS / FedEx** | 速度可控 + 商业网络强 |
| **隔日 / 急件** | **FedEx Express / UPS Next Day** | 1-2 day 保证 |
| **国际寄到中国** | **USPS Priority Mail International** 或 **DHL Express** | USPS 便宜（$45-$80）/ DHL 快（3-5 day） |
| **从中国寄到美国** | **顺丰国际 / EMS / 转运公司** | 看物品类型 |

**3 个最大坑**：

1. **包裹被偷 / 丢了不知道怎么追**——多数情况能 refund，但要走流程
2. **跨境寄件没正确报关**——海关扣留 / 退回 / 罚款
3. **USPS 钓鱼短信**——假“重新付邮费 $0.30 否则退回” → 偷信用卡

# 3. 美国 4 大快递对比

## 3.1 USPS（United States Postal Service）

**联邦邮政服务** —— 最便宜 + 最普及。

- **价格**：最便宜
- **服务范围**：**只有 USPS 能投递到 P.O. Box**——这是它的独特地位
- **网点**：全美 31,000+ 邮局
- **强项**：信件 / 小包裹 / Media Mail（书 DVD）/ 国际邮件
- **弱项**：商业 B2B + 跨州大件慢 + 跟踪 OK 但不及 UPS / FedEx

## 3.2 UPS（United Parcel Service）

**B2B 大件物流**最强。

- **价格**：中等
- **服务范围**：UPS Store + Amazon Hub Locker
- **网点**：全美 5,000+ UPS Store
- **强项**：商业大件 + 跨州可控时间 + 跟踪精确
- **弱项**：比 USPS 贵 + 不投递 P.O. Box

## 3.3 FedEx

**速度最快 + 商业**。

- **价格**：贵
- **服务范围**：FedEx Office / FedEx Express / FedEx Ground
- **网点**：FedEx Office 是 Kinkos —— 学生打印 / 复印中心
- **强项**：FedEx Express（隔日 / 2 day）+ 国际 + 高价值 / 时间敏感
- **弱项**：贵 + 不投递 P.O. Box

## 3.4 DHL

**国际为主，美国境内极少**。

- **价格**：国内贵 / 国际竞争力强
- **服务范围**：主打国际
- **网点**：DHL Service Point 不多
- **强项**：**国际 B2B + 中美 / 美中跨境**
- **弱项**：美国境内不常用

## 3.5 选哪家速查表

| 你要寄 | 推荐 |
|---|---|
| **信件 / 明信片** | **USPS First Class** ($0.73) |
| **轻 < 13 oz 小包裹** | **USPS First Class Package** ($5-$10) |
| **2-5 lb 包裹** | **USPS Priority Mail** ($10-$25) |
| **5+ lb 跨州** | **USPS Priority** 或 **UPS Ground** |
| **隔日 / 时间敏感** | **FedEx Express / UPS Next Day** ($30+) |
| **书 / DVD（大量便宜）** | **USPS Media Mail**（极便宜）|
| **国际寄到中国** | **USPS Priority Mail International** 或 **DHL Express** |

# 4. USPS 服务类型

## 4.1 First Class Mail

- **轻量**（< 13 oz / 0.37 kg）
- **价格**：信件 $0.73（一张 Forever Stamp）/ 小包裹 $5-$10
- **速度**：1-5 个工作日
- **跟踪**：信件无 / 包裹有

**适合**：常规信件 / 小包裹 / 普通文件。

## 4.2 Priority Mail

- **2-3 day 全美**
- **价格**：起 $10（Flat Rate Box / Envelope）
- **跟踪 + 包险 $100**（含）
- **Flat Rate Boxes**：**只看体积不看重量**——大件重物用这个最划算

**Flat Rate Box 价格**（2026）：
- Small Flat Rate Box（$10.40）
- Medium Flat Rate Box（$18.40）
- Large Flat Rate Box（$24.50）

**实战**：寄 5 lb 书去外州 → Medium Flat Rate Box $18.40 vs Priority Mail by weight $30+ —— **Flat Rate 省一半**。

## 4.3 Priority Mail Express

- **隔日 / 2 day 保证**
- **价格**：$30-$50+
- **退款保证**——超时退邮费

**适合**：紧急文件 / 高价值物品 / 时间敏感。

## 4.4 Media Mail

- **专为书 / DVD / CD / 出版物**
- **极便宜**：1 lb $4-$5 / 5 lb $5-$8
- **慢**：2-8 个工作日

**适合**：寄学位证 / 教科书 / 长 PDF 打印件 ——只要有“出版”内容。

**注意**：不能用于私人手稿 / 普通文件。

## 4.5 Ground

- **3-7 day 美国境内**
- **价格**：低
- **大件 OK**

**适合**：不急的大件物品。

# 5. 寄信流程

## 5.1 准备

- **信封**：标准 #10 envelope（4-1/8 × 9-1/2 inch）
- **邮票**：Forever Stamp（$0.73 一张）—— **永远涨价但 stamp 永久有效**

## 5.2 写地址（关键格式）

```
Sender Name              ←── 自己（左上）
Sender Address Line 1
City, State ZIP


              Recipient Name             ←── 收件人（中间）
              Recipient Address Line 1
              City, State ZIP

         [Stamp here →]   ←── 右上贴邮票
```

**ZIP code 是 5 位**（如 16802 = State College PA）—— 错一位可能延迟一周。

## 5.3 投递

3 种方式：

- **Mailbox 投递**（家门口或公寓 mailbox）：把信塞进去 → mail carrier 拿走
- **Post Office 柜台**：邮局直接 drop
- **USPS Drop Box**：街道蓝色 mailbox（写“USPS”的）—— **不要投到普通垃圾桶颜色相似的箱子里**

# 6. 寄包裹流程

## 6.1 选 Box / 包装材料

- **USPS Flat Rate Box**：免费拿（邮局 / 在线申请）—— **重物首选**
- **自己的 Box**：任何结实的 box 都行
- **包装材料**：bubble wrap / 报纸 / 防震填充

## 6.2 打包

- 物品塞紧 —— 摇晃不动
- **易碎物品**：**双层 box**（小 box 套大 box，中间填充）
- **液体 / 粉末**：密封 + 单独袋装 + 写“FRAGILE”

## 6.3 称重 + 计费

3 种方式：

### A. 邮局柜台

- 走进 USPS 邮局
- 员工称重 + 选服务
- 直接付款 + 投递

### B. 在线 Click-N-Ship

- [USPS.com Click-N-Ship](https://cns.usps.com/)
- 自己量 + 称重 + 选服务
- 在线打印 label
- **节省 5-15%**（在线 vs 柜台）
- Drop 到任何 USPS 网点 / mailbox

### C. PirateShip / Stamps.com（高 volume）

- 第三方服务
- **更大折扣**（20-30%）
- 适合 eBay 卖家 / 经常寄件的人

## 6.4 投递

- 到 USPS 邮局 drop
- 或在家 schedule pickup（USPS / FedEx / UPS 都有）
- 或 drop 到 USPS drop box（小包裹）

# 7. 收件 + 跟踪

## 7.1 Tracking Number

每个包裹有 tracking number：

- USPS：22-digit
- UPS：1Z + 16 chars
- FedEx：12-digit

**跟踪**：

- USPS: [tools.usps.com/go/TrackConfirmAction](https://tools.usps.com/go/TrackConfirmAction)
- UPS: [ups.com/track](https://www.ups.com/track)
- FedEx: [fedex.com/fedextrack](https://www.fedex.com/fedextrack)
- **整合**：[Parcel App](https://parcelapp.net/) iOS app 一键追踪所有快递

## 7.2 Informed Delivery（USPS 免费）

**[Informed Delivery](https://informeddelivery.usps.com/)** —— 注册后 USPS **每天早上 email 给你**当天将到的所有信件预览图：

- **看清是不是垃圾邮件**——节省时间
- **重要信件提前知道**（账单 / 政府信件）
- **包裹也给预览**

**所有美国住址都可以注册**——免费 + 强烈推荐。

## 7.3 Porch Pirate 防范

**美国包裹被偷率**：每年 ~14% 美国家庭至少一次 porch pirate。

**预防**：

- **送到学校 / 公司 mailroom**——不送家
- **送到 Amazon Locker / Hub**——免费 + 安全
- **送到 UPS Store / FedEx Office**——hold for pickup
- **设置安全摄像头**（Ring / Nest）
- **要求 signature required**（额外费用 $5-$10）

# 8. 公寓 mailroom / Amazon Locker

## 8.1 公寓 Mailroom

大型公寓集团（Greystar 等）：

- **专门 mailroom**——所有包裹送这里
- **取件方式**：app 通知 / 实体钥匙 / locker code
- **大件 oversized**：通常放 manager office

**State College 公寓**多数是这样——比单户家庭安全。

## 8.2 Amazon Hub Locker / Amazon Counter

- **Amazon Hub Locker**：放在 Whole Foods / 7-Eleven / 其它合作点
- 下单时选 Locker → 收 6 位 code → 输入打开柜子
- **免费**——只需要去取
- **安全**——不会被偷
- **限**：包裹尺寸限制（适合中小件）

**Amazon Counter**：在合作店铺（CVS / Whole Foods）现场 hold —— 类似 Locker 但人工取。

## 8.3 学校 Mailroom（Penn State 例）

**Penn State Mailroom**（在 HUB 大楼）：

- 学生申请 mailroom 编号
- 地址写：“[Your Name] / [Mailroom Number] / Penn State University / [Building] / State College, PA 16802"
- **多数学校第一年提供给国际新生**
- 大件包裹：通常 mailroom 通知后取

**地址格式**（PSU 学生）：

```
Zhang San
HUB-Robeson Center
123 Pollock Rd, Box 1234
University Park, PA 16802
```

**优点**：**安全 + 不会被偷 + 不限投递时间**——特别适合刚到美国第一周。

# 9. 包裹丢失 / 被偷怎么办

## 9.1 USPS Missing Mail Search

- **包裹超过 7 day 没到** → [USPS Missing Mail Search](https://www.usps.com/help/missing-mail.htm)
- 提交 search request
- USPS 内部查找
- **找到的概率 ~30-40%**

## 9.2 Amazon 包裹丢失

- Amazon **几乎 100% 退款**
- 进 Order → ”Get help“ → ”Item not received“
- 当天 / 次日 refund
- **多数情况下 Amazon 比 carrier 更愿意 refund**

## 9.3 信用卡 Chargeback

- 卖家发了但你没收到 + 卖家不退
- 用信用卡 chargeback —— 信用卡公司退你钱 + 找卖家追讨
- **发现 60 天内**报告

## 9.4 报警（高价值）

- 物品 > $500 → 可以报警
- 警察会出 police report
- **作为 chargeback / insurance claim 的证据**
- **回收概率 < 5%**——但官方记录有用

# 10. Forward Mail（搬家时）

## 10.1 USPS Change of Address

**[USPS Change of Address](https://moversguide.usps.com/)** —— 把所有 mail 转到新地址：

- **价格**：$1.10 信用卡验证
- **覆盖**：6 个月（有些地方 12 个月）
- **不转**：包裹（除非 pre-shipment status）/ 部分政府信件

## 10.2 Update 各处地址

不光 USPS —— 还要给：

- 银行 / 信用卡
- IRS / Tax authority
- DMV / 驾照地址
- 学校 / 雇主
- Amazon / 在线购物
- 各家保险

**清单工具**：[MailMoveAlerts](https://www.usps.com/manage/mail-after-move.htm)

## 10.3 Old Address 收信怎么办

- USPS forward 6 个月——之后退回 sender
- 让前房东 / 朋友帮忙转交（如熟悉）
- 重要邮件提前**让 sender 寄到新地址**

# 11. 国际寄件：美国 → 中国

## 11.1 选哪家

| 服务 | 速度 | 价格（2 lb 包裹）| 跟踪 |
|---|---|---|---|
| **USPS Priority Mail International** | 6-10 day | **$45-$60** | 良好 |
| **USPS Priority Mail Express International** | 3-5 day | $80-$120 | 优秀 |
| **DHL Express International** | 3-5 day | $80-$150 | 最优 |
| **FedEx International Priority** | 3-5 day | $100-$180 | 优秀 |
| **UPS Worldwide Express** | 3-5 day | $100-$180 | 优秀 |

**实战**：

- **不急 / 普通物品** → USPS Priority International（最划算）
- **急 / 商业 / 高价值** → DHL Express

## 11.2 海关申报（CN22 / CN23）

寄到中国 / 任何国家**必须填海关申报表**：

- **CN22**（小包，价值 < $400）：贴在包裹上
- **CN23**（大包，价值 ≥ $400）：附 + 多份

填写：

- **Detailed description**：”Cotton T-shirt“（不要写”Gift“）
- **Value**（按真实价值）—— 申报低 vs 实际：海关查到罚款 + 没收
- **HS Code**（商品分类）
- **Sender / Recipient info**

**USPS 在线 Click-N-Ship 自动生成**——填一次就行。

## 11.3 关税

**寄到中国**：收件人付关税：

- **个人物品价值 ≤ ¥1,000（~$140）**：免税
- **超过 ¥1,000**：按品类 13-50% 关税
- **食品 / 化妆品 / 奢侈品**：高关税

**热门物品关税参考**（中国海关）：

| 品类 | 关税 |
|---|---|
| 服装 | 20-30% |
| 鞋类 | 25% |
| 化妆品 | 50%+ |
| 婴儿用品 | 10-13% |
| 食品 | 看具体（部分免）|
| 电子（手机 / 笔电）| 13-20% |

## 11.4 不能寄什么

**绝对禁运**：

- ❌ 锂电池 / 充电宝（**普遍禁运**）
- ❌ 香水 / 含酒精化妆品
- ❌ 含麻黄药品
- ❌ 武器 / 弹药 / 军用
- ❌ 易燃液体（除少数）
- ❌ 烟酒 / 大麻 / 管制药

**限制**（要申报）：

- 食品（看具体清单）
- 处方药
- 现金 / 银行卡
- 古董 / 文物

# 12. 国际寄件：中国 → 美国

## 12.1 选哪家

| 服务 | 速度 | 限制 | 推荐 |
|---|---|---|---|
| **EMS（中国邮政国际特快）** | 7-15 day | 多数物品 OK | 普通物品 |
| **顺丰国际** | 5-10 day | 限制比 EMS 严 | 普通物品 |
| **DHL（中国发）** | 4-8 day | 商业件强 | 商业 / 急件 |
| **转运公司**（顺心 / 帆海等）| 7-30 day | 海运 / 集运便宜 | 大件 / 大量 |

## 12.2 食品 / 中药能寄吗

**多数情况下不能寄到美国**：

- ❌ **任何肉类**——包括方便面调料、月饼里有肉
- ❌ **新鲜水果蔬菜**
- ❌ **大量种子 / 中药材**
- ❌ **奶制品**（限制）
- ⚠️ **包装食品**——少量可以 + 密封商业包装

**详见**：[FDA 食品入境规则](https://www.fda.gov/) + [USDA 食品入境](https://www.aphis.usda.gov/)

## 12.3 美国海关 + $800 免税

- **个人物品 < $800**：免关税
- **> $800**：按品类 0-30% 关税
- **商业用途货物**：另算

## 12.4 转运公司 vs 直接寄

**转运公司**（先发到美国国内仓库再 forward）：

- ✅ 慢但便宜（海运 1-2 月 / 空运 1-2 周）
- ✅ 大件 / 大量物品省钱
- ❌ 需要先注册转运地址
- 推荐：**顺心 / 帆海 / 海鸥淘**

**直接寄**（顺丰国际 / EMS）：

- ✅ 快
- ❌ 贵
- 适合：紧急 / 小件

# 13. 邮政诈骗警惕

## 13.1 USPS 钓鱼短信（2024-2025 年最常见）

**典型话术**：

> ”USPS: Your package cannot be delivered due to incomplete address. Please confirm at https://usps-confirm[.]xyz/“

**实质**：钓鱼网站 → 偷信用卡 / 个人信息。

**识别**：

- USPS 真正的网址：**usps.com**——不是 usps-xxx[.]xyz
- USPS **不会通过短信要你”重新付邮费 $0.30"**
- 真正的 missing package → 走 USPS Missing Mail Search

## 13.2 假 Amazon 退款骗局

- “你的 Amazon 订单 $X 取消了，点这退款”
- 链接到假 Amazon 网站
- 输入信用卡 → 被偷

**识别**：

- Amazon 不会通过短信发链接
- 退款直接走 Amazon app

## 13.3 假 IRS / Bank 邮件

- 类似套路
- IRS 不会发邮件 / 短信——**只发信件**
- 银行不会通过短信要你“立刻验证”

**通用规则**：**不要点短信里的链接** —— 真有事走官方 app / 网站。

# 14. 几个反直觉常识

## 14.1 USPS 周日不投递（除 Amazon 包裹）

- USPS 周日**不投递信件 / 普通包裹**
- **唯一例外**：Amazon 包裹——USPS 和 Amazon 协议每周日投递

## 14.2 节日延迟

- **Thanksgiving / Christmas 前后 1 周**：所有快递延迟 2-5 day
- **大雪天**（PA / NY / 中部）：延迟更多
- 12 月 18 日后寄不一定圣诞前到——**提前发**

## 14.3 邮票永远涨价（Forever Stamp）

- USPS 邮票 1995 = $0.32
- 2026 = $0.73
- **每 1-2 年涨 $0.05-$0.10**

**Forever Stamp 的妙用**：

- 写“Forever”的邮票永久有效——**今天买，10 年后用还是 first class**
- 听到要涨价的消息**前买 100 张**——锁定低价

## 14.4 P.O. Box 优势

**P.O. Box**（邮局信箱）：

- $40-$200 / 年（看尺寸 + 城市）
- **防 porch pirate**
- **不在家也能收**
- **永久地址**——搬家不影响

**适合**：经常出差 / 不想透露家庭地址 / 隐私党。

## 14.5 节日邮票 / 纪念邮票

USPS 经常发**主题邮票**（Star Wars / Disney / 历史人物）：

- 同价格的 first class
- 写信 / 寄包裹用同样有效
- 收藏价值（如果不用）

## 14.6 USPS Money Order

- USPS 发的“无条件支付凭证”
- 类似支票但**预付 + 安全**
- $0.01-$1,000 / 张
- $1.65-$2.40 fee
- 适合：付房租给个人房东（不接 personal check）/ 给陌生人付款

**美国邮政系统**和中国不一样——4 大快递并行 + USPS 是公共服务（联邦邮政）+ 商业 UPS / FedEx 各有强项。

留学生第一年用得多的是：**Amazon 包裹（USPS / UPS / FedEx 都可能）+ Amazon Locker（防被偷）+ 学校 mailroom（最安全）+ 偶尔寄回中国（USPS Priority International）**。

读完这一篇你应该知道：

- **第一周**到 USPS 注册 Informed Delivery + 让 Amazon 选 Hub Locker
- **第一次寄件**用 USPS Click-N-Ship 在线打 label 省 5-15%
- **第一次跨境寄**用 USPS Priority Mail International + 正确填 CN22
- **遇到诈骗短信** —— 不点链接 / 走官方网站

留学攻略系列写到这里——**16 + 5 = 21 篇**——已经覆盖了“中国背景留学生在美国第一年到工作”几乎所有的“信息盲区”。希望读完这套系列你能少踩 80% 的坑、多省 20-30% 的预算。
