# 1. 这篇文章给谁看

写给刚到美国第一周面对一堆超市不知道去哪的留学生 / 学者：

- **Costco / Walmart / Target / Whole Foods / Trader Joe's / Aldi**——名字都听过，但每家定位是什么、卖什么、价格档位差多少？
- 朋友说“我每周去 Costco 囤货 + Trader Joe's 买零食 + 亚超买中餐食材”——这套组合是怎么决定的？
- 我每月伙食费 $400 总觉得不够用，是因为没去对超市吗？

这是**美国超市三部曲的第一篇**——侧重**认识地图 + 各家定位**。后两篇分别是：

- **第二篇**：[亚洲超市完全指南](/life/us-asian-grocery)——中餐食材性价比对照表
- **第三篇**：[美国超市采购实战](/life/us-grocery-tactics)——场景化配置 + 省钱招数 + 配送

读完第一篇你会知道**这片“超市丛林”长什么样、每片树林的特点、自己应该去哪些**——之后两篇再讲怎么“高效打猎”。

# 2. 结论先行

**留学生最优组合**（4-5 家轮流去）：

| 角色 | 推荐 | 频率 | 一次大约 |
|---|---|---|---|
| **囤货主力** | Costco 或 Sam's Club | 每月 1-2 次 | $150-$250 |
| **日常补充** | Walmart 或 Aldi | 每周 1 次 | $40-$80 |
| **中餐食材** | 亚洲超市（H Mart / 99 Ranch / 大华）| 每 2-3 周 | $50-$100 |
| **零食 / 简餐** | Trader Joe's | 每月 1-2 次 | $40-$80 |
| **应急 / 高端** | Whole Foods / Kroger 系 | 偶尔 | 看情况 |

**月伙食预算速查**：

| 人数 + 习惯 | 月预算 |
|---|---|
| 单身 + 食堂 / 外卖为主 | $150-$250 |
| 单身 + 自己做饭 | $250-$400 |
| 两人 + 自己做饭 | $500-$800 |
| 四口之家 | $1,000-$1,500 |

# 4. 主流超市详解

## 4.1 Costco / Sam's Club（仓储会员制）

**核心模式**：会员制大包装零售。**单价比普通超市低 20-40%**，但需要付年费 + 有车 + 有空间存货。

### Costco

- **会员费**：Gold $65/年（基础）/ Executive $130/年（额外 2% 返现，**年消费 > $3,250 时回本**）
- **门店**：全美 ~600 家 + 加拿大 / 国际
- **自有品牌**：**Kirkland Signature**——业内公认品质极高，常常是 name brand 的 OEM 代工产品
- **招牌单品**：
    - **$1.50 Hot Dog Combo**（含一个大热狗 + 22oz 饮料，**40 年没涨过价**）
    - **$4.99 Rotisserie Chicken**（3 lb 整鸡）
    - **Kirkland 香草冰淇淋 / Kirkland 红酒 / Kirkland 杂粮**
- **品类强项**：肉蛋奶（保鲜期长 + 大包装）、轮胎、汽油（比外面便宜 $0.20-$0.30/gal）、眼镜部、药房、旅行预订、家电
- **不强项**：调味料（中餐调料几乎没有）、极少 SKU 的小众食物

**留学生该不该办？**

- ✅ **办**：有车 + 室友 / 配偶分摊 + 家里有冰柜或冰箱大 + 会做饭
- ❌ **不办**：没车（不带车去 Costco 不现实）+ 单身住宿舍 + 储物空间小 + 经常出差不在家
- 💡 **聪明的留学生玩法**：**和 2-3 个室友 / 朋友合办一张 Executive Membership**——买大包装回来分。**法律上 Costco 不允许“member 给非 member 进店”**，但实务上 Costco 不查（特别是带朋友进去）

**几个反直觉的“Costco 文化”**：

- **出门查收据**：每个 Costco 出门必须出示购物收据 + 商品袋——**不是怀疑你偷**，是收银员防止漏扫的复检流程
- **退货政策极宽松**：**几乎无理由退**——开过一半的食物可以退、用了 6 个月的电器可以退（除少数电子产品）。这是 Costco 的核心客户粘性
- **只收 Visa 信用卡**：MC / Discover / Amex 信用卡都不收（详见 [支付指南](/life/us-payment-methods)）
- **样品试吃**：周末店里大量免费试吃——很多人下午饿了去 Costco “蹭饭”，**这是 Costco 的官方营销**，不丢人

### Sam's Club

- 沃尔玛旗下的仓储会员
- **Club $50 / Plus $110**
- 全美 ~600 家
- **Member's Mark** 自有品牌
- 总体：**Costco 的廉价版**——价格略低、品质略低、装修简陋、但更接受多种支付方式（含 Apple Pay）

**Sam's Club vs Costco 怎么选？**

- 你所在地区有 Costco 的话——**优先 Costco**（Kirkland 品质 + 退货政策 + 加油站 + 社区氛围）
- 只有 Sam's Club 的地区（Walmart 总部所在地附近）——**Sam's Club 也行**

## 4.2 Walmart / Target（大众市场二选一）

这两家是美国非会员超市的“两大支柱”。**Walmart 价格更低、Target 调性更好**。

### Walmart

- **全美最大零售商**：4,600+ 门店，年销售额 $640 billion
- **价格定位**：**最便宜**的非折扣超市（仅 Aldi 比它便宜）
- **品类**：食品 + 杂货 + 家居 + 服装 + 药品 + 家电——**几乎什么都卖**
- **自有品牌**：
    - **Great Value**（食品类）—— 留学生省钱党的必选
    - **Equate**（健康用品 / 药品 / 化妆品）
    - **Mainstays**（家居）
    - **Marketside**（fresh produce / deli 部分）
- **招牌操作**：
    - **Walmart+ 会员**$98/年：免费 $35+ 配送 + 加油折扣 + Paramount+ Plus 流媒体
    - **Walmart Pay**：app 内支付（[详见支付指南](/life/us-payment-methods)）
    - **Pickup**：online 下单，门店 curbside 取，**免费**

**留学生使用建议**：

- 日常杂货首选——**单价低 + 品类全**
- **可以网上下单 + curbside pickup**——不用进店、不用排队
- **不接受 Apple Pay**——出门必带实体卡或 Walmart app
- 大宗买的话，比 Costco 单价高一些，但**不要会员费 + 不要 bulk**

### Target

- 全美 ~2,000 门店
- **价格定位**：比 Walmart 贵 5-15%，但**装修、设计、调性高一档**
- **品类**：食品 + 家居 + 美妆 + 服装 + 玩具——**美感更强**
- **自有品牌**（Target 的设计感强项）：
    - **Good & Gather**（食品）—— 包装设计精美、品质中上
    - **Up & Up**（健康用品）
    - **Threshold / Project 62**（家居）
    - **Cat & Jack**（童装）
    - **Goodfellow & Co.**（男装）
    - **A New Day**（女装）
- **招牌操作**：
    - **Target Circle**（**免费会员**）：自动 personalized 折扣
    - **Target Circle 360 $99/年**：免费 same-day delivery（Shipt）+ 5% 返现
    - **Drive Up**：app 下单，开车到门店外，员工把货放进后备箱——**5 分钟完成**，留学生最爱

**Walmart vs Target 怎么选？**

- **省钱党 + 不在意装修**：Walmart
- **设计感党 + 愿意为体验多花 10%**：Target
- **多数人选择**：附近哪家更近 / 哪家停车更方便就去哪家

## 4.3 Kroger 系（区域性传统超市）

这一类是**美国“老牌传统超市”**——有 deli、肉摊、烘焙、海鲜、新鲜 produce，价格中等，看人熟人多（这种“buyer 是邻居”的氛围在大城市仍存在）。**美国不同地区有不同的“主导品牌”**：

| 地区 | 主导品牌 |
|---|---|
| 全美（最广）| **Kroger**（Cincinnati 总部，~2,800 家）|
| **加州** | Ralphs / Vons（Albertsons 旗下）/ Safeway |
| **德州** | **HEB**（德州人骄傲）/ Central Market |
| **东北部**（NY / NJ / MA）| Stop & Shop / ShopRite / **Wegmans**（最受好评）|
| **中西部** | Meijer / Hy-Vee |
| **南部**（FL / GA / SC）| **Publix**（南部人最爱）|
| **西北部** | Fred Meyer / Albertsons |

**为什么这一类值得提**：

- **生鲜品质比 Walmart 好** —— 蔬菜更新鲜、肉类切割更精细
- **deli 区可以现场买切片冷餐肉 / 奶酪 / 烤鸡** —— Walmart 没这个
- **会员系统不要钱**：Kroger Plus / Safeway Just for U / Publix Club —— **app 内自动 clip 数字 coupon + 累积加油 fuel points**
- **fuel points**（独特好处）：每消费 $1 拿 1 fuel point，攒够后**在 Kroger / Safeway 加油站打折 $0.10-$1.00 / gal**（相当于 1-3% cashback）

**Wegmans 单独提一笔**：东北部传统超市，**美国零售业公认最佳超市连锁**——员工待遇好、生鲜部超棒、热食区像高档餐厅自助、店内有座位区可坐着吃。如果你在它的服务区域，**值得专门去逛一次**。

**留学生使用建议**：

- 日常补充 + 想吃比 Walmart 新鲜的生鲜——选这里
- **Fuel points** 长期累积有意义，注册 Kroger / Safeway 免费会员
- 周末 weekly ad 上有大量 BOGO（Buy One Get One）—— **囤罐头 / 调味料的好时机**

## 4.4 Whole Foods / Trader Joe's（高端 + 特色双雄）

这两家**都是单店 SKU 少、品牌精选、调性高**——但定位完全不同。

### Trader Joe's（特色高性价比）

- **门店**：全美 ~580 家
- **核心特点**：
    - **极简 SKU**：单店 ~4,000 SKU（普通超市 30,000-50,000）—— 选项少 = 不用纠结
    - **80% 是自有品牌**——价格中等，品质高
    - 价格其实**不贵**——许多商品比 Walmart / Whole Foods 都便宜
- **招牌商品**（跟“Trader Joe's”几乎是同义词）：
    - **Mandarin Orange Chicken**（橙子鸡冷冻款）—— 中国胃也喜欢
    - **Cookie Butter**（speculoos 饼干酱）—— 烤面包神器
    - **Everything But the Bagel Seasoning**——万能调料
    - **Frozen Meals 系列**（印度咖喱 / 意面 / 中式炒饭等）
    - **冷冻饺子 / 冷冻馄饨 / 冷冻蒸饺**——一袋 $4，够吃 2 顿
    - **Mini Brie / 各种 cheese**——cheese 党的天堂
    - **$3 Buck Chuck 红酒**（Charles Shaw）—— 加州独家便宜红酒
- **店内文化**：员工穿 Hawaiian shirt + 极友好、店内放老歌、**免费试喝咖啡 / 试吃**——氛围比正常超市轻松
- **不强项**：fresh produce 不全、肉类有限、传统美式生活必需品（垃圾袋 / 洗衣粉 / 大米）有但选项少

**留学生使用建议**：

- **单身 / 不会做饭 / 想试新东西的人** —— Trader Joe's 是**隐藏甜区**
- 月去 1-2 次，每次 $40-$80，囤一周的零食 + 几顿冷冻 meals
- 周末早上 / 平日下午去——**周末下午人最多排长队**

### Whole Foods（有机精品）

- **门店**：全美 ~530 家（被 Amazon 2017 年收购）
- **核心特点**：
    - **主打有机 organic + 高品质 + 注重 sourcing**
    - 价格最贵——常被戏称 “Whole Paycheck”（“整月薪水”）
    - **Amazon Prime 会员有 5-10% 折扣 + 部分商品额外折扣**——这是 Prime 在超市的最大用处
- **自有品牌**：**365 by Whole Foods Market**——**其实价格不贵**，是性价比之选
- **品类强项**：
    - 有机蔬菜水果（远好于 Walmart 的有机线）
    - 高品质肉类（grass-fed beef / pasture-raised chicken）
    - Local / artisanal 烘焙 / 奶酪 / 红酒
    - 健身餐 / vegan / gluten-free 选项最全
- **店内体验**：装修比 Whole Foods 高一档（地板光亮、陈列像艺术展）、热食吧台 / 沙拉吧台 / pizza 现场做、**自带座位区**

**留学生使用建议**：

- **不日常去**——价格让多数留学生肉痛
- **特定场合去**：约会 / 朋友聚餐买好材料 / 心情不好需要小确幸
- **365 自有品牌不贵**：买基础品的话，Whole Foods 365 价格 = Walmart Great Value
- **Prime 会员 + Whole Foods**：部分商品 Prime 价格其实接近 Trader Joe's

## 4.5 Aldi / Lidl（欧洲折扣双雄）

这两家是**德国连锁**——把德国的“极简折扣模式”搬到美国。**比 Walmart 还便宜 10-15%**，但需要适应几个反直觉操作。

### Aldi

- **门店**：全美 ~2,400 家（增长非常快，**全美第三大食品零售商**）
- **价格**：**最便宜的主流超市**（仅次于个别仓储 / 大批发）
- **核心特点**：
    - 单店 ~1,400 SKU（**比 Trader Joe's 还少**）
    - 90% 是自有品牌（Aldi 几十个子品牌）
    - 装修极简——纸箱直接放在货架上没拆
    - 收银员**坐着扫码**，速度比普通超市快 2-3 倍
- **几个反直觉的“Aldi 操作”**：
    - **购物车要投 25 美分硬币锁**——用完归还退回。**这是为了让你自己把购物车归位**，省人工
    - **不提供购物袋**——自带或现场买。多数人**带帆布袋来 + 现场打包**（柜台之外有打包桌）
    - **不接受 manufacturer coupon**——所有 promo 已经在标价里
    - **付款选项有限**：debit / credit card / Apple Pay / 现金——**不接受支票 / WIC**（部分店 EBT）
- **招牌**：
    - **Aldi Finds**（每周限量特色商品）——德国进口巧克力 / 红酒 / 手工奶酪
    - **烘焙**——欧式面包品质比 Walmart 好
    - **生鲜**——价格惊人地低，但选择有限

**留学生使用建议**：

- ✅ **省钱党必选**——单身月省 $50-$100 完全可能
- ✅ **进店 15 分钟搞定**——店小 + SKU 少
- ❌ 不要期待“大牌”商品——多是 Aldi 自有品牌（你需要试一次品牌后建立信任）
- 💡 **Aldi 的鸡蛋 / 牛奶 / 黄油**——多次盲测里和 name brand 没区别但价格只一半

### Lidl

- **门店**：~180 家（主要东北：VA / MD / NJ / PA / NY）
- **总体**：和 Aldi 类似，但 SKU 略多 + 烘焙更突出
- 美国市场份额还小，多数地区没有

## 4.6 亚洲超市（简介）

亚洲超市是中国留学生的“必去清单”——主流品牌包括：

| 品牌 | 主战场 |
|---|---|
| **H Mart** | **韩国系，全美**（NJ 总部，已开 ~100 家）|
| **99 Ranch** | **西海岸 + 德州 + 东北部分**（华人系）|
| **大华超市（Tawa Supermarket）** | **西海岸**（华人系）|
| **Great Wall** | **东北 + 中部** |
| **Fubonn** | **西北**（Portland）|

**价格优势**：中餐 / 韩餐 / 日餐 / 越南餐**专门进口** + **量大** + **批发链短** = 中餐食材远便宜于美式超市。比如：

- 老抽：亚超 $3-$4 一瓶 vs Walmart Pearl River Bridge $7-$8
- 干香菇：亚超 $5/lb vs Whole Foods $25/lb
- 五花肉：亚超 $4-$5/lb vs Walmart 不卖（要买 pork belly skin-on，多数地区找不到）
- 各种亚洲蔬菜（小白菜 / 茼蒿 / 韭菜花）：亚超有 vs 多数美超不卖

**这是中国留学生第一年最大的“省钱发现”**——把中餐食材都搬到亚超买，每月伙食费可省 $100-$200。

# 6. 几个反直觉的“美国超市常识”

## 6.1 Costco 出门要查收据

Costco / Sam's Club 出门时**所有人都要出示收据 + 商品袋**——员工拿笔在收据上画一个标记表示已查。

**这不是怀疑你偷东西**，是 Costco 的**漏扫复检**——收银员忙的时候可能漏扫一两个商品，门口复查保护商家利益。**所有人都查，不针对你**。

**正确姿势**：把收据拿在手里 + 推车经过门口时把收据递给员工。员工扫一眼商品 + 数大件商品数量是否对得上 + 在收据上画一个 √ —— **5 秒完成**。

## 6.2 多数 produce 按 each 卖

中国超市的菜按斤卖（按重量），美国超市**多数 produce 按 each 卖**：

- 一颗 broccoli：$2.49 ea
- 一束 banana：$0.59 / lb（这种是按 lb 的）
- 一颗 avocado：$1.49 ea
- 一颗 lettuce：$2.99 ea

**结果**：买 lemon / lime / apple / banana 之类**按 lb 卖的合理**；买 broccoli / lettuce / avocado 按 each **小颗反而吃亏**——挑最大颗的。

## 6.3 Generic brand 经常和 name brand 是同一工厂

**最大的“省钱秘密”**：

- **Great Value 牛奶** = Kroger 牛奶 = 你家附近某家奶厂代工的——**装瓶不一样而已**
- **Kirkland Signature 维他命** = Nature Made / Pharmavite 代工
- **365 by Whole Foods 燕麦** = 多家 organic 农场轮替供货
- **Equate 阿司匹林** = Bayer / Pfizer 代工

**结论**：80% 的“基础品”（牛奶 / 鸡蛋 / 大米 / 卫生纸 / 阿司匹林 / 维他命 / 洗衣液）—— **buy generic 是无脑赢**。

**例外**：调味料 / 巧克力 / 咖啡 / 红酒——name brand 和 generic 差距大，**这些值得买大牌**。

## 6.4 退货政策极其宽松

美国大型连锁的退货政策**比中国宽松得多**：

| 超市 | 退货时限 | 几个特殊点 |
|---|---|---|
| **Costco** | **几乎无理由 / 无时限** | 电子产品 90 天；食物开过一半也能退 |
| **Walmart** | 90 天 | 有 receipt 几乎必退；无 receipt 用 Walmart credit |
| **Target** | 90 天 | 婴儿用品 / 服装 365 天 |
| **Whole Foods** | 30 天 | 不满意可退（包括开过的食物）|
| **Trader Joe's** | “Anytime, no questions asked” | 包括开过吃了一半的零食 |
| **Aldi** | 60 天 + double guarantee | 不满意退钱 + 再送一个免费 |

**实战**：买了不喜欢的食物 / 用了不满意的产品 / 衣服尺寸不对——**直接退**，不要硬吃下去。这是美国超市的核心客户体验。

## 6.5 收银员问 “Did you find everything you were looking for?”

每次结账，收银员都会问。**这是话术**——不是真的关心你找到什么。**正确回答**：“Yes” + 微笑 = 30 秒完事。

**例外**：你**真的没找到**某件东西，可以说“No, I couldn't find X”——收银员会**叫人帮你找**或者**告诉你卖完了 / 调到了某区域**。这时候是真的好用。

## 6.6 Self-checkout 普及但商家正在缩减

美国大型连锁过去 5 年大规模上了 self-checkout（自助结账机），但 2023-2024 年起**部分连锁开始缩减**——原因：盗窃率上升 + 员工调度反而成本更高。

**实战**：

- **Walmart / Target self-checkout 仍是主流**——便宜 / 高效
- **Costco 大件不允许 self-checkout** —— 必须真人收银员
- **Kroger 系开始重新启用真人收银**——但人少时仍走 self-checkout

## 6.7 鸡蛋 5 种分级（多数是营销）

美国鸡蛋常见标签：

| 标签 | 含义 | 价格档 | 实际营养差异 |
|---|---|---|---|
| **Conventional**（常规）| 笼养，最常见 | 最便宜 | 基础 |
| **Cage-Free** | 不关笼但仍室内 | +30% | **几乎无差** |
| **Free-Range** | 有户外活动空间 | +50% | 略好（omega-3 略高）|
| **Pasture-Raised** | 户外放养 + 自由觅食 | +100% | **真正的好蛋**（蛋黄橙色、营养密度高）|
| **Organic**（有机）| 喂有机饲料 | +50-80% | 不必然好（仍可能笼养）|

**结论**：

- **省钱党**：买 conventional / cage-free 都行，区别小
- **真正在意的**：买 **pasture-raised** —— 这是唯一有营养差距的
- **organic 标签是营销陷阱**——不必然代表好养殖

## 6.8 牛奶有效期通常超过 2 周

美国超市的牛奶 **use-by date** 通常是**包装日 + 21 天**（Costco / Whole Foods）或 +14 天（普通超市）。

- **Use-by date** != 必须丢——多数情况**过期 5-7 天仍可饮用**（自己闻 / 尝判断）
- **Lactose-free / Ultra-Pasteurized** 牛奶有效期更长（30-60 天）
- **Plant-based**（杏仁奶 / 燕麦奶）保质期最长 90 天+

## 6.9 牛肉 3 级（Prime / Choice / Select）

USDA 牛肉分级 3 档：

| 等级 | 占市场 | 价格 | 哪里有 |
|---|---|---|---|
| **USDA Prime** | 顶级 ~3% | 最贵 | Costco / Whole Foods / 高端超市 |
| **USDA Choice** | 主流 ~50% | 中等 | 所有超市的主力 |
| **USDA Select** | 较瘦 ~30% | 最便宜 | Walmart / Aldi 多数 |

**实战**：日常做菜用 **Choice** 即可；想吃 steak / 烤肉 / 招待朋友 → **Costco Prime**（性价比远好于 Whole Foods）。
