# 1. 问题

超市的油货架上能看到二三十种——大豆油、菜籽油、玉米油、葵花籽油、橄榄油、花生油、芝麻油、椰子油、牛油果油、亚麻籽油……每种都有人鼓吹“健康”。**它们到底有什么区别？什么菜该用什么油？家里需要备几瓶？**

# 2. 结论先行

挑油本质上看两个维度：

- **烟点（smoke point）**：决定能加热到多高。低于烟点煎炒安全，超过烟点油会冒烟、热分解、产生有害物质并发苦
- **风味强度**：从中性到浓郁，决定它是“调味的”还是“做饭的”

两个维度组合出 4 个**生态位**：

| | 中性风味 | 浓郁风味 |
|---|---|---|
| **高烟点** | 通用主炒油（菜籽、大豆、玉米、葵花、米糠、牛油果） | 高温调香（精炼花生油） |
| **低烟点** | 凉拌中性（罕见） | 凉拌 / 增香（EVOO、烤芝麻油、亚麻籽油、黄油、椰子油） |

**家庭最小备油集合（3 瓶）**：

1. **1 瓶通用主炒油**（菜籽 / 大豆 / 米糠都行，约 $5-10/L）
2. **1 瓶 EVOO**（凉拌、面包蘸、地中海菜）
3. **1 瓶烤芝麻油**（中餐调香，几滴起神效）

90% 的家常菜这三瓶就够了。其它油按需补——做西式烘焙加黄油，做煎牛排加牛油果油，做油炸（少做）加精炼花生油。

# 3. 科学原理

## 3.1 烟点：决定一切的温度

油加热到一定温度会开始冒烟——这就是**烟点**。超过烟点意味着脂肪分子开始**热分解**：

- 产生丙烯醛（acrolein）等刺激性化合物，呛人
- 释放游离脂肪酸，让油发苦发酸
- 多次过烟点的油会产生反式脂肪和氧化产物，长期摄入有害

所以选油的第一原则：**烟点要明显高于你的烹饪温度**。常见烹饪方法对应的温度：

| 烹饪方法 | 温度 | 需要烟点 ≥ |
|---|---|---|
| 凉拌、面包蘸、油醋汁 | 室温 | 任何 |
| 低温煮、慢炖 | < 100 ℃ | 任何 |
| 中火煎（牛排、鱼） | 150-180 ℃ | 200 ℃ |
| 大火爆炒 | 200-230 ℃ | 230 ℃ |
| 油炸 | 170-190 ℃ | 220 ℃ |
| 空气炸 | 180-200 ℃ | 220 ℃ |

**经验法则**：烟点比烹饪温度高 30 ℃ 以上是安全裕度。

## 3.3 精炼 vs 初榨：同一种油两种性格

很多油有两个版本：

- **初榨 / 冷压（cold pressed / extra virgin / unrefined）**：保留风味、营养、抗氧化物；烟点低
- **精炼（refined）**：高烟点、中性味道；维生素 E 等部分被破坏，但更适合高温

举例：

| 油 | 初榨版本 | 精炼版本 |
|---|---|---|
| 橄榄油 | EVOO，烟点 200 ℃，浓果香 | 精炼橄榄油（“light olive oil”），240 ℃，几乎无味 |
| 椰子油 | extra virgin coconut，177 ℃，浓椰香 | refined coconut，204 ℃，无椰香 |
| 芝麻油 | 烤芝麻油（toasted），177 ℃，浓芝麻香 | 生芝麻油（plain），210 ℃，淡香 |
| 花生油 | unrefined，160 ℃，浓花生香 | refined，227 ℃，淡花生味 |

**关键判断**：标签上写 “extra virgin” / “cold pressed” / “unrefined” / “first press” 都是初榨；写 “refined” / “pure” / “light” 都是精炼。

EVOO 的 “extra virgin” 是法定标签，必须**冷压、酸度 ≤ 0.8%、无添加剂**。买便宜的 “olive oil” 或 “pure olive oil” 大概率是精炼或混合，没有 EVOO 的健康光环。

## 3.4 饱和、单不饱和、多不饱和：健康选择

脂肪酸的化学结构决定它的物理性质和健康影响：

- **饱和脂肪（SFA）**：碳链上没有双键，分子排列紧密 → 室温**固态**或半固态（黄油、猪油、椰子油、棕榈油）。**最稳定**、不易氧化。但摄入过多与心血管病关联（虽然 2010 年后这个共识有所松动）
- **单不饱和（MUFA）**：1 个双键，室温**液态**（橄榄油、菜籽油、牛油果油、花生油）。**健康学界最推荐**——大量研究关联到地中海饮食的心血管获益
- **多不饱和（PUFA）**：≥ 2 个双键（葵花籽、玉米、大豆、亚麻籽、深海鱼）。包含必需脂肪酸 omega-3 和 omega-6。**最不稳定**，最易酸败，加热也最易产生氧化产物

现代饮食常见问题是 **omega-6 ：omega-3 比例失衡**——理想 ~4:1，现实 ~15-20:1（因为我们吃太多大豆 / 玉米 / 葵花籽油，太少深海鱼 / 亚麻籽）。这是为什么营养学家会建议**少量加亚麻籽油 / 紫苏油凉拌**，或多吃深海鱼。

# 4. 实践建议

## 4.1 家庭备油的最小集合（3 瓶）

90% 家常菜用这三瓶搞定：

1. **通用主炒油 1 瓶**：菜籽油 / 大豆油 / 米糠油 / 玉米油，挑便宜的就行（$5-10/L）
2. **EVOO 1 瓶**：凉拌、油醋汁、面包蘸、地中海菜。买**深色玻璃瓶 + 标 “extra virgin”** 的，避免 “pure olive oil”
3. **烤芝麻油 1 小瓶**：几滴起神效。出锅前淋；不主炒

加分项（按家庭习惯）：
- **黄油**（西式烘焙、煎食、酱汁——西餐党必备）
- **猪油 / 鸭油**（中式点心、油酥皮、增香）
- **牛油果油**（高温煎牛排、烘焙）
- **花生油**（如果常做爆炒鲁菜 / 苏菜）
- **亚麻籽油 / 紫苏油**（追求 omega-3 的人——少量凉拌）

## 4.2 烹饪方法 → 选油对照

- **凉拌、油醋汁、蘸料**：EVOO（果香）/ 烤芝麻油（中餐）/ 亚麻籽油（健康向）
- **煮汤、炖菜**：通用主炒油即可，量少（汤里油不主导味道）
- **中火煎鱼煎肉**：通用主炒油 / 黄油（西式）/ 牛油果油（追求中性）
- **大火爆炒（中餐）**：通用主炒油 / 精炼花生油（增香）
- **油炸（少做）**：精炼花生油 / 牛油果油 / 棕榈油（工业）
- **空气炸**：通用主炒油喷壶；EVOO 也可（200℃ 内安全）
- **西式烘焙**：黄油（首选）/ 椰子油（vegan 替代）/ 菜籽油（中性）
- **中式烘焙、油酥点心**：猪油（传统）

## 4.3 EVOO 的常见误区

**“EVOO 不能加热”——这是错的**。EVOO 烟点 190-210 ℃，足以应付煎、炒、烤。多项研究（Casal et al. 2010、de Alzaa 2018）显示 EVOO 在 180 ℃ 长时间加热的稳定性反而**优于**很多精炼油，因为它的天然多酚抗氧化能力强。

但有几个真实的注意点：

- **风味独特**：EVOO 在炒中餐里会盖菜本身的味道，不是味道差，是不搭
- **价格高**：用 EVOO 大火爆炒确实浪费——风味在高温下会破坏
- **不要油炸**：长时间高温会大量损失抗氧化物，性价比极低

**结论**：EVOO 用于煎、烤、低温炒、凉拌——OK；中餐爆炒、油炸——换通用油。

## 4.4 储存与过期

- **避光、密封、阴凉**：紫外线和氧气加速氧化
- **多不饱和油**（亚麻籽、葵花籽）开封后**冷藏**，3-6 个月用完
- **EVOO** 用深色玻璃瓶或不透光油壶，常温阴凉处可放 1 年
- **饱和油**（猪油、椰子油、黄油）最稳定，常温几个月没问题
- **酸败信号**：哈喇味、苦味、颜色变深、油变浑浊——立即扔。酸败的油含**自由基和氧化醛**，长期摄入有害

## 4.5 重复用油（炸过的油）

家用油炸后想回收再用：

- **过滤**（咖啡滤纸 / 细纱布）
- 装入干净玻璃罐，**密封冷藏**
- **最多重复 2 次**，每次烧之前闻一下、看一下颜色
- **3 次以上必须扔**——氧化产物累积
- 颜色明显变深、起泡多、有异味——立即扔，不管是第几次

## 4.6 健康选择优先级

研究证据强度从强到弱：

1. **EVOO**：地中海饮食 PREDIMED 等大型 RCT 证据最充分，关联心血管病 30% 风险下降
2. **菜籽油 / 牛油果油**：单不饱和高、烟点高、价格亲民——日常用最理想
3. **深海鱼油 / 亚麻籽油**：补 omega-3，但要**少量、新鲜、低温**
4. **可以适量用**：黄油、猪油、椰子油（虽然饱和高，但 2010 年后研究显示与心血管病关联弱于过去认为）
5. **少用**：棕榈油（健康 + 环境双重问题）、部分氢化植物油（含反式脂肪，多国已禁）
6. **千万别加热**：亚麻籽油
