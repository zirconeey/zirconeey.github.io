# 1. 这篇文章给谁看

写给从中国来美国、第一次撞上“美国为什么还要打印”这堵墙的留学生 / 学者。

我自己就栽过——某个工作日下午去 PennDOT 办笔试，材料清单看了三遍以为齐了，到现场工作人员要 I-20。我打开邮箱给她看 PDF，**她头都没抬**：“I need a physical copy. Print it out and come back next time.”——就这样被退回去，一来一回多花了半天。

很多刚来美国的人都有过类似的反应：**手机里明明有 PDF，扫码就能看，为什么非要纸？** 而且一旦你以为“这次电子版能用”——下次就栽在另一个地方。这篇文章把美国生活里**还坚持要实体材料**的场景一次性列清楚，并解释为什么会这样、读者该怎么准备。

> 这篇是观察类文章，不是法律建议。具体每个机构对“电子 vs 实体”的容忍度还在变，但**整体趋势是“实体优先”**——所以**最稳的策略是默认打印**，遇到接受电子版反而是惊喜。

# 2. 结论先行

**美国的“实体优先”不是落后，是三件事的综合：**

1. **法律传统**：wet signature（手写签名）+ paper trail（纸质留痕）是 200 多年判例的路径依赖
2. **联邦 + 50 州 + 数千郡的分权架构**：政府电子化由州主导，落地速度参差不齐——很多州的 DMV 系统还跑在 1990 年代的代码上
3. **责任分摊**：实体证件被工作人员“亲手看过”，万一出问题责任清晰；电子版的合规链条复杂，很多基层员工宁可不接

**作为读者，最简单的应对是——**

- **家里常备一台打印机**（$80 一台 brother 黑白激光，比每次跑学校 / 图书馆便宜）
- **重要 PDF 永远先打印一份**：I-20 / DS-2019 / I-94 / I-797 / EAD（备份）/ 签证页
- **出门前默认打印 vs 疑问时打印**：打印的成本是 $0.10，被卡半天的成本是几十美刀和一下午

## 3.1 政府机构：几乎全要纸

| 场景 | 必须实体的文件 | 备注 |
|---|---|---|
| **DMV / DOT** 办驾照 / Permit / 路考 / Real ID | I-20、SSN 卡 / SSA letter、地址证明 ×2、签证页 | 电子版 99% 拒收 |
| **SSA**（申请 SSN / 拒绝信） | 护照、I-94、I-20 / DS-2019、雇主信 | 现场办，纸质材料 |
| **USCIS** 各种申请（I-765 / I-129 / I-140 / I-485）| 表格 + 支持文件 + 实体支票 | 部分表格已支持 e-filing 但带支持材料的复杂 case 多数仍纸质 + 邮寄 |
| **海关 CBP**（入境）| 护照（含签证 stamp）、I-20 / DS-2019、I-94（电子但需要时打印） | 海关有时不要看；要看时**纸质胜于打开手机** |
| **IRS** 报税 / W-7 ITIN 申请 | 1040NR 表格、W-2 实体、外国身份证明件 | F-1 的 1040NR 多数仍纸质邮寄 |
| **法庭 / 法律送达** | 起诉状、传票 | 美国法庭对“service of process”的实体送达要求严格 |
| **选民登记 / 投票**（公民才需）| 选民登记卡 | 不影响留学生但邻居可能聊起 |

**最常踩的坑**：DMV / SSA 工作人员看到电子版时**往往不会“灵活通融”**——不是他们刻薄，是他们的内部 SOP 写着“必须实体”，违规对他们个人风险大。**抱怨他们没用，下次记得打印。**

## 3.2 学校 / 学术：成绩单 / 推荐信 / 标准化考试

| 场景 | 实体要求 |
|---|---|
| **GRE / TOEFL / SAT 进考场** | **必须实体护照**（驾照、学生卡都不接受）。某些考点连复印件都不收 |
| **学校 ISSO 旅行签证（Travel Signature）** | DSO 必须**亲笔在 I-20 第二页签蓝色墨水**（数字签名不算）|
| **官方成绩单（Official Transcript）** | 学校只发**密封信封 + 学校章**的纸质版本（电子也行但收件方多数要密封纸质）|
| **推荐信 / 入学材料** | 部分申请要密封纸质推荐信（教授直接寄给学校）|
| **学位证 / Diploma** | 实体颁发，丢了补办 $50-$200 |
| **入学体检 / 疫苗记录** | 多数学校要纸质 + 医生签字 + 诊所章（即便诊所有 patient portal）|

**特别要警惕的**：**Travel Signature**（每年 / 每学期一次的 ISSO 签字）——很多学生以为 DSO 在 PDF 上签名就行，但**美国海关在你回美国时只看 I-20 第二页上有没有“原始蓝色墨水手签 + 当年日期”**，PDF 签名一律不认。每次回国前 2 周去 ISSO 办，是必走流程。

## 3.3 雇主入职 I-9（实体优先 + 部分电子例外）

[Form I-9](https://www.uscis.gov/i-9) 要求雇主**亲眼看到实体证件原件**——这是联邦法律。具体怎么“看”：

- **传统**：HR 当面 inspect 你的证件原件（驾照 + SSN 卡 / 护照 / EAD 等）
- **远程例外（COVID 后）**：USCIS 2023 年起允许 [E-Verify-enrolled 雇主](https://www.uscis.gov/i-9-central) 用 video call 的方式 remote inspect 证件——但**仍然要看到证件原件在镜头前**，不接受 PDF / 截图

**关键**：HR 看的是**实体证件本身**，**不是 PDF 复印**。即便你的 EAD 卡刚收到、还没拿在手里——只能等卡到才能完成 I-9。

## 3.4 银行 / 健康保险 / 医院

| 场景 | 通常要的实体 |
|---|---|
| **银行开户** | 现场开户：实体身份证件 + 社安号 + 地址证明（部分接受电子账单截图）。线上开户：可以用照片 / 扫描，但**首笔大额交易 / 取现时仍要实体** |
| **健康保险卡** | 多数医院 / 药房柜台**坚持收实体卡**（即便保险公司有 app）—— 部分小诊所连扫码都不会 |
| **处方药** | 联邦层面已经强制 e-prescribing，但**部分管制药品 + 部分老派医生**仍开纸质处方 |
| **疫苗 / 体检报告** | 学校 / 雇主 / 移民局要的体检表，必须**医生纸面签字 + 诊所盖章**——电子档案打印出来不算 |

**实战建议**：保险公司 app 里**永远存一份 PDF + 钱包里带实体卡**，两边都备一份。

## 3.5 房东 / 租房

这一栏**美国差异最大**：

- 大型公寓集团（Greystar / Equity Residential 等）：**全 paperless**，DocuSign 签约、ACH 自动扣款、Portal 报修
- 个人房东（特别是郊区 / 大学城那种家庭房）：**坚持纸质租约 + paper check 房租**——部分房东根本不会用 Zelle / Venmo
- 中间地带：纸质租约 + 接受 Venmo / Zelle 房租

**典型坑**：找到便宜的个人房东房子，签约才发现房东要每月**亲手交支票**——支票本去哪买？Chase / BofA 网银上申请，$15-$30 一沓。**支票本是租房党必备**。

## 3.6 车辆 / 保险（车上必带实体）

宾州法律明文要求**车上随时备齐三样**实体：

1. **驾照**（开车人随身）
2. **车辆注册卡**（Vehicle Registration —— 一张半张 A4 纸）
3. **保险卡**（Insurance ID Card —— 一张小卡）

虽然手机 app 都有电子版，**警察 traffic stop 时仍坚持要看实体**——不是 100% 拒收电子，但**实体最稳**。我自己手套箱里这三样常备齐。

## 3.7 公证 / 法律 / 房产

- **Notary（公证）**：必须**亲自到 notary 面前**（in-person notarization）。最近少数州（PA 是其中之一）允许 [Remote Online Notarization (RON)](https://www.dos.pa.gov/) 但仍需视频 + 实体记录
- **房产 closing**：买房交易仍是 wet signature 大场面——20+ 张文件每张都要手签
- **婚姻 / 出生 / 死亡证书副本**：必须申请 **certified copy**（带州章的实体版本），$15-$25 一份

# 4. 为什么美国这么“传统”？

这一节是观察类内容，不是要“批评”美国——理解原因后**对应对方法很有帮助**。

## 4.1 法律传统：wet signature 和 paper trail

美国法律体系建立在 200 多年的英美普通法判例上，“wet signature（手写签名）+ paper trail（纸质留痕）”是默认。**电子签名（ESIGN Act, 2000）**虽然合法，但**州级法律 + 行业惯例**仍偏纸质——尤其是政府文件、不动产、遗嘱、领养、离婚等高风险场景。

实务上：法庭对“纸质 + 手签”文件的可信度判断**有 200 年判例支撑**，电子证据的判例还在积累。律师会保守选择纸质。

## 4.2 联邦 + 州 + 郡的分权

美国是 50 个州 + 3000+ 郡的联邦制国家——**没有“全国统一系统”**。这意味着：

- DMV 是州级机构，每个州系统独立。**Colorado 的电子化做得很好，宾州 / 纽约的就老**
- IRS 全国统一，但州税还要再单独交一份
- USCIS 是联邦级的，电子化推进相对快
- 法院 / 学校 / 银行都是各自独立的 IT 系统

**结果**：你在硅谷 Apple 工作觉得“美国全 paperless”，转头去乡村 DMV 又被打回原形。**两个美国共存。**

## 4.3 防欺诈 + 责任分摊

实体证件**比 PDF 难伪造**——

- 驾照有全息图、激光雕刻、UV 标记
- I-20 有 SEVIS 系统打印的条形码 + DSO 蓝墨水签字
- 护照有防伪线、芯片、签证 stamp
- 美元钞票上的水印、防伪线

工作人员“亲眼 + 亲手”看一遍，万一是假的——**责任不在 TA**（“我按 SOP 看了”）；如果只看 PDF 错了，TA 要承担“接受了伪造文件”的责任。**所以基层员工天然偏好实体。**

## 4.4 数字基础设施滞后

美国的政府 IT 投入比私营企业晚 10-20 年——

- DMV 系统不少是 1990 年代 COBOL 的
- IRS 直到 2024 年才推 Direct File（联邦报税官方在线服务），且仅限部分州
- USCIS 部分表格 e-file 才几年

**私营**部分（科技公司、连锁酒店、电商）则相反——**全美最 paperless 的国家之一**。这造成留学生的反差感：日常买东西比中国还方便（Amazon Prime 当日达），但办身份证件比国内都麻烦。

# 5. 实战建议：家里 / 钱包里该备什么

## 5.1 一台家用打印机：必备

家里**强烈推荐有一台黑白激光打印机**——

- **Brother HL-L2350DW** 或 **HL-L2390DW**：约 $80-$100，墨粉一年用不完
- 黑白激光不要喷墨（喷墨墨水贵 + 半年不用就干掉）
- 无线 + 双面打印是基本要求

> 一年下来你打印的东西：I-20 复印件、租约、报税表、医保理赔单、签证 renewal 材料、签证面试表 DS-160 confirmation、USCIS 表格、Notary 文件、车辆维修发票……平均每月 3-5 次。一台 $100 打印机 ROI 极高。

## 5.2 重要文件的“两份策略”

每份重要文件都至少有**两份**：

| 文件 | 原件 / 实体 | 复印件 | 电子备份 |
|---|---|---|---|
| 护照 | 钱包 / 保险柜 | 实体复印件 + 钱包 | iCloud / 手机加密 PDF |
| 签证 stamp | 在护照里 | 复印 1 份 | PDF |
| **I-20 / DS-2019** | 至少打印 2 份 | 实体备份在抽屉 | PDF（每次 DSO 签字后扫描更新）|
| I-94 | [i94.cbp.dhs.gov](https://i94.cbp.dhs.gov/) 打印 | — | PDF |
| EAD 卡 | 钱包 | 复印 1 份 | PDF |
| SSN 卡 | **保险柜**（不带钱包）| 复印 1 份藏好 | 加密 PDF |
| 出生证 / 学位证 | 保险柜 / 国内 | — | 翻译公证后扫描 PDF |
| 健康保险卡 | 钱包 | — | 保险公司 app + PDF |
| 车辆注册 + 保险 | 车上手套箱 | — | PDF |

## 5.3 钱包随身清单（最小化版）

钱包里**只放高频 + 丢失影响小**的：

- ✅ 美国驾照
- ✅ 信用卡 / 借记卡 1-2 张
- ✅ 健康保险卡
- ✅ 学校 ID
- ✅ 几张 $20 现金

**不放钱包的**：SSN 卡、护照、I-20 原件——丢了恢复成本太高。

## 5.4 出门前 5 秒检查

去这些地方前**默认打印 + 带齐**：

| 去哪 | 必带实体 |
|---|---|
| **DMV / DOT / SSA** | 全套：护照 + I-20 + I-94（打印）+ SSN / SSA letter + 地址证明 ×2 + 出生证（如要） |
| **银行办大事** | 驾照 + 护照 + SSN（数字记心里）+ 地址证明 |
| **签新工作 I-9** | 护照 + 签证页 + I-94 + I-20 / EAD（看 List A 哪个组合） |
| **房东签约** | 驾照 + SSN（数字）+ 收入证明（pay stub / offer letter） + 几张支票（押金）|
| **医院 / 药房** | 健康保险卡（实体）+ 驾照 + 当下处方纸 |
| **GRE / TOEFL 考场** | **护照（必须）**——任何其他证件不接受 |
| **国际旅行入境美国** | 护照（含 vis stamp） + I-20 + I-94 打印件 + EAD（如有）|

**5 秒规则**：出门前打开钱包 + 文件夹快扫一眼：“今天要去哪？要哪几样实体？”

# 6. 中国 vs 美国对比（速查）

| 场景 | 中国（典型）| 美国（典型）|
|---|---|---|
| 政府办事 | “随申办 / 国家政务服务平台”扫码 | DMV 现场排队 + 纸质表格 |
| 银行开户 | 手机银行 5 分钟 | 现场柜台 + 实体身份 + 30 分钟 |
| 健康保险 | 医保电子凭证扫码 | 实体保险卡刷 |
| 工作 I-9 | 没这一关（中国没有相当物） | HR 必须看实体证件原件 |
| 报税 | 个税 app | IRS 多数仍纸质邮寄 |
| 签合同 | 电子签名（e-签宝）| Wet signature 仍主流 |
| 大学成绩单 | 校园网下载 PDF | 学校发密封实体 + 寄到对方手里 |
| 处方 | 电子处方 | 多数纸质 / 部分电子 |
| 房租 | 微信 / 支付宝转账 | Paper check / Zelle 二选一 |

中国在过去 10 年完成了“无纸化跃迁”，美国还在过渡期——**这是观察事实，不是好坏判断**。两套系统各有取舍：中国效率高、可追溯但隐私让渡多；美国低效但**纸质留痕 + 多机构互不串数据**也是一种隐私保护。
