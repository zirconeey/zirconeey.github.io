# 1. 问题

洗衣机面板上一堆按钮和档位——「标准」「快洗」「棉麻」「羊毛」「轻柔」「强洗」「混洗」、温度从冷水到 95 ℃、转速从 400 到 1200 转，到底怎么选？内裤、袜子、外衣能不能扔一起洗？深色和浅色每次都要分开吗？洗衣机塞太满还是太空？洗衣袋什么时候用？

# 2. 结论先行

**温度**：日常 30-40 ℃ 已足够杀掉大部分微生物；月经 / 病期内裤、贴身衣物消毒用 60 ℃；只有床品 / 厚毛巾每月一次 90 ℃ 深度消毒

**程序**：90% 的衣服用 **标准 / 棉麻** 程序就够；只有羊毛、丝绸、文胸、贵重衣物需要 **轻柔 / 手洗** 程序；快洗适合“刚穿一次没多脏”的衣服

**装载量**：**滚筒装到 70-80%**（你能伸进一只手的空间）；上开式波轮装到 80%；**塞太满洗不干净，太空浪费水电**

**分洗规则**：

- ✅ 颜色：深色 / 浅色 / 白色 **必须分**；新衣服第一次单独洗（怕掉色）
- ✅ 材质：粗糙（牛仔、毛巾）和细腻（衬衫、丝绸）**最好分**
- ✅ 内裤袜子：**和外衣分开**——不是因为脏，是因为粪便菌 / 真菌交叉污染
- ✅ 重污：油渍、血、霉斑的衣物先单独预处理再洗

**关于「内裤袜子能不能和其他衣服一起洗」**：

> 短答：**最好分开。** 不是因为内裤袜子最脏（理论上 60 ℃ 洗就足以杀菌），而是 30 ℃ 冷水机洗下，**大肠杆菌、足部真菌确实会交叉污染到上衣 / 床单**。多个微生物学研究（如亚利桑那大学 Gerba 团队 2010s 系列研究）证实了这一点。

# 3. 科学原理

## 3.1 洗衣机做的三件事

不论什么型号的洗衣机，本质上做这三件事：

1. **化学**：洗涤剂溶解到水里，表面活性剂把油污从纤维上“撬”下来，包裹成胶束悬浮在水里
2. **物理**：搅动 / 滚筒翻滚 / 水流冲击 → 让水 + 洗涤剂渗透到纤维深处 + 把松动的污渍带走
3. **温度**：加快化学反应速率（每升 10 ℃ 反应速率约翻倍）+ 杀灭微生物（具体见 3.2）

理解了这三件事，所有“档位选择”问题都能转成“我需要多少化学 / 物理 / 温度”。

<p class="img-caption">**关键数字 60 ℃**：这是微生物学的"消毒线"——大多数家庭常见的致病菌、真菌、病毒在 60 ℃ 都会失活（接触时间 ≥ 5-15 分钟）。冷水洗能洗干净（油污是化学问题，靠洗涤剂）但**不能消毒**——这是为什么贴身衣物 / 病人衣物 / 月度床品建议用热水。</p>

**实操规则**：

- **大多数衣服**：30 ℃ + 标准程序（节能 + 不缩水）
- **运动衣 / 严重出汗**：40 ℃（更能溶解皮脂）
- **婴儿衣物 / 内裤每周一次 / 月经期内裤 / 病人衣物**：60 ℃
- **床品 / 浴巾每月一次**：90 ℃（深度消毒）
- **深色衣物 / 牛仔**：30 ℃ 冷水（防褪色）
- **羊毛 / 丝绸**：冷水或 20 ℃（高温会让羊毛缩水）

> **澳大利亚 / 美国习惯**：北美洗衣机大多有 cold / warm / hot 三档，对应 ~20 / 35 / 50 ℃。**不是数字越大越好**——大多数衣物 cold 已经够。

## 3.3 程序档位的本质

| 程序 | 实际做什么 | 适合 |
|---|---|---|
| **标准 / 棉麻**（Normal / Cotton）| 中速翻滚 + 中等水量 + 长时间漂洗 | 大多数日常衣物 |
| **快洗 / 15 分钟**（Quick）| 短时间 + 多水 | 刚穿一次、表面污染轻、急用 |
| **强洗 / 大件**（Heavy / Bulky）| 长时间 + 大水量 + 强翻滚 | 床单、毛巾、外套 |
| **轻柔 / 手洗**（Delicate / Hand Wash）| 慢速翻滚 + 低水位 + 短洗时 | 文胸、丝绸、刺绣、毛衣 |
| **羊毛**（Wool）| 极慢翻滚 + 短时间 + 低温 | 羊毛 / 羊绒 |
| **婴儿 / 除菌**（Baby / Sanitize）| 高温 + 多次漂洗 | 婴儿衣物 / 病期 |
| **预洗 + 主洗**（Pre-wash + Main）| 先短洗冲掉粗污 + 再正式洗 | 工作服、孩子的脏衣服 |
| **混洗**（Mixed）| 折中参数 | 不推荐——什么都不专 |

> **原则**：选程序就是选「翻滚强度 + 时长 + 水量」三件事的组合。**衣服越脏越粗 → 选强洗 / 长程序；衣服越精细 → 选轻柔 / 短程序**。

## 3.4 转速：脱水强度的取舍

转速决定脱水后衣物含水量。常见档位：

| 转速 | 脱水程度 | 适合 |
|---|---|---|
| **400 rpm** | 微脱（保留 80% 水）| 丝绸 / 羊毛（防变形）|
| **600 rpm** | 轻脱（60% 水）| 衬衫、文胸、合成纤维 |
| **800 rpm** | 中脱（45% 水）| 大多数日常衣物 |
| **1000 rpm** | 重脱（35% 水）| 牛仔、毛巾、床单 |
| **1200-1600 rpm** | 强脱（25-30% 水）| 厚重外套、毛巾（缩短晾晒时间）|

**取舍**：转速越高 → 衣物越干（晾晒时间短）→ 但**离心力越大也越容易让衣物变形 / 起皱 / 损伤纤维**。

实操：

- 衬衫、连衣裙：**用低转速** + 立刻拿出来甩平挂晾，几乎不用熨
- 牛仔、毛巾：**用高转速**（晾晒时间从 12 小时缩到 4-6 小时）
- 不知道怎么选：默认 800 转

## 3.5 装载量：为什么塞满或装太空都不对

家用洗衣机的最佳装载量是**桶容量的 70-80%**：

- **塞太满**（> 90%）：水和洗涤剂渗不进、翻滚空间不够、**洗不干净**；脱水时不平衡、洗衣机抖动剧烈
- **装太空**（< 30%）：水和洗涤剂浪费、翻滚不平衡也抖动、**实际清洁效果反而下降**（因为衣物之间机械摩擦不够）

**简单判断**：滚筒洗衣机塞到能伸进**一只手手掌**的空间为最佳；上开式波轮塞到**离顶部 10 cm**。

## 3.6 「内裤、袜子能不能和其他衣服一起洗」的真相

亚利桑那大学微生物学家 Charles Gerba 团队的多项研究（最有名的是 2010 年的家庭洗衣微生物学调查）发现：

- 一条穿一天的内裤上**平均含 0.1 g 粪便残留**（约 10⁷-10⁸ 个细菌，主要是大肠杆菌、肠球菌）
- 一双穿一天的袜子（特别是脚汗多的人）会沾上 **足部真菌**（白色念珠菌、皮肤癣菌）
- **冷水（30 ℃）机洗 + 普通洗衣液**对这些菌**只能减少而不能完全杀死**
- 测试发现：即使洗后的“干净衣物”上仍然能检出 **来自内裤的大肠杆菌**——证实了交叉污染

后续研究发现：

- 烘干机的高温（55-65 ℃）能完成大部分杀菌工作
- 60 ℃ 以上水温能彻底消毒
- 但中国 / 欧洲多数家庭只用冷水 + 自然晾干

**实操建议**：

- **理想方案**：内裤、袜子单独一桶，60 ℃ 水温（每周一次）
- **折中方案**：和外衣一起洗也可以，但用 ≥ 40 ℃ 水 + 烘干机高温烘干
- **绝对要分开**：婴儿衣物（皮肤屏障弱）、病人衣物（病期）、严重出汗后的健身衣
- **特别提醒**：抹布、地巾、宠物布料**绝对不要和内衣 / 床单一起洗**——交叉污染明显

# 4. 实践建议

## 4.1 一周洗衣的标准流程

**周一到周五（积攒）**：

- 内裤、袜子塞进**专用洗衣袋 / 单独脏衣篓**（不要和外衣堆在一起）
- 深色 / 浅色 / 白色衣物用 **三个分类袋** 分开
- 严重污渍的衣物（油渍、酱料、血）—— 立刻预处理（见 [污渍急救](/life/stain-removal)）

**周末（集中洗）**：

| 第几桶 | 内容 | 温度 | 程序 |
|---|---|---|---|
| 第 1 桶 | 白色衣物（白 T 恤、袜子、毛巾）| 40-60 ℃ | 标准 |
| 第 2 桶 | 浅色 / 中性色衣物 | 30 ℃ | 标准 |
| 第 3 桶 | 深色 / 黑色衣物 | 30 ℃（冷水）| 标准 + 低转速 |
| 第 4 桶 | 内裤 / 贴身衣物 | 60 ℃ | 强洗 + 长漂洗 |
| 第 5 桶 | 床品（每周一次）| 40-60 ℃ | 强洗 |

> 第 4 桶可以并到第 1 桶（同样高温白色），但内裤多的话分开洗体验更好。

## 4.2 颜色分类的最严格版

- **白色**：纯白、米白、浅灰
- **浅色**：天蓝、淡粉、米黄、卡其
- **中性色**：深灰、咖啡、酒红、深绿
- **深色 / 黑色**：黑色、海军蓝、深紫
- **红色 / 酒红**（特殊）：第一次洗 **必须单独洗**，第 2-3 次和深色一起；之后才能和浅色混
- **新衣服**：永远第一次单独洗（特别是深色、新牛仔——掉色严重）

## 4.3 洗衣袋什么时候用

**必须用洗衣袋**：

- 文胸（钢圈会卡住其他衣物 / 变形）
- 丝袜 / 网纱（容易破）
- 婴儿衣物（小件容易卡住翻滚轴）
- 内裤（隔离 + 减少摩擦）
- 衬衫领口（避免摩擦褪色）

**不要用**：

- 毛巾、床单（需要充分翻滚）
- 牛仔、外套（需要强力翻滚）

**洗衣袋本身**：每 6 个月换一次（破了立刻换）；用完打开摊开晾干（避免霉味）。

## 4.4 洗衣机挑刺常见错误

- ❌ **塞满到关不上盖**：清洁度大幅下降
- ❌ **不分颜色一锅端**：染色 / 串色风险
- ❌ **倒洗衣液过量**：“泡沫多 = 洗得干净”是错觉，反而残留多
- ❌ **湿衣服在洗衣机里放过夜**：闷出霉味
- ❌ **从来不开门通风**：洗衣机本身长霉（详见 [洗衣机也要清洁](/life/clean-the-washer)）
- ❌ **拉链 / 纽扣不扣**：钩坏其他衣服 + 自己变形
- ❌ **羊毛丝绸用普通程序**：缩水变形毁掉
- ❌ **白色衣物不分洗**：染色变灰

## 4.5 滚筒 vs 波轮的差异

| 维度 | 滚筒 | 波轮（上开式）|
|---|---|---|
| 清洁度 | 中等偏好 | 偏强（机械力大）|
| 衣物损伤 | 较小 | 较大（牛仔可能磨损快）|
| 节水节电 | 好 | 一般 |
| 时间 | 长（1.5-3 小时）| 短（40-60 分钟）|
| 温度选择 | 多档（含 95 ℃）| 一般冷水 |
| 价格 | 较贵 | 便宜 |
| 大件 | 较好 | 一般 |
| 噪音 | 较小 | 一般 |

> 整体来说：**滚筒更适合精细衣物 + 多温度需求**；**波轮更适合“扔进去洗干净就好”的简单需求**。北美 / 欧洲家用以滚筒为主；中国家用历史上波轮多，近 10 年滚筒越来越普及。

## 4.6 几个误区

**误区 1：「快洗程序就是省时间」**

错。快洗用更多水 + 同样的去污剂浓度 + 短翻滚——**洗不掉重污**。只适合“刚穿一次没多脏”的衣服。重污用快洗 = 浪费水 + 没洗干净。

**误区 2：「洗衣液越多越干净」**

错。家用洗衣液有最佳浓度——**多了**会让漂洗不彻底、残留导致皮肤敏感、衣服发硬。看包装上的“单次推荐量”，不要多倒。

**误区 3：「冷水洗就 = 没洗干净」**

部分错。**化学污渍**（油污、汗渍）现代洗衣液在冷水下也能解决；**只有微生物消毒**冷水做不到。日常衣服 30 ℃ 冷水完全够。

**误区 4：「贴身衣物和外衣一起洗才省事」**

可以但有取舍。微生物会交叉污染；如果你用的是 30 ℃ 冷水 + 自然晾干，建议分桶。

**误区 5：「洗衣机塞越满越省水电」**

错。塞太满洗不干净，可能要重洗——更费水电。70-80% 是甜区。
