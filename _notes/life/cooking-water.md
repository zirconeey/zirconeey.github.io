# 1. 问题

家里水龙头出来的自来水到底能不能直接做饭？煮饭、煮面、炖汤、洗菜、泡茶、煮咖啡、给娃冲奶粉——是不是都该用瓶装纯净水才安全？还是大多数情况下自来水煮沸就行？「自来水含氯」「老房子有铅管」这些担心到底是不是过度焦虑？

# 2. 结论先行

中国合规自来水（GB 5749 标准）煮沸后，**绝大多数烹饪场景直接用就好**。需要“升级”用水的场合很少，按必要性排序：

| 场景 | 推荐 | 理由 |
|---|---|---|
| **婴儿冲奶粉** | ✅ 瓶装纯净水或煮沸冷却的自来水 | 婴儿肾未成熟，怕氯副产物 + 矿物超标 |
| **泡茶（特别是绿茶 / 白茶）** | ⚠️ 滤过的自来水或低硬度瓶装水 | 硬水的钙镁会让茶汤浑浊、单宁沉淀 |
| **手冲精品咖啡** | ⚠️ 中等硬度水（50-175 mg/L） | 太软萃取过度，太硬钝化风味 |
| **直接饮用** | ✅ 自来水煮沸即可（北方硬水建议过滤） | 煮沸杀菌、氯挥发；嫌口感差再升级 |
| **煮饭、煮面、蒸菜** | ✅ 自来水 | 极硬水（> 300）米饭略硬，但绝大多数地区无所谓 |
| **炖汤、焖菜** | ✅ 自来水 | 长时间高温氯早就跑光了 |
| **焯水、洗菜、洗水果** | ✅ 自来水 | 水会丢弃 / 后续还要再洗，没必要升级 |
| **做面包、馒头（发酵）** | ✅ 自来水 | 适度硬水反而对面筋 / 酵母**有利** |
| **制冰（清澈度优先）** | ⚠️ 纯净水 | 杂质少，冰更透明；但味道无差 |

底层逻辑就两条：

1. **氯（消毒剂）会挥发**——煮开水的过程基本就消除了氯味
2. **钙镁矿物对大多数烹饪影响很小**——只有泡茶 / 煮咖啡 / 某些精细发酵会放大差异

> “瓶装水更健康”是营销叙事而不是科学结论。膳食里水的矿物质贡献占比 **不到 1%**，正常吃饭的人不会因为喝自来水缺矿，也不会因为喝矿泉水补出多少。

## 3.2 自来水的“氯味”是怎么回事

水厂消毒大多用 **氯（次氯酸钠 / 液氯）** 或 **氯胺**——这是杀灭细菌、病毒、原虫的关键步骤，全球近 100 年公共卫生最重要的发明之一（1908 年新泽西首次氯化消毒后，伤寒发病率断崖下降）。

出厂水余氯标准：**0.3-2 mg/L**（GB 5749 要求出厂 ≥ 0.3，末梢 ≥ 0.05）。

氯的“味道”主要来自：

- **游离氯**（次氯酸 HClO）—— 加热或敞口放置会快速挥发
- **氯胺**（NH₂Cl）—— 美国部分城市改用氯胺，挥发慢、味道更“塑料感”
- **氯化消毒副产物（DBPs）**—— 三卤甲烷类（THMs）、卤乙酸（HAAs）等。GB 5749 限值：氯仿 ≤ 0.06 mg/L，THMs 总量 ≤ 1.0 mg/L

> **DBPs 致癌吗？** 流行病学有微弱关联（极长期高暴露下膀胱癌略升），但日常剂量的风险被氯化消毒**杀灭病原**带来的健康收益**远远抵消**。WHO / EPA / 国家卫健委一致结论：**继续氯化消毒是必须的，DBPs 在限值内是可接受的代价**。

**简单去除氯味的物理方法**：

1. **煮沸**：加热到 80 ℃ 以上，氯几分钟内基本挥发完
2. **静置**：敞口放置 1-2 小时，氯也会挥发大半
3. **活性炭滤水壶（Brita 等）**：吸附氯 + 部分有机物
4. **反渗透（RO）净水器**：完全去除

## 3.3 硬水对烹饪的真实影响

水的硬度（钙镁含量）会和食物中某些成分反应。哪里影响大、哪里影响小：

**🟢 几乎没影响（直接用自来水）**：

- **煮饭、煮面、蒸菜、炖汤、煮饺子**：钙镁离子在大量水里浓度低，对食物风味的影响在普通人尝不出的水平
- **焯水、烫菜**：水最终丢弃，水质完全无关
- **洗菜、洗水果**：自来水直接冲，担心余氯就最后冲一遍冷水放置过的水

**🟡 有可见影响（建议软化）**：

- **泡茶**（见 3.4 详细）：硬水让茶汤浑浊、苦涩
- **煮咖啡**（见 3.5 详细）：水质大幅影响萃取和风味

**🔵 反而需要适度硬度（不要纯净水）**：

- **做面包 / 馒头（发酵）**：钙离子加固面筋蛋白结构，让面团更有弹性；纯净水做的面包面筋偏弱
- **做豆腐**：石膏 / 卤水点豆腐时，**水的钙含量参与凝固**，硬水反而“省石膏”
- **腌制 / 泡菜**：钙能让蔬菜保持脆度（这就是为什么泡菜厂会主动加氯化钙）

**⚠️ 极硬水（> 300 mg/L）的小坑**：

- 米饭略偏硬、不够软糯（北方很多人感觉煮出来的饭“差点意思”）
- 绿叶菜焯水后颜色不够亮（钙会让叶绿素镁离子被钙置换，变橄榄绿）
- 茶具、咖啡机水垢加速（这是设备问题，见 [电热水壶水垢能喝吗？](/life/kettle-scale)）

## 3.4 茶水：水质对茶味的影响

唐代陆羽《茶经》就说“其水，用山水上、江水中、井水下”，已经在讲水质和茶味的关系。现代科学解释如下：

**茶的风味化学**：

- 茶多酚（儿茶素 EGCG 等）—— 涩味
- 咖啡因 —— 苦味
- 茶氨酸 —— 鲜味、回甘

**硬水对这些的影响**：

- **钙离子**会和茶多酚结合形成微小不溶颗粒 → **茶汤变浑浊、出现“金圈”或浮膜**
- **碳酸氢根**（重碳酸盐）会让茶汤 **pH 升高** → 单宁结合更紧 → **苦涩感更强**
- 镁、钙也会**钝化**茶氨酸的鲜味

实测：用 50 mg/L 硬度的水和 300 mg/L 的水分别冲同款绿茶，盲品大多数人都能区分，且认为软水冲的“更鲜亮”。

> **泡茶水推荐**：硬度 **30-100 mg/L** 是甜区。低于 30 mg/L 的纯净水也行但有些茶人觉得风味“立不起来”；高于 150 mg/L 茶汤就明显受影响。

各茶类的敏感度（高 → 低）：

1. **绿茶 / 白茶 / 黄茶**（最敏感）—— 浅色清饮，水质问题立刻暴露
2. **乌龙茶**（中等）—— 工艺重，对水质宽容一些
3. **红茶**（中等）—— 加糖加奶可掩盖
4. **黑茶 / 普洱**（最不敏感）—— 重发酵 + 浓汤，硬水反而能“压一压”

**实操**：

- 滤水壶过滤的自来水：性价比最高
- 瓶装水选低硬度款：农夫山泉（硬度 ~30-50）、怡宝（< 50）适合泡茶；依云（硬度 ~310）反而**不适合泡茶**——是的，依云不是泡茶神水
- 蒸馏水 / RO 纯水：可以用但风味偏“空”

## 3.5 咖啡：水质比咖啡豆讲究多了

精品咖啡协会（SCA / SCAA）有一份官方水质标准：

> **理想冲煮水**：TDS 75-250 ppm，硬度 50-175 mg/L（CaCO₃），碱度 40-70 mg/L，pH 6.5-7.5

为什么精品咖啡圈对水这么讲究？因为咖啡里有 1000+ 种风味化合物，水的硬度直接决定哪些被萃取出来：

- **太软的水**（< 50 mg/L，包括纯净水）：缺乏萃取所需的镁离子，会导致**萃取过度 + 苦味突出 + 风味平淡**——这是反直觉但反复被实验证实的
- **太硬的水**（> 200 mg/L）：钙离子和咖啡风味化合物结合形成沉淀，**钝化香气、增加涩感**；还会让咖啡机水垢飙升
- **碱度太高**（> 70 mg/L）：抵消咖啡的酸度，让本来该亮的水果调变“闷”

**实操**：

- 咖啡店用的“咖啡水”：通常是**反渗透 + 后矿化**——先过滤干净再人工加入特定比例的镁 / 钙离子
- 家里冲咖啡：滤水壶过的中等硬度自来水通常 OK；要进阶可以买专门的咖啡水（如 Volvic 火山岩水）或者按公式自己调（搜 “Third Wave Water” 看商业方案，自己 DIY 也行）
- **绝对不要用纯净水冲手冲咖啡**——这是高级玩家最常犯的错

## 3.6 婴儿用水：唯一不可妥协的场景

婴儿（特别是 6 个月以下）的肾脏功能、免疫系统都不成熟，对水中的污染物比成人敏感得多。

WHO + 中国卫健委建议冲泡奶粉用水的几条硬性要求：

1. **必须煮沸**（≥ 70 ℃ 冲泡，杀灭奶粉中可能的阪崎肠杆菌）
2. **优先用低矿物的瓶装纯净水**或**煮沸冷却后的自来水**
3. **不要用矿泉水**——婴儿肾脏处理不了高钠（很多矿泉水钠含量 50+ mg/L）；钙磷比也不一定适合婴儿
4. **不要用反复烧的水**（虽然成人不用焦虑，但婴儿建议每次新水）
5. **绝对不要用井水 / 山泉水 / 不明源水**——硝酸盐风险（“蓝婴综合征”）

> 选瓶装水时看标签：「**饮用纯净水**」（蒸馏 / RO，钠 < 5 mg/L）或「**婴儿水**」（专门标注 “infant water” / “适合婴儿”）。**避开** 「天然矿泉水」「山泉水」（钠、钙、氟可能偏高）。

## 4.2 几个特殊情况

**早上起来的“头水”**：

水管里整夜静置的水可能含微量金属离子（铜、铁），特别是老房子。建议**早上先放水冲 30 秒**再接饮用 / 做饭水。冲下来的水可以接桶用来浇花、冲马桶，不浪费。

**夏天高温时段后**：

楼顶水箱在烈日下温度可达 30-40 ℃，水温升高加速生物膜形成。夏天高温后头水也建议先冲。

**搬到新小区 / 新装修房子**：

PEX、PPR 等现代塑料管道相对干净；但装修后头几个月水管可能有黏合剂、PVC 增塑剂等微量溶出。建议入住前冲洗水管 1-2 小时，再装净水器。

**长期出差回家**：

水管里水静置 1 周以上的话，**先放水冲 2-3 分钟**再用。

**老房子（30+ 年）**：

如果不确定水管材质，建议装一个**带活性炭 + 离子交换 + 重金属滤芯**的台下净水器（不是简单的活性炭滤水壶）；超老房子（1980 年代前可能含铅管）尤其要重视。

## 4.3 净水器 / 瓶装水怎么选

**家用净水方案性价比排序**（从低到高）：

1. **凉白开**：自来水煮沸晾凉。**对绝大多数家庭已经够了**。氯挥发、细菌灭活、暂时硬度部分沉淀
2. **滤水壶（Brita / 海尔 / 3M）**：100-200 元壶 + 几十块钱一个滤芯（每月一换）。去氯味 + 部分软化 + 改善口感。**性价比之王**
3. **台下三级 / 五级过滤**：PP 棉 + 活性炭 + 超滤膜。1000-2000 元，覆盖大部分需求。**适合老房子**
4. **反渗透（RO）**：去除 99% 矿物 + 重金属 + 有机物。1500-4000 元。**最干净但水“空”**——出水接近纯净水。**适合水质特别差的地区**
5. **后矿化 RO**：RO 之后人工加回少量钙镁。最贵、最适合品茶 / 品咖啡

**瓶装水选购**：

| 类型 | 矿物含量 | 适用 |
|---|---|---|
| **饮用纯净水**（怡宝、康师傅、娃哈哈） | ~0（蒸馏 / RO） | 婴儿冲奶、家电（滚筒洗、加湿器）、煮咖啡需要软水时 |
| **饮用天然水 / 天然山泉水**（农夫山泉） | 低（30-80 mg/L） | 日常饮用、泡茶 |
| **饮用天然矿泉水**（百岁山、依云、Perrier） | 中到高（100-500+ mg/L） | 日常饮用；不适合婴儿 / 泡茶 / 煮咖啡 |

> 注意瓶身的「**饮用纯净水**」「**饮用天然水**」「**饮用天然矿泉水**」是国标分类，矿物含量差很多。看看 NUTRITION FACTS 一栏的钠 / 钙 / 镁标注就知道了。

## 4.4 几个误区

**误区 1：「自来水致癌」**

错。氯化消毒副产物（DBPs）含量在 GB 5749 限值内（远低于流行病学风险阈值），且煮沸还会减少。**比 DBPs 微弱风险更大的是不消毒的水带来的霍乱、伤寒、肠胃炎**。

**误区 2：「矿泉水比纯净水营养，长期喝纯净水会缺矿物」**

错。膳食里水的矿物贡献 < 1%。一杯牛奶的钙比一升矿泉水还多。正常吃饭不会因为喝什么水缺钙。

**误区 3：「瓶装水比自来水安全」**

不一定。多个独立检测发现，部分瓶装水的微生物指标 / 微塑料含量比合规自来水**还差**——特别是塑料瓶在高温下储存的。瓶装水的优势不是“更安全”而是“口感稳定 + 携带方便”。

**误区 4：「煮沸后水里全是氧气没了，喝起来死气沉沉」**

是真的（沸水溶解氧确实低）但**和健康无关**。“活水”的概念是民俗不是科学。如果不喜欢凉白开口感，加冰或在水里加柠檬片即可。

**误区 5：「高级食材必须配高级水」**

部分对（茶 / 咖啡 / 婴儿奶粉），大部分错（炖鸡汤、煮饭、焯菜没区别）。把钱花在食材上，比花在升级用水上回报率高。
