# 1. 这篇文章给谁看

写给**已经懂了医保基础**（看过 [第一篇](/life/us-health-insurance-basics)）但**第一次面对“我生病了我应该去哪”**的留学生：

- 感冒了 / 喉咙疼——去哪？
- 突然腹痛 / 高烧——去 ER 吗？
- 想看皮肤 / 妇科 / 牙——怎么开始？
- 想吃中国带过来的药能不能直接吃？需要医生开方吗？
- 学校 health center 真的免费吗？什么时候去？

中国看病：**走到任何医院挂号 → 排队 → 看医生 → 拿药 → 一次完成**。
美国看病：**先选 PCP → 预约 → 看 PCP → 拿 referral → 预约专科 → 几周后看 → 取药要去 pharmacy → 几个月后收到账单**。

**两套系统的“动作差异 100%”**——这一篇手把手讲清楚每一步。

读完这一篇你会知道：

- **怎么选 PCP**——人生第一个 PCP 怎么找、怎么 schedule
- **不同病去哪**——感冒去 telehealth / 急诊去 ER / 慢病去 PCP——一张就医分级图
- **普通病的看病流程 8 步**——从预约到拿药
- **Referral 怎么拿**——HMO 用户必经
- **处方药怎么取**——pharmacy 流程 + 取药省钱
- **牙科 / 眼科 / 心理咨询**——这些为什么不在主医保里 + 怎么单独安排

第三篇会讲账单 / 谈判 / 留学生特殊场景（带药 / 怀孕 / 心理健康深入），见 [医疗实战](/life/us-medical-bills-and-tips)。

# 2. 结论先行

**就医分级“金字塔”**：

| 严重程度 | 去哪 | 价格 | 等待 |
|---|---|---|---|
| 真急 / 危及生命（呼吸困难 / 大量出血 / 心脏症状）| **ER（Emergency Room）** | $1,000-$5,000+ | 立即 |
| 中度急 / 不能等到明天（高烧 + 严重呕吐 / 中度受伤）| **Urgent Care** | $100-$300 | 1-3 小时排队 |
| 普通病 / 当天搞定（喉咙痛 / 轻微皮疹 / 处方续）| **Telehealth / Walk-in clinic** | $0-$80 | 同日 |
| 想见医生但不急（年度体检 / 慢病管理）| **PCP appointment** | $20-$80 | 1-3 周排预约 |
| 学生 / 校园生活相关（一般小病 / 心理咨询）| **学校 Health Center** | **$0-$30** | 当日或几天 |

**核心原则**：**别去 ER 看不该去的病**——ER 一次平均账单 $1,500-$3,000（即便你有保险），是 Urgent Care 的 10 倍。**判错医疗等级是留学生最贵的踩坑**。

# 3. 第一步：选 PCP（人生第一个主治医师）

如果你是 HMO，**没 PCP 看不了任何专科**。如果是 PPO 也建议有一个——长期看病有连续病历比“每次新医生从零问起”省事得多。

## 3.1 怎么找 PCP

**3 个找 PCP 的渠道**：

### A. 保险公司 Provider Directory

- 登录保险公司网站 / app
- 找 “Find a Doctor / Provider Search”
- 筛选条件：**距离 + 专业 = “Family Medicine” 或 “Internal Medicine”** + 性别偏好 + 语言（部分网站可以筛“Mandarin”）+ accepting new patients

### B. 学校 Health Center

- 学生医保的 PCP **通常默认是学校 health center 的 family medicine 医生**
- 多数学生不需要单独选——**第一次去 health center 时系统自动配 PCP**

### C. 推荐 / 评论

- **Yelp / Google Reviews**——读过去患者评论
- **Healthgrades / Zocdoc**——专业医疗评分平台
- **学长学姐 / 学生群**——华人圈最实用的推荐渠道

## 3.2 选 PCP 的关键标准

- ✅ **In-Network**：必须！否则 100% 自付
- ✅ **Accepting new patients**：很多 PCP 不接新患者
- ✅ **离学校 / 公司近**（驾车 < 20 分钟）
- ✅ **有空位预约**（多数好 PCP 排到 4-6 周后）
- ✅ **能用中文 / Mandarin-speaking**（如果你英文医疗术语不熟）—— 大学城周边经常有华人医生
- ✅ **女医生 / 男医生偏好**——心理舒适度也重要

**实战**：第一年用学校 health center 的 default PCP 即可——校外 PCP 等你想换再换。

## 3.3 第一次 PCP 预约：怎么 schedule

**3 个预约方式**：

### A. 电话（最稳）

- 打 PCP 诊所电话（Member Services 给你的电话）
- 报你的 Member ID + 出生日期 + 想约什么类型 visit
- 接线员给你最近可约的时间槽
- **first-time patient appointment 通常 30-45 分钟**——比一般 visit 长

### B. 在线（保险公司 / 诊所 app）

- 登录 patient portal（每个诊所给你的）
- 找“Schedule Appointment”
- 选医生 + 选时间槽

### C. Zocdoc / 第三方平台

- [Zocdoc.com](https://www.zocdoc.com/)：搜索你保险接受的 PCP + 在线预约 + 整合多个诊所
- 部分 PCP 不在 Zocdoc 上，要直接打电话

## 3.4 第一次 PCP visit 干什么

**典型流程（45-60 分钟）**：

1. **Check-in**（10 分钟）：确认 ID / 保险卡 / 填表（病史 / 家族史 / 过敏 / 当前药物）
2. **Vitals**（5 分钟）：护士测体重 / 身高 / 血压 / 体温 / 心率
3. **PCP visit**（20-30 分钟）：
    - 自我介绍
    - 问你**为什么来 + 病史 + 生活习惯（吸烟 / 喝酒 / 运动 / 饮食 / 性生活）**
    - **physical exam**——心肺听诊 / 触诊 / 反射
    - **建议 baseline labs**（血常规 / 胆固醇 / 血糖）—— 通常另约
    - **疫苗记录核对**（特别是留学生，可能学校已经查过）
4. **Wrap-up**：医生给你 follow-up 计划 + 任何 referral

**第一次 visit 一般是免费 / 低 copay 的“年度体检”**（preventive care）—— **新患者要争取把 first visit 设为 annual physical**。

# 4. 不同病去哪：5 个就医选项详解

## 4.1 学校 Health Center（留学生第一选择）

**适合**：感冒 / 喉咙疼 / 轻微皮疹 / 流感 / 妇科常规 / 心理咨询 / 各种小病。

- **价格**：通常 $0-$30 / visit（部分学校全免，包含在学费里）
- **预约**：很容易，多数当天 / 第二天可看
- **服务范围**：诊断 + 处方 + 简单实验室 + 部分疫苗 + 心理咨询
- **限制**：通常**只服务在校生** + 暑假可能减少服务 + 较严重病会被 referred 到外面

**核心建议**：**学校 health center 是留学生第一年看病的“性价比天花板”**——能在这看的就在这看，能省一大笔。

## 4.2 PCP Office Visit（计划性看病）

**适合**：年度体检 / 慢性病管理 / 处方续药 / 不急但需要见医生 / 健康咨询。

- **价格**：copay $20-$80
- **预约**：1-3 周排（年度体检常常 4-6 周）
- **限制**：**不适合急病**——你打电话约时通常医生已经满了

**实战**：PCP visit 提前 1-2 个月排——**别等病了才约**。

## 4.3 Telehealth / Telemedicine（线上问诊）

**适合**：感冒 / 喉咙疼 / 轻微皮疹 / 处方续 / 心理咨询 / 简单咨询。

- **价格**：$0-$80（多数保险已涵盖）
- **预约**：5-30 分钟内可上
- **平台**：保险公司自家 app（Aetna Telemedicine / Cigna 等）+ Teladoc / Doctor on Demand / Amwell
- **服务范围**：视频通话 15-30 分钟 + 处方 + 简单建议
- **限制**：不能做 physical exam / 不能开管制药 / 不能做实验室

**实战**：**疫情后 telehealth 已经是主流**——半夜咽喉肿痛、周末轻度皮疹—— **比急诊性价比高 10 倍**。

## 4.4 Urgent Care（小急诊）

**适合**：扭伤 / 中度切伤 / 中度发烧 / 中度过敏 / 不能等明天但不致命的病。

- **价格**：copay $100-$300
- **等待**：1-3 小时（看 walk-in vs 预约）
- **服务范围**：X-ray / 缝合 / IV 输液 / 简单处方
- **特点**：不需 referral / 多数 in-network / 24x7 营业

**实战**：**ER 之前优先去 Urgent Care**——价格 1/10 + 多数小急能搞定。**周末晚上突然发烧 / 流感**——typical urgent care 场景。

**找 in-network Urgent Care**：保险公司 app 里“Find Urgent Care Near Me”。

## 4.5 Emergency Room (ER)

**适合**：

- ✅ 真急 / 致命：胸痛 / 呼吸困难 / 大量出血 / 严重头部受伤 / 意识丧失 / 急性中风症状
- ✅ 怀疑骨折 / 严重外伤
- ✅ 严重过敏反应（anaphylaxis）

**不适合**：

- ❌ 一般感冒 / 喉咙疼 / 中度发烧——这是去 Urgent Care 的
- ❌ 慢性病管理 / 复诊
- ❌ 取处方药

**为什么 ER 这么贵？**

- **ER copay $200-$500（在保险里高）**
- **真实账单 $1,500-$5,000**（保险报多数后你付剩余）
- **超贵的治疗**：$10K-$50K 也常见（手术 / ICU）
- **ER 不能拒绝你**——但必须按“医院全价”开账单（即便你最后只是被告知“可以回家”，也至少 $500-$1,500）

**实战**：**ER 决策测试**——“如果不立刻处理，我会不会死 / 残废 / 严重恶化？” Yes 去 ER；No 去 Urgent Care。

## 4.6 Walk-in Clinic（药店里的诊所）

**CVS Minute Clinic / Walgreens Healthcare Clinic / Walmart Care Clinic**：

- 在 CVS / Walgreens 店内
- **价格**：$70-$150（有保险可能 $0-$50）
- **服务范围**：流感 / 咳嗽 / 简单皮疹 / 疫苗 / 简单 lab
- **不需要预约**：walk-in，等 30-60 分钟
- **覆盖时间**：通常 8AM-9PM 每天

**实战**：**周末 / 晚上 / 学校放假**且不严重——这是 telehealth 之外的另一个好选择。

# 5. 普通病的看病流程 8 步

下面是从预约到拿药的**完整流程**，以“PCP 看普通病”为例：

## 步骤 1：判断病情严重度

参考第 4 节就医分级——**判对很重要**。

## 步骤 2：找 in-network 医生

- 保险公司 app / Provider Directory
- **Verify in-network**：直接打诊所确认“My insurance is X, plan Y, are you in-network?”

## 步骤 3：预约（schedule appointment）

- 电话 / portal / Zocdoc
- 提供：Member ID / 出生日期 / 想约什么 / 紧急程度

## 步骤 4：填新患者表 + 准备文件

- 第一次去：填新患者表（病史 / 过敏 / 家族史 / 当前药）
- 带：保险卡 / Photo ID / 当前所有药物清单（中英文翻译）

## 步骤 5：到达诊所 / Check-in

- 提前 15 分钟到
- 出示保险卡 + Photo ID
- 复印保险卡 / 拍照
- 收 copay（**当场付**）—— 信用卡 / debit / cash 都行
- 等候

## 步骤 6：Vitals + Doctor Visit

- 护士测体温 / 血压 / 心率（5 分钟）
- 医生 visit（10-30 分钟）
- 医生开 lab / 处方 / referral / follow-up

## 步骤 7：拿处方 / 后续

- **Lab 抽血**：诊所内 lab / Quest Diagnostics / LabCorp（约 1-3 天结果）
- **处方**：医生**电子转送 pharmacy**（你不需要拿纸条），告诉医生你想用哪家 pharmacy
- **Referral**：HMO 用户拿 referral 后预约专科

## 步骤 8：取药

- 去你选的 pharmacy（CVS / Walgreens / Costco / Rite Aid）
- 报 Member ID（pharmacy 会查保险）
- 付 copay（如 $5-$80）
- 拿药 + 阅读说明

## 步骤 9（几周后）：账单到家

- EOB（Explanation of Benefits）— 保险公司寄
- 医院 / 诊所账单
- **核对二者**——见 [第三篇](/life/us-medical-bills-and-tips)

# 6. Referral 流程详解

## 6.1 什么时候需要 Referral

**HMO 计划下，看任何专科 / 做任何复杂检查通常都要 referral**：

- 皮肤科（Dermatology）
- 妇科（OB/GYN）—— 多数 HMO 妇科直接看不需 referral
- 心理科（Psychiatry / Therapy）
- 眼科（Ophthalmology）—— 多数眼科是分开的 vision plan
- 骨科（Orthopedics）
- 心血管（Cardiology）

**不需要 referral 的**：急诊 ER / Urgent Care / 部分妇科 / 学校 health center。

## 6.2 Referral 怎么拿

1. 看 PCP（首先）
2. PCP 评估你的情况，**如果他能处理就直接处理**（很多皮肤问题 PCP 直接给药）
3. 必要时 PCP **写 referral**——电子发送给保险公司（你看不到，但**可以打保险公司确认**）
4. **保险公司批准** referral（多数自动）
5. 你预约专科——预约时**告诉接线员“I have a referral from Dr. X”**
6. 专科诊所**会和保险公司核对**——通常自动

## 6.3 Referral 注意事项

- **Referral 通常 6-12 个月有效**
- **Referral 限专科**：如果 referral 是“皮肤科”，不能用来看心脏科
- **Referral 限医生**：referral 通常指定具体医生 / 诊所
- **过期 / 用错**：保险拒赔，账单全自费

# 7. 处方药完整流程

## 7.1 处方药基本概念

- **Prescription**（Rx）：医生开的处方
- **Pharmacy**：药房（CVS / Walgreens / Costco / Walmart / Sam's Club / Kroger）
- **Generic vs Brand**：仿制药 vs 品牌药——**generic 通常便宜 70-90%**
- **Mail-order**：90 天 supply 邮寄到家——通常更便宜

## 7.2 取药流程

### A. 医生开方

- 现代多数都是**电子处方**——医生在 EHR 系统里直接发给你的 pharmacy
- 你要告诉医生：**“Please send to CVS at [address]”** 或类似
- **不再有纸质处方**（管制药例外）

### B. Pharmacy 处理

- Pharmacy 收到电子方
- **打开保险**：扫描 Member ID + Rx BIN + Rx Group
- 报销审核——确认在 formulary（药物清单）+ 适当 copay
- **打电话 / SMS / app**通知你药已 ready

### C. 取药

- 报姓名 / 出生日期
- 付 copay（generic $5-$15 / brand $30-$80）
- Pharmacist **简单解释药用法 + 副作用**
- 装袋拿走

## 7.3 处方药省钱技巧

- ✅ **Generic 替代 brand**——告诉医生“prescribe the generic version please”
- ✅ **Mail-order 90 天 supply**：用保险的 mail-order pharmacy（如 CVS Caremark / Express Scripts），**90 天的 copay 通常等于 60 天**
- ✅ **GoodRx**：**没保险时**比保险还便宜的 coupon（pharmacy 接受 GoodRx code）
- ✅ **Costco Pharmacy**：**会员制但药价超低**——非会员也能去 pharmacy（联邦法律强制开放）
- ✅ **Walmart $4 generic 列表**：~400 种 generic $4 / 30 天，$10 / 90 天
- ✅ **Patient assistance programs**：brand 药（如 Insulin）—— manufacturer 直接补贴

## 7.4 几个反直觉点

- **Pharmacist 比 medical doctor 更懂药物相互作用**——有疑问问 pharmacist，**他们专业且免费咨询**
- **同一种药不同 pharmacy 价格差几倍**——多比较
- **没保险时 GoodRx + cash 价 < 保险 copay**——尤其是 generic

# 8. 牙科 / 眼科 / 心理咨询的特殊安排

这三类是美国医保的“灰色地带”——通常**不在主医保里**，需要单独安排。

## 8.1 牙科（Dental）

**为什么牙不在医保里？**——历史原因 + 牙医工会和医生工会几十年互相独立运作。**ACA 强制儿童牙保险，但成人牙保险可选**。

**留学生牙保险路径**：

| 路径 | 保费 | 覆盖 |
|---|---|---|
| **学校学生医保附带牙保**（部分学校）| 含在学费里 | 通常 1-2 次清洁 + 80% basic + 50% major |
| **学校另买 dental plan** | $20-$50/月 | 同上 |
| **工作 employer dental plan** | $5-$30/月 | 80-100% 预防 + 80% basic + 50% major |
| **个人 ACA dental** | $20-$60/月 | 同上 |
| **不买保险** | $0 | 全自费 |

**最常见牙服务价格**（自费）：

| 项目 | 自费价 | 有保险后 |
|---|---|---|
| 清洁（cleaning，6 个月一次）| $80-$200 | **$0**（多数保险全报）|
| 基础蛀牙补（filling）| $100-$300 | $30-$100（80% 报）|
| 根管（root canal）| $1,000-$2,000 | $400-$800（50% 报）|
| 牙冠（crown）| $1,000-$2,500 | $500-$1,200 |
| 拔智齿（wisdom tooth extraction）| $300-$800 / 颗 | $150-$400 |
| 牙套（braces）| $4,000-$8,000 | 部分保险 50%（≤ $1,500 上限）|

**留学生省钱建议**：

- 一般牙问题 → 学校 dental plan / 雇主 dental plan
- 大牙问题（根管 / 牙冠 / 牙套）→ **回国处理**（中国牙价 1/5）
- **大学的 Dental School 诊所**：比私人牙医便宜 30-50%（学生在导师监督下做）—— 适合不急的大手术

## 8.2 眼科 / Vision

**也是单独保险**：

- 学生医保**通常不含**——要另外买 vision plan
- 工作 employer vision：$5-$15/月
- 覆盖：1 次年度眼检 + 框架 / 镜片 / 隐形

**典型眼科服务**：

| 项目 | 自费价 | 保险后 |
|---|---|---|
| Annual Eye Exam | $80-$200 | $0-$30 |
| 眼镜（frame + lens）| $200-$600 | $50-$200 |
| 隐形眼镜（年用量）| $200-$500 | $80-$200 |
| LASIK 激光 | $4,000-$6,000 | 多数不报 |

**省钱建议**：

- 眼检：**Costco Optical / Sam's Club Optical / Walmart Vision** —— 比私人最少便宜 30%
- 眼镜框：**Zenni Optical / Warby Parker** 在线买，$30-$150
- 隐形眼镜：**1-800-Contacts / Hubble**——网购便宜
- LASIK：回国做（1/3 价）

## 8.3 心理咨询 / 心理治疗

**留学生**心理健康问题被**严重低估**——异国 / 学业 / 文化冲击 / 孤独 / 想家——是非常常见的。

**就医路径**：

| 路径 | 价格 | 适合 |
|---|---|---|
| **学校 Counseling Center** | **$0-$30**（多数 free for students）| 第一次 / 一般压力 / 焦虑 |
| **Telehealth: Talkspace / BetterHelp** | $200-$400 / 月 | 想要持续咨询 + 不见面 |
| **学校 health center 转 referral** | 看保险（通常 copay）| 需要药物治疗 |
| **私人 Therapist + 保险** | copay $30-$80 / 次 | 个体化深度治疗 |
| **私人 Psychiatrist** | copay $60-$200 / 次 | 需要药物处方 |

**关键认知**：

- **学校 counseling center 多数免费**——这是留学生圈被严重低估的资源
- **不需要“病到不能上课”才去**——常规压力 / 焦虑 / 适应问题都可以咨询
- **咨询师 / Therapist 是中性专业人士**——不会“判断”你 / 你的家人 / 你的国家

**第三篇会更详细讲心理健康路径**——这里仅是入门。

## 8.4 妇科（OB/GYN）

- 多数 HMO **妇科直接看不需 referral**（女性健康关键）
- 学校 health center 通常有妇科服务
- 包括：年度 pap smear / 避孕咨询 / pregnancy test / STD 检查
- **Title X clinics（联邦资助）**：低价 / 免费妇科服务，**Planned Parenthood 是其中一家**

# 9. 几个反直觉的看病建议

## 9.1 提前 1-2 个月预约年度体检

**Annual Physical 是 ACA 强制 100% 免费**——**每年至少做一次**：

- 提前 1-2 个月排预约（好医生排队长）
- 体检日期可以**约在生日附近**——容易记
- **空腹去**（lab 抽血需要）
- **带所有药物清单 + 病史**

## 9.2 永远 verify in-network

**这是省钱的最大单一动作**：

- 看任何医生前打保险公司客服 verify
- 大手术前 verify “all parts in-network”（医生 + 麻醉 + lab + 病理）
- 每次新医生时验证

## 9.3 ER 之外都可以“shop”价格

- Urgent Care / 影像（MRI / CT）/ Lab —— 价格在不同诊所差 5-10 倍
- **去 hospital 系统的 imaging 比去 standalone imaging center 贵 3-5 倍**
- **告诉医生“please send to standalone imaging center”** 而不是医院系统

## 9.4 学会用 patient portal

每个诊所现在都有**patient portal**（MyChart / Athena / 等）：

- 看实验室结果
- 给医生留言
- 预约 / 取消
- 看历史病历

**注册并使用**——比电话方便 10 倍。

## 9.5 永远带保险卡 + Photo ID

- 钱包里**永远有实体保险卡**
- 手机相册存正反面照片
- Apple Wallet 加电子保险卡
- ER / Urgent Care 任何时候都可能用

## 9.6 处方药跨州不能转移

如果你要搬到另一个州：

- **联系新州 PCP 重新开方**——本州医生开的处方在其它州可能不合法（特别是管制药）
- 提前 60 天计划

## 9.7 急诊判断方法：胸口测试

不确定是否要去 ER 时，问自己：

- **能等到明天吗？**（能 → 不去 ER）
- **能开车去吗？**（能 → 看 Urgent Care）
- **症状在恶化吗？**（是 + 严重 → 去 ER）

**不确定时打保险公司 24/7 nurse hotline**（卡背面）—— 免费咨询，让护士判断。
