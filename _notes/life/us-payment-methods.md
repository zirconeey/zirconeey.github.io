# 1. 这篇文章给谁看

写给从国内来美国、习惯了“出门只带手机、扫码 / 刷脸全包圆”的留学生 / 学者。在美国买东西你会很快发现——**支付方式没那么“统一”**：

- **Walmart 不收 Apple Pay / Google Pay**
- **小餐馆 / 街角咖啡店 “We don't accept American Express”**
- 加油站的牌子上同时贴着 **“credit price $4.39 / cash price $4.29"**
- BYOB 的中餐馆悄悄说 **”现金结账打 9 折“**
- Sweetgreen / Cava 这种新派沙拉店有时是 **”Cashless—only cards“**
- Costco 不收 Mastercard / Discover / Amex，**只收 Visa**

这些规则**都不是商家在为难你**——背后是同一套”信用卡手续费 + 商家利润“的经济学逻辑。这篇文章把这套逻辑讲清楚，并具体给出**几个最常见场景的实战应对**——特别是文末专门讲 **Walmart 没带实体卡 / 没现金时怎么办**（我自己刚来美国时栽过这个坑，最后只能放下整车东西出门）。

# 2. 结论先行

**留学生支付三件套**：

1. **一张实体 Visa / Mastercard 信用卡或借记卡**（90% 场景的硬通货）
2. **手机 Apple Pay / Google Pay 绑同一张卡**（70% 场景能用，越来越普及）
3. **钱包里有 $20-$40 现金**（处理停车 / 小费 / 现金折扣 / app 不工作的应急）

**几个最常见的”特殊店“**记一下：

| 店 | 收什么 | 不收什么 |
|---|---|---|
| **Walmart** | 实体卡 / Walmart Pay / 现金 / Gift Card | Apple Pay / Google Pay |
| **Costco** | **只 Visa** + Costco Shop Card + 现金 / debit | Mastercard / Discover / Amex |
| **小餐馆 / 独立小店** | Visa / MC / Discover / 现金 | 经常拒 **Amex** |
| **加油站** | 看牌子，credit / cash 价格不同 | — |
| **Sweetgreen / Cava 等 cashless** | 卡 + 移动支付 | 现金 |

**最大原因都来自同一件事**：**信用卡手续费（interchange fee）**。商家每收一笔信用卡，要付 1.5%-3.5% 的手续费——拒收特定卡 / 给现金折扣，本质都是想**少出这笔手续费**。

# 3. 一个尴尬的开头：Walmart 不收 Apple Pay

我自己来美国第一年，有一次在 Walmart 推了一整车东西到收银台——洗漱用品、冰冻食品、几桶大瓶水——大概 $80。结账时**才发现钱包没带，只带了手机**。

> ”Apple Pay?“ 我把手机贴上去。
>
> 收银员看了一眼：”We don't take Apple Pay.“
>
> ”Tap to pay?“ 我换个说法。
>
> 她直接摇头：”Sorry, only cards or Walmart Pay.“

我那时候根本不知道 Walmart Pay 是什么、也没装 Walmart 的 app。最后只能让收银员把购物车推回去 hold 着，自己开车 20 分钟回家拿卡，再回来——那一趟 Walmart 我加了 40 分钟和半箱油的成本。

这件事让我后来研究了一下”为什么 Walmart 不收 Apple Pay“——背后的逻辑很简单：

**Walmart 不想给 Apple 抽成。**

Apple Pay 是基于 NFC 的支付协议——但**用 Apple Pay 时，Apple 会从交易里抽 0.15%**（虽然不算多）。更重要的是——Apple Pay 走的是商家的现有信用卡通道，所以**信用卡手续费照样付**。**Walmart 自己开发了 Walmart Pay**（在 Walmart app 里），用这个支付时 Walmart 直接绑你的银行账户 / 借记卡，**绕开 Apple 抽成 + 选择性使用更便宜的支付通道**。

所以 Walmart 拒绝 Apple Pay 不是技术问题，是**商业策略**——它要把客户推到自己家的 Walmart Pay。

# 4. 美国支付方式光谱

下面这张表是按**接受度**从高到低排——你出门带哪几样最稳妥，看这张表：

| 支付方式 | 接受度 | 商家手续费 | 备注 |
|---|---|---|---|
| **现金** | ~99%（除 cashless 店）| 0% | 永远的兜底 |
| **Visa / Mastercard 实体卡** | ~98% | 1.5%-2.5% | 最稳的卡 |
| **Visa / Mastercard 借记卡（Debit）** | ~97% | < 1% | 商家最爱（费率最低）|
| **Discover 卡** | ~90% | 2-3% | 部分小店 / 国外不收 |
| **American Express（Amex）** | ~80% | 2.5%-3.5% | 餐馆 / 小店常拒 |
| **Apple Pay / Google Pay / Samsung Pay** | ~75% | 同等卡 + 0.15% Apple 抽成 | **Walmart / Home Depot / Lowe's 不收** |
| **Tap to Pay（NFC 卡片）** | ~70% | 同等卡 | 大型连锁多 |
| **Walmart Pay** | 仅 Walmart | 极低 | 独家 |
| **Costco Shop Card** | 仅 Costco | 0 | 礼品卡可代支付 |
| **PayPal / Venmo** | ~30%（结账场景）| 较高 | 主要在线 |
| **加密货币** | < 1% | 高 | 个别科技公司 |

**结论**：钱包里有 **Visa / Mastercard 实体卡 + Apple Pay 同卡备份 + 一些现金**，能解决 95%+ 场景。

# 5. 商家”挑卡“的真相：interchange fee 解读

理解美国支付的关键是这一节——**为什么有的卡商家爱收、有的卡商家拒收？**

## 5.1 一笔信用卡交易的钱是怎么流的

你刷一张 Visa 信用卡 $100，钱在背后实际经过的环节：

```
你（消费者）
   │
   │ $100
   ↓
商家（merchant）
   │
   │ 商家收到的不是 $100，是 $97-98
   ↓
中间被三方分掉 $2-3：
   ├─ 发卡行（Issuer，比如 Chase）：拿走最大头 ~1.5%-2%
   ├─ 收单行（Acquirer / Payment Processor）：~0.3%
   └─ 卡组织（Visa / MC / Amex）：~0.1%-0.5%
   
这三笔加起来叫 "interchange fee"
```

商家收到的 = $100 - interchange fee。

**对一家年流水 $1000 万的中型超市，2% 手续费 = $20 万/年**——这是巨大的成本压力。所以商家有强烈动机：**拒收高手续费的卡 / 鼓励低手续费的支付方式**。

## 5.2 不同卡的手续费档位（典型值）

| 卡类型 | 手续费区间 |
|---|---|
| **Debit 借记卡（受 Durbin 修正案限制）** | < 1% |
| **Visa / Mastercard 普通信用卡** | 1.5%-2.5% |
| **Visa / Mastercard 高端卡**（Chase Sapphire Reserve、Amex Platinum 等）| 2.5%-3.0% |
| **Discover** | 2.0%-2.5% |
| **American Express** | **2.5%-3.5%** |

**关键认识**：

- **Amex 手续费最高**——所以**小餐馆 / 小店最常拒 Amex**
- **借记卡手续费最低**——所以**部分商家鼓励借记**
- **高端卡（白金 / Reserve）手续费比普通卡高 0.5%**——这是为什么部分小店看到你刷一张闪闪发亮的金属卡，会”建议你换张普通卡“

## 5.3 为什么 Costco 只收 Visa？

Costco 是个极端例子——**只收 Visa 信用卡**（外加 Costco Shop Card / debit / cash）。原因：

- Costco **跟 Visa（具体是 Citi Visa）签了独家合同**——Costco 拿到极低手续费（业内传 0.4% 左右）
- 作为交换，Costco 拒收所有其他信用卡（MC / Discover / Amex）
- 历史上 Costco 是 **Amex only**（1999-2016），后来把 Amex 换成 Visa——切换的最大原因就是手续费

所以 Costco 不是”歧视别的卡“，是**它把卡战争玩成了商业策略**——用”独家“换更低的手续费，省下来的成本变成更低的会员费 + 更便宜的商品。

# 6. 几个常见的”特殊店“

下面这一节是工具书部分——下次进这些店知道带什么：

## 6.1 Walmart

| 收 | 不收 |
|---|---|
| 实体 Visa / MC / Amex / Discover | **Apple Pay / Google Pay / Samsung Pay**（绝对不收） |
| Walmart Pay（Walmart app 内）| 第三方 NFC 移动支付 |
| Walmart Gift Card / eGift Card | — |
| 现金 / 支票 | — |
| EBT（食品券）/ WIC | — |
| Affirm 分期（部分大件） | — |

**关键点**：**没带卡 = 用 Walmart Pay（详见 §8）**，不用空手而归。

## 6.2 Costco

| 收 | 不收 |
|---|---|
| **Visa 信用卡 / 借记卡**（仅此一家信用卡组织）| **Mastercard / Discover / Amex 信用卡** |
| Costco Shop Card（实体 / 电子）| — |
| Apple Pay / Google Pay（**绑 Visa 卡的**）| Apple Pay / Google Pay 绑其他卡 |
| Mastercard / Discover **借记卡**（pin debit）| — |
| 现金 / 支票 / EBT | — |

**实战**：去 Costco **必须有一张 Visa 信用卡**，或者 debit / 现金。如果你只有 Amex Gold 这种——白跑。

## 6.3 小餐馆 / 小店：经常拒 Amex

**典型场景**：街角的中餐馆、独立咖啡馆、Mom-and-pop convenience store。

- 收银台贴着 ”We accept Visa / Mastercard / Discover“ —— **没写 Amex 就是不收**
- 偶尔贴明确 ”**No American Express**“
- 部分会说 ”Amex has $20 minimum“——意思是低于 $20 不让用 Amex

**应对**：钱包里**永远有一张 Visa / Mastercard 备用**，别只带 Amex 出门。

## 6.4 加油站：”credit price“ vs ”cash price“

加油站显示牌经常写两个价格：

```
Regular Gas
  Credit: $4.39 / gal
  Cash:   $4.29 / gal
```

差价 $0.10/gal = 约 2-3% 折扣——**正好是信用卡手续费的反向折扣**。加油站给现金客户折扣，本质是把”省下来的手续费“还一部分给客户。

**实战**：

- 用现金加油 → 走 cash 价 → 算下来通常省 $1-$2/箱
- 用 debit 卡加油 → **多数加油站算”cash price“**（debit 手续费很低）
- 用信用卡加油 → 走 credit 价
- **手续费回血卡**（如 Costco 加油 + Costco 信用卡 4% 加油返现）：拿 4% 返现 - 0.10 credit price 差 = 仍然净赚

## 6.5 Cashless 店

Sweetgreen / Cava / Shake Shack 部分门店 / 一些 NYC 商业中心：

- **只收卡 + 移动支付**
- **不收现金**

历史背景：2020 年前后这种店越来越多，但**几个州（NYC、NJ、PA Philadelphia 等）通过法律禁止”完全 cashless“**——理由是**对穷人 / 无银行账户人群不公平**。所以现在 Cashless 在政治上有反弹，部分店重新接受现金。

## 6.6 BYOB / 中餐馆”现金 9 折“

部分独立餐馆（特别是中餐 / 韩餐 / 越南餐）会悄悄说：

> ”你要是给现金，我给你打 9 折。“

或者直接给小费当折扣（”现金小费就给少点没事“）。

**逻辑同上**——商家省 2-3% 手续费 + 偷偷少报税，**把省下的一部分给客户**。这游走在”合法折扣“和”逃税“之间，**作为客户不违法**——但你拿的是没写在收据上的现金折扣。

# 7. 现金折扣 vs 刷卡附加费

商家有两种”导向现金“的玩法——**法律地位不一样**：

## 7.1 Cash Discount（现金折扣）

> ”刷卡 $10，现金 $9.70（3% off for cash）“

**所有 50 州合法**——商家可以给现金客户折扣。这是上面加油站、中餐馆的玩法。

## 7.2 Credit Card Surcharge（刷卡附加费）

> ”Credit card adds 3% surcharge“

**少数州禁止**（比如**康州、马萨诸塞州**），其他州合法但有上限（通常 3-4%）。

实质上”刷卡 $10.30 / 现金 $10" 和“刷卡 $10 / 现金 $9.70" 是同一回事，但在法律上：
- 写”现金折扣“——所有州合法
- 写”刷卡附加费“——部分州违法

所以你看到的牌子大概率是”Cash Discount“措辞，**这是律师建议的写法**。

## 7.3 你应该怎么应对？

- 现金折扣 ≥ 3% 时：**用现金值得**（特别是大额消费如餐馆 $100+）
- 现金折扣 < 2% 时：**刷信用卡更值**（你的信用卡返现 / 积分通常 1-2%，加上消费保险等隐形权益）
- **小额（< $20）**：刷卡更方便，不用算

# 8. 在 Walmart 没带实体卡时的实战方案

这一节解决我开头那个尴尬场景——**没带钱包、没有现金，怎么在 Walmart 完成结账？**

## 8.1 Walmart Pay（核心方案）

**Walmart Pay 是 Walmart 自己的支付系统**，藏在 Walmart 官方 app 里，**比 Apple Pay 还方便（前提是你提前装好）**。

**前期一次性设置**（5 分钟）：

1. App Store 下载 **Walmart - Shopping & Grocery** 官方 app
2. 注册账号 / 登录（用 Walmart+ 账号也行）
3. 进 app → 右下角”Account“ → ”Wallet“ / ”Walmart Pay“
4. **绑定支付方式**：可以绑
   - 信用卡（Visa / MC / Amex / Discover 都行）
   - 借记卡
   - PayPal
   - Walmart eGift Card
5. 设置 4 位 PIN 或开 Face ID

**结账时使用**：

1. 把商品摆上传送带，结账小妹 / 自助机扫完所有商品
2. 屏幕显示 **”How would you like to pay?“** + 一个 **QR 码**
3. 打开 Walmart app → 主页”Pay“按钮（或下拉到 ”Walmart Pay“）
4. 摄像头扫描屏幕上的 QR 码
5. 输入 PIN / Face ID 确认
6. 完成——收银员屏幕会跳”Approved“

**优势**：

- ✅ 没带钱包也能付
- ✅ 自动累积 Walmart Cash / Walmart+ 返现
- ✅ 自动留收据在 app 里——退货 / 报销时不用翻纸条

## 8.2 Walmart Gift Card

如果你**之前买过 / 收到过 Walmart Gift Card**（实体或电子），可以：

1. **实体卡**：直接拿出来给收银员刷（卡背面有个 16 位号 + PIN）
2. **eGift Card**：在 Walmart app 里”Add gift card“，绑进 Walmart Pay 后用 Walmart Pay 支付
3. 也可以**直接报 gift card 号 + PIN** 给收银员（但有点慢）

**留学生小贴士**：很多信用卡的 sign-up bonus 可以兑换成 Walmart eGift Card（Chase Ultimate Rewards / Citi ThankYou Points 都支持）—— **存几张在手机里，没卡时救命**。

## 8.3 其他备选方案

| 方案 | 可行性 | 备注 |
|---|---|---|
| **Apple Pay / Google Pay** | ❌ Walmart 不支持 | 别试，浪费时间 |
| **手机里实体卡照片 / Apple Wallet 卡号** | ⚠️ 收银员通常拒绝 | 部分友好的收银员愿意手动输入卡号，但慢且少见 |
| **拿出手机银行 app 里的”虚拟卡 PAN“** | ⚠️ 同上 | Chase / Capital One 现在都有”Card not present“虚拟卡号，但 Walmart 收银台输不进去 |
| **联系朋友 / 家人远程付 Cash App / Venmo 给收银员** | ❌ 几乎所有美国超市不收 | — |
| **打车回家拿卡** | ✅ 但贵 | 除非购物车非常重 |
| **结账小妹”hold“购物车 30 分钟，自己开车回家** | ✅ Walmart 政策友好 | 我那次就是这么做的 |

**最稳建议**：刚到美国第一周下载 Walmart app + 绑卡 + 设 PIN —— 5 分钟，未来三年再没这种尴尬。

# 9. 留学生钱包随身建议

经过前面的”光谱 + 特殊店 + 现金折扣“，钱包随身配置可以这样：

```
钱包标准三件套：
  ├─ Visa / Mastercard 信用卡 1 张 → 95% 商家通用
  ├─ Visa 信用卡 / 借记卡 1 张 → Costco / 备用
  └─ 现金 $20-$40 → 停车 / 小费 / 现金折扣 / 应急

手机里默认装：
  ├─ Apple Wallet（Apple Pay）→ 70% 商家
  ├─ Walmart app（Walmart Pay）→ Walmart 没卡救命
  ├─ Costco app（数字会员卡）→ 不忘带实体会员卡
  ├─ Target Circle / CVS / 各家会员 app
  └─ Venmo / Zelle / Cash App → 朋友间转账
```

**几条延伸经验**：

- ❌ **不要钱包里只放一张 Amex 出门**——小餐馆 / 小店拒收概率高
- ❌ **不要钱包里只放一张 Discover 出门**——部分小店不收，国外更不收
- ✅ **每张大额信用卡都同时绑 Apple Pay + Google Pay**——丢实体卡时手机仍能付
- ✅ **Walmart / Target / Costco / Whole Foods 各装一个 app**——每家都有自己的优惠 / 数字会员
- ✅ **加油站买卡前看牌子 cash vs credit 价差**——超过 $0.10/gal 时刷 Costco 信用卡 + 加油站 cash price 二选一
- ✅ **应急用现金**：万一 app 崩了 / 卡被锁了，$20 cash 能让你买到回家的水和食物

刚来美国的留学生最容易在这套规则里”栽小跟头“——但跟多数美国生活的”陌生规则“一样，**栽过一次就再也不会**。希望这篇文章能让你少栽至少一次。
