# 1. 这篇文章给谁看

写给三类人：

- **刚来美国不久**——餐厅吃完饭账单后面那个空格“Tip $___”不知道填多少
- **第一次过感恩节 / 圣诞节 / Halloween**——同学说“come over”不知道要不要带东西
- **想避免文化冒犯**——见到导师 / 房东 / 同事不知道哪些话题不能聊

中国生活里**没有给小费的习惯**——服务费已经包在标价里。
美国不是——**标价是“裸价”，小费是工资的一部分**——服务员的法定最低工资可能只有 $2.13/hour（联邦 tipped wage），剩下全靠你给。

中国节日体系：**春节 / 中秋 / 国庆 / 端午**——以家庭团聚 + 法定假为主。
美国节日体系：**Thanksgiving / Christmas / Halloween / July 4th / Easter**——以宗教 + 历史 + 商业为主——和中国节日逻辑几乎完全不同。

这篇文章把**小费的算法 + 主要节日的过法 + 社交礼仪 + 文化禁忌**全讲清楚。文章以 **Penn State 留学生**为典型场景——State College 的餐厅 + Halloween 的 frat party + Thanksgiving 美国同学家做客——这些是第一年最常遇到的场景。

> ⚠️ 文化是动态的——这里讲的是“主流惯例”，具体看你所在的州 / 城市 / 社交圈。

# 2. 结论先行

**小费速查**：

| 场景 | 给多少 | 怎么算 |
|---|---|---|
| **正餐厅（sit-down）** | **18-20%** | 税前 / 税后都行（差异 $1-$2，不纠结）|
| **快餐 / Counter** | **0-10%** | 不强制，前台 tip jar 看心情 |
| **咖啡店** | **$1 或 10-15%** | 复杂订单多给 |
| **Uber / Lyft** | **15-20%** | app 内可选 0 / 15% / 20% / 25% |
| **外卖（DoorDash / UberEats）** | **15-20%** | 提前给（不给可能没人接单）|
| **理发** | **15-20%** | $50 cut 给 $10 |
| **酒店 housekeeping** | **$2-5/day** | 离店前留床头 + 写“Thank You” |
| **酒店行李员（bellhop）** | **$1-2/件** | 帮你搬行李 |
| **出租车** | **15-20%** | 现金 / 刷卡都行 |
| **送 furniture** | **$5-20/人** | 看搬运难度 |

**4 个最大坑**：

1. **快餐店 tip jar 不强制**——你不给完全 OK，但服务员看你的表情可能微妙
2. **海外信用卡**——账单的 tip 字段填了，回去对账可能多扣（手动给现金更稳）
3. **节日不带礼物**——美国同学说“casual”是真的 casual，但带瓶酒 / 一道菜永远没错
4. **Politics 话题**——美国导师 / 同事最大忌——别主动聊川普 / 拜登 / 中美关系 / 移民政策

| **Spring Break** | 3 月某周 | **放 1 周** | 旅游（Florida / Cancun 是经典）|

# 3. 小费文化（Tipping Culture）

## 3.1 为什么要给小费

美国服务业小费历史可以追溯到 19 世纪，**最重要的事实**：

**联邦 tipped minimum wage = $2.13/hour**（2026 年仍然是这个数字，30 多年没变过）。

服务员的真实收入逻辑：

- **基础工资**：$2.13/hour（联邦）— 部分州更高（比如 PA = $2.83，CA / WA / OR / NV / AK = 全额最低工资 + tip）
- **小费**：占总收入的 **70-90%**——这是真正的工资
- **如果小费没拿到联邦最低工资 $7.25/hour**——雇主必须补差——但实务中**小费几乎永远超过差额**

**含义**：你给小费**不是“额外感谢”——是付服务员工资**。不给 = 让他白干。

## 3.2 餐厅小费：标准 18-20%

**最重要的场景**——也是新手最容易出错的。

### 怎么算

| 服务质量 | 给多少 |
|---|---|
| 一般 | **15%** |
| 满意 | **18%** |
| 很好 | **20%** |
| 非常出色 | **25%+** |

**默认给 18-20%**——不要给 15%（被理解为“服务差”）。

### 算的是税前还是税后？

**两者都行**——差异只有 $1-$2。但精打细算的常见做法：

- **税前**算 20% — 我认识的多数美国人这样
- **税后**算 18-20% — 更“宽容”的版本

PA 餐饮税 = 6%（State College = 7%，含 1% 地方税）。$50 餐 + $3.50 税 = $53.50。
- 税前 20% = $10
- 税后 20% = $10.70

### 怎么操作

**刷卡场景**：

- 服务员把账单 + 信用卡终端拿来——上面有 tip 选项（15% / 18% / 20% / Custom）
- 选好后输 PIN / 签字
- **重点**：默认选项可能是**税后**——你想税前需要选 Custom

**纸账单场景**（老式）：

- 服务员先刷信用卡 → 账单回来三联——你在 **Tip** 行写小费 + **Total** 行写总金额——签字
- **不要把 Tip 行留空**——可能被理解为 0 / 也可能服务员自己填（少见但发生过）
- 想给现金小费 → 在 Tip 行写 “Cash” + Total = 餐费 → 钱放桌上

### 大桌自动小费（auto-gratuity）

**6 人以上**的桌子，餐厅经常**自动加 18-20% 小费**——账单底部会写 “Gratuity Included” 或 “Service Charge”。

**这种情况下不用再给**——但**要看清楚**——很多人没注意又多给了一份。

## 3.3 快餐 / Counter Service

**不强制小费**——但常见场景：

- 你点了餐 → 在前台等→ 自己端到桌子
- 收银员给账单时点击“Tip”屏幕——选项 15% / 18% / 20% / Custom / **No Tip**

**怎么决定**：

- 拿了就走（Subway / Chipotle）→ **0% 或 $1**——不给完全 OK
- 服务员帮你做了什么（咖啡复杂调制 / 端到桌子）→ **10-15%**
- 中等坐下 fast-casual（Panera / Sweetgreen）→ **10-15%**

**坑**：iPad 屏幕翻给你看——你站那犹豫，店员盯着你——心理压力大。学会**自然地点“No Tip”**。

## 3.4 Uber / Lyft / 外卖

### Uber / Lyft

**App 内**：行程结束后弹出小费选项 — 0 / 15% / 20% / 25% / Custom。

| 行程 | 给多少 |
|---|---|
| **5 分钟内的短途** | $1-$2 (15%) |
| **10-30 分钟正常行程** | **15-20%** |
| **超长 / airport pickup** | **20-25%** + 帮搬行李额外 $5 |
| **司机超 nice / 帮搬箱** | **25%+** |

**好不好评分**：5 stars 是默认——给 4 stars 司机收入会受影响——慎给。

### DoorDash / UberEats / Grubhub

**关键差异**：**小费在下单时就给**——不是送达后。

为什么？因为 driver **看到 tip 才决定要不要接**——你给 $2 / 15 mile 远 → 没人接 → 订单挂在系统里 30 分钟。

| 距离 | 给多少 |
|---|---|
| **2 mile 以内** | $3-$5 |
| **2-5 mile** | $5-$8 或 15-20% |
| **5+ mile** | $8-$15 或 20%+ |

**坑**：很多人给 10% / $2 → 等了 1 小时不送 → 食物冷了。

## 3.5 理发 / 美容

| 服务 | 给多少 |
|---|---|
| **男士理发** ($30) | $5-$8 (15-20%) |
| **女士剪发 + 染发** ($150) | $25-$30 (18-20%) |
| **修指甲** ($25) | $5 (20%) |
| **按摩 1 hour** ($80) | $15-$20 (18-25%) |
| **染色 + 烫发 + 复杂** ($300+) | **25%**——这是手艺活 |

**给现金更受欢迎**——美容院的 tip 走信用卡有些会被克扣 / 延迟到工资。

## 3.6 酒店

| 服务 | 给多少 |
|---|---|
| **Bellhop**（行李员） | **$1-2/件** |
| **Housekeeping**（房间清洁） | **$2-5/day**（每天留床头）|
| **Concierge**（前台咨询）| 帮预约难订餐厅 → $5-$20 |
| **Valet**（停车） | $2-5（取车时）|
| **客房服务**（送餐到房间）| 多数酒店账单已含 18% gratuity——不需再给 |

**Housekeeping 怎么放**：

- **每天留**——不是离店时一笔——因为可能不是同一个清洁工
- **桌子上写 “Thank you”**+钞票放枕头 / 床头
- **不要塞床下 / 抽屉**——清洁工不会拿别人的钱

## 3.7 其他场景

| 场景 | 给多少 |
|---|---|
| **出租车 / Limo** | 15-20% |
| **送 furniture / 大件搬运** | $5-20/人 |
| **搬家工人**（一整天） | $20-40/人 |
| **披萨外卖**（餐厅自己的）| $3-5 或 10% |
| **Doorman / 保安** | 节日给 $25-100（看公寓档次）|
| **狗 grooming** | 15-20% |

## 4.1 反直觉的几个常识

- **税前 vs 税后**——多数本地人懒得算，**直接看 total 翻 5 倍**（20%）—— 简单粗暴
- **Take-out 不需要给小费**——但有些餐厅 iPad 还是会问——可以选 No Tip
- **小费走 cash 还是 card**——card 最方便 / cash 服务员喜欢（偷税）
- **海外信用卡刷不出 tip 的情况**——美国餐厅极少见 / 有发生过 → 准备 cash 更稳
- **服务很差怎么办**——可以给 10%（明显信号）/ 完全不给——但**最好告诉经理为什么**

# 5. 主要节日（学生最重要的 5 个）

## 5.1 Thanksgiving（感恩节）—— 11 月第 4 个周四

**最重要的家庭节日**——比 Christmas 还重要的“国庆级”节日。

### 历史背景

1621 年清教徒来到 Plymouth（马萨诸塞）感谢印第安人帮助度过第一个冬天的庆祝活动。1863 年林肯定为联邦节日。

### 学生场景

- **学校放 5-7 天**（周三到下周日）—— Penn State 一般放 1 周
- **机场是一年最忙**——Sunday after Thanksgiving 是全美航班量最大的一天（注意飞回来的票要早订）
- **Black Friday + Cyber Monday**——最大的购物季 → 见[美国支付潜规则]({% post_url 2026-04-28-us-payment-methods %})

### 怎么过

**3 种典型方式**：

1. **回家 / 旅游**——美国同学回家庭，国际生可以借机便宜出游（但机票贵）
2. **美国同学家蹭 Thanksgiving Dinner**——这是**国际生最值得做的事**
3. **自己 / 朋友 cook 一餐**——华人留学生群里常组织“中式 Thanksgiving”

### 蹭饭礼仪

如果美国同学说 “Come over for Thanksgiving”：

- **带一道菜或饮料**——不要空手——bottle of wine / 中式甜点 / 苹果派 / Sparkling cider 都行
- **下午 1-3 点开吃**——不是晚饭——典型的 turkey dinner 是中午到下午
- **不要太早走**——典型节奏：吃饭 → 看 NFL 比赛 → pie + coffee → 7-8 点散
- **写 thank-you note**——节后给个 message 道谢——加分

### 必备食物

| 食物 | 解释 |
|---|---|
| **Turkey**（火鸡） | 整只烤——但说实话很多美国人也觉得干 |
| **Stuffing**（填料） | 面包 + 香料的拌料 |
| **Mashed potatoes** | 土豆泥 |
| **Cranberry sauce** | 蔓越莓酱（解腻）|
| **Pumpkin pie** | 南瓜派（甜点）|
| **Green bean casserole** | 青豆配菜 |

## 5.2 Christmas + New Year —— 12.25 + 1.1

**学校放 2-3 周**——**国际生回国 / 旅游的最大窗口**。

### Christmas 礼仪

**送礼文化**：

- **Secret Santa**——朋友 / 同事抽签互送——预算 $20-$50
- **家庭礼物**——美国家庭有送礼传统——量大 + 25 早上 unwrap
- **国际生**：导师 / 实验室 lab 可能搞——按当地氛围给 $20-$30 的小东西

**Christmas Card**——美国传统是寄纸质卡片——多数留学生跳过 / 同行可以发 e-card。

### 实际操作

| 行动 | 时机 |
|---|---|
| **机票早订**（回国 / 旅游）| 9-10 月就要订 — 12 月票翻倍 |
| **办公室 / 实验室关门** | 12.20-1.5 多数大学 / 公司停摆 |
| **超市抢购** | 12.23-24 是抢购最高峰 — 提前一周采购 |
| **Boxing Day 12.26** | 加拿大 / 英国大型 sale — 美国不流行（黑五已结束）|

### New Year（新年）

**比 Christmas 简单**：

- **12.31 跨年**——朋友家 / 餐厅 / Times Square ball drop（但纽约很冷）
- **Resolution**（新年决心）——typical：减肥 / 戒酒 / 读书——常被嘲笑
- **1.1 多数地方关门**——准备好食物

## 5.3 Halloween —— 10.31

**装扮 + party + 糖果**——年轻人 + 学生**很爱的节日**。

### 历史背景

凯尔特节日 → 基督教 All Saints Eve → 美国变成现在的 candy + costume + horror movies 文化。

### 学生场景

| 场景 | 怎么参与 |
|---|---|
| **Costume party** | 朋友家 / frat house / 酒吧——typical 提前 1-2 周准备装扮 |
| **Trick-or-Treat** | 小朋友拿着南瓜桶挨家敲门要糖——你**不参与**，但**要准备糖**给来你公寓 / 房子门口的小孩 |
| **Pumpkin carving** | 雕南瓜灯——超市 $5-$10 一个大南瓜 |
| **Haunted house** | 鬼屋——大学城常有 |

### 装扮文化

- **Costume**——可以原创 / 经典（吸血鬼 / 巫婆）/ Pop culture（最近的电影 / 游戏角色）
- **Group costume**——朋友 / couple 一起搞主题更有趣
- **不要乱选 cultural**——印第安头饰 / 黑人 minstrel / 日本和服 cosplay = **culture appropriation**——会被骂

### 给糖

如果你住房子 / 一楼公寓——**可能有 trick-or-treaters**：

- 18:00-21:00 是高峰
- 准备 1-2 袋小糖（$10）→ 没人来你自己吃 → 给完关灯
- **如果不想给**——直接关灯——大家都懂

## 5.4 July 4th —— Independence Day（独立日）

**美国国庆**——1776 年独立宣言签署。

### 学生场景

- **暑假中**——多数学生在 family / 实习
- **典型活动**：BBQ + 烟花 + 国旗 + 红白蓝 outfit
- **Fireworks**——城市 / 公园放——免费看
- **国际生**——凑热闹 OK，不用特别庆祝

### 注意事项

- **私人放烟花在多数州非法**——别学美国邻居
- **酒驾**——独立日是全年酒驾最多的一天——警察查得严
- **Pet 怕烟花**——有狗的话提前关好

## 5.5 Spring Break —— 3 月某周（每个大学不同）

**1 周春假**——**学生最爱旅游窗口**。

### 经典去处

| 目的地 | 价位 |
|---|---|
| **Florida**（Miami / Orlando） | 飞机 $200-$500 + 住 $1500-$3000 |
| **Cancun（墨西哥）** | $800-$1500 套餐——最经典 spring break |
| **Las Vegas** | $300-$1000 |
| **加州**（LA / SF） | $500-$1500 |
| **回家** | 国际生回国少（机票贵 + 时间紧）|

### 为什么 Spring Break 这么火

美国大学生 18-22 岁——**一年一次合法借口去佛罗里达喝酒**——是文化符号。

国际生：可以借机去美国其他州 / 加拿大旅游——省钱方式是和朋友一起开车 + airbnb。

# 6. 其他次要但重要的节日

## 6.1 联邦假日（学校 / 公司放）

| 节日 | 日期 | 背景 |
|---|---|---|

# 7. 社交礼仪（Etiquette）

## 7.1 见面礼仪

### Handshake（握手）

- **正式场合**（导师 / 教授 / 面试 / 商业 meeting）——**握手是默认**
- **力度**：firm but not crushing——比中国略 firm
- **眼神**：握手时**要直视对方眼睛**——别看地上 / 别飘走
- **持续 1-2 秒**——不要握太久

### Hug / Cheek Kiss

- **熟朋友 / 朋友介绍朋友**——hug 在年轻人中常见
- **法语 / 西语社区**——cheek kiss——但美国主流不用
- **first time meet**——握手就够了——hug 太亲密

### Names（名字）

- **First name basis**——美国主流是直接叫 first name——包括对教授
- **Professor X**——更正式 / 老一辈教授偏好
- **PSU 实务**：第一次邮件用 “Dr. [Last Name]”——他回复用 first name → 之后跟随
- **Asian 学生坑**：很多人觉得“直呼老板名字不礼貌”——其实不叫反而显得疏远

## 7.2 Small Talk（小聊天）

**美国生活的润滑剂**——电梯里 / 排队时 / 收银员 → 都会闲聊。

### 安全话题

- **天气**——永远的安全话题
- **周末计划 / 假期**——“How was your weekend?” “Got any plans?”
- **Sports**——尤其是当地球队（Penn State football / Pittsburgh Steelers）
- **TV / Movies**——最近热门剧
- **Food**——餐厅推荐 / 你做了什么菜
- **宠物**——狗 / 猫永远的话题

### 话题模板

| 你说 | 他回 |
|---|---|
| “How's it going?” | “Pretty good, you?” |
| “How was your weekend?” | “Good, just relaxed. You?” |
| “Did you watch the game last night?” | “Yeah, terrible call at the end.” |
| “Any plans for the holidays?” | “Going home to see family.” |
| “How do you like the weather?” | “Way too cold for April.” |

### 中国学生坑

- **回答只有 “Yes”**——美国 small talk 期待 **2-3 句的回应** + **反问**
- **沉默不舒服**——尤其电梯——说一句简单的 hi / nice weather 就够了
- **太深入**——人家只是闲聊——不要 5 分钟讲你的生活困扰

## 7.3 Eye Contact

**直视对方眼睛**——美国文化的核心。

| 场景 | 眼神 |
|---|---|
| **对话** | 70-80% 时间直视对方 |
| **演讲 / presentation** | 扫视全场——不要只看一个人 / 一片屏幕 |
| **面试 / job talk** | 必须 firm eye contact |
| **餐厅 / 服务员** | 点单时看服务员 / 不是 menu |

**中国 / 东亚文化**：眼神接触少 = 谦虚。
**美国文化**：眼神接触少 = **不诚信 / 心虚 / 不专业**。

→ **强迫自己练习**——尤其面试 / 导师面谈。

## 7.4 Personal Space

**美国 personal space ≈ 60-100cm**（中国 ≈ 30-50cm）。

| 场景 | 距离 |
|---|---|
| **排队** | **保持 1m**——不要紧贴 |
| **电梯 / 地铁拥挤** | OK 暂时贴近——但避免眼神 / 不交谈 |
| **聊天** | 站 60-100cm 远 |
| **拍肩膀 / 摸手臂** | 熟人 OK / 陌生人不行 |

**中国学生坑**：

- 排队站太近——美国人会本能后退
- 拍同事 / 教授肩膀——不熟的话不行
- 在窄过道遇到人——侧身让 + 说 “Excuse me”

## 7.5 餐厅礼仪

| 场景 | 怎么做 |
|---|---|
| **进店等位** | 找 host / 不要自己坐——除非店小写“Seat Yourself” |
| **点餐** | 服务员问 “Are you ready?” — 没决定说 “Need a few more minutes” |
| **要服务员** | **眼神接触** + 招手——不要喊 / 不要打响指 |
| **Refill 续杯** | 美国 soda / coffee 多数 free refill — 服务员主动来 |
| **Doggy bag** | 吃不完打包——说 “Could I get a to-go box?”——**没人会觉得失礼** |
| **结账** | 服务员把账单放桌上 → 你放卡 / 现金 → 服务员拿走刷 → 回来签 |
| **签字** | 信用卡终端 / 纸账单 — 写好 tip + total + 签名 |

**坑**：

- **拍照 / 刷手机太久**——美国人觉得要专心吃饭 + 聊天
- **大声说话**——美国餐厅普遍比中国安静
- **吃饭吧唧声**——美国文化中**特别忌讳**——要嘴巴闭着嚼

## 7.6 公共场所

| 场景 | 注意 |
|---|---|
| **电梯** | 站立面向门 / 进门排队 / 让人先出再进 |
| **地铁 / 公交** | 不要打电话大声 / 不要吃带味食物 |
| **图书馆** | 静音 / 谈话出去——美国图书馆比中国安静 |
| **健身房** | 用完器械擦汗 / 重物归位 / 不要盯陌生人 |
| **公园 / 步道** | 走路靠右 / 跑步 / 骑车从左过——说 “On your left” |

# 8. 文化禁忌（Don't Talk About）

**美国主流文化**——**有些话题就是不聊**——尤其 work / 不熟的人。

## 8.1 5 大禁忌话题

### 1. Politics（政治）

**最大禁忌**——尤其川普 / 拜登 / 民主党 / 共和党。

- **导师 / 同事**——可能任何政治立场——别假设
- **2020 后美国分裂严重**——同事吵起来 = 工作关系完蛋
- **国际生处境**——你说 “Trump is bad” / “Biden is bad”——同事可能强烈反对

**怎么做**：

- 别人主动说政治 → “I try to stay out of politics” / “I don't follow it much”
- 别人坚持要听你意见 → “I'm still learning about US politics”
- 朋友圈 / 社交媒体——别转发美国政治内容（影响签证 / 工作机会）

### 2. Race / 种族

**极敏感**——尤其黑人 / 亚裔 / 拉丁裔议题。

- **不要**讲“黑人都怎么怎么样”的概括
- **不要**说“China is better than America at X”——容易被理解为种族对立
- **不要**评论别人的肤色 / 头发 / 长相

**N-word 一律不说**——即使在歌词里 / 引用——禁忌词中的禁忌。

### 3. Religion（宗教）

- **不要主动问** “Do you believe in God?” / “Are you Christian?”
- **如果别人说**——尊重 → “That's interesting” / 不评价
- **同事 / 教授** religious 程度差异巨大——保守 Christian 到无神论都有

### 4. Salary / 收入

**美国职场禁忌**——不要问 / 不要主动说。

- **不要问** “How much do you make?”
- **不要问 student colleague** “What's your stipend?”——虽然 PhD 间普遍知道
- **房租 / 房价 OK 聊**——但具体 salary 不

**PSU 例外**：州立大学**所有员工工资公开**（PennWatch / state employee salaries）—— 你能查到导师工资 — 但**不要主动提**。

### 5. Age / Weight / 长相

| 不能问 | 例子 |
|---|---|
| **年龄**（尤其女性 / 30+） | “How old are you?” |
| **体重** | “Have you lost weight?”（除非很熟）|
| **婚姻状态** | “Are you married?” |
| **小孩计划** | “When are you having kids?” |
| **性取向** | 不要问——别人愿意说会主动说 |

## 8.2 美国职场敏感细节

- **Workplace harassment**——美国对职场骚扰 zero tolerance —— 不开任何 sexual joke / 不评论同事外貌
- **Diversity / Inclusion**——可以谈 / 但慎言——不熟的人面前别评论
- **#MeToo**——任何不当的肢体接触 / 言语都可能严重——比中国职场严格 100 倍
- **Title IX**——大学专门的性骚扰处理流程——一个不当邮件可能让 PhD 学生开除

## 8.3 国际生特殊场景

- **签证 / 移民话题**——不要在朋友圈 / X 上讨论自己签证 / 移民策略——可能被翻
- **中国敏感话题**——不要主动评论 / 别人问就 stay neutral —— “It's complex”
- **学校政治**——不要在系里 mailing list 评论选举 / 抗议活动

# 9. 几个反直觉的常识

1. **小费不是“额外”——是工资**——不给 = 让人白干。这是为什么“我吃了 $50 餐给 $5 小费”在中国看似慷慨，在美国是侮辱。
2. **Thanksgiving 比 Christmas 更“美国”**——多数美国人认为 Thanksgiving 是最家庭的节日。
3. **Halloween 是大学生 + 小孩节**——中年人多数只是给糖 + 看小孩 trick-or-treat。
4. **“How are you?” 不是真问**——typical 回答 “Good, you?”——不要真讲你今天怎么样。
5. **First name basis 是默认**——叫教授 Dr. X 一开始 OK / 他回 first name 之后用 first name —— 不叫反而疏远。
6. **Sports 是终极 small talk**——不会聊运动 → 损失 50% 的同事 small talk 题材——至少懂当地球队赢没赢。
7. **Politics 真的不能聊**——以前可以“agree to disagree”——2020 后不行——同事 / 邻居因此撕破脸是真的。
8. **餐厅“Service Charge” 不全是 tip**——有些是给房东 / 经理——这种情况你**还是要给小费**——看清楚 receipt。
9. **送礼“It's the thought that counts”**——美国节日礼物**不在贵**——一瓶 $15 红酒 + 手写卡片 > $200 没卡片的礼物。
10. **打喷嚏说 “Bless you”**——别人打喷嚏你说“Bless you”是默认礼仪——你打喷嚏不要说 sorry —— 说“Excuse me”。

- **去 office hour**——和教授 + 同学非正式聊天
- **加 club / volunteer**——Penn State 有 1000+ 学生组织
- **接受 invitations**——同学说“come over”就去——一开始可能不舒服 / 但越拒绝越孤立
