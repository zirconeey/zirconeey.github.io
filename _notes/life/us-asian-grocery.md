# 1. 这篇文章给谁看

写给**想自己做中餐**的中国留学生 / 学者——这是**中国留学生在美国生活的“省钱主战场”**。一个简单的事实：

> **同一瓶老抽，亚超 $3.99，Walmart $9.99。**
>
> **同一斤干香菇，亚超 $5，Whole Foods $25。**
>
> **同一磅带皮五花肉，亚超 $4.50，Walmart 根本没有（要专门订特殊切肉）。**

**月伙食费 $400 vs $600 的差距，多数时候就是“会不会去亚超”的差距**。

这是[美国超市三部曲](/life/us-grocery-stores)的第二篇——第一篇讲了八大美式超市的整体地图，**这一篇专门讲亚洲超市**：主流品牌、地区分布、**30+ 项中餐食材的亚超 vs 美超价格对照表**、亚超才有的特色品类、几个反直觉的购物提示。第三篇讲采购实战（场景化配置 + 配送 + 省钱招数）。

# 2. 结论先行

**去亚超的省钱效应**（按月伙食费 $400 的留学生估算）：

| 习惯 | 月伙食费 | 占比来源 |
|---|---|---|
| **完全不去亚超**（中餐食材都在 Walmart / Whole Foods 买）| **$550-$650** | 调料 / 干货 / 肉类全是美超价 |
| **偶尔去亚超**（一月一次，只买调料）| **$450-$520** | 调料省了 50%，其它仍美超价 |
| **常去亚超**（每 2 周一次，调料 + 干货 + 肉 + 蔬菜）| **$350-$420** | **省最多** |

**结论**：**坚持去亚超 = 每月省 $150-$200 = 每年省 $1,800-$2,400** —— 这是中国留学生**最大的“日常省钱机会”**。

**最佳采购策略**：

- **每 2-3 周**去一次亚超，一次买 $50-$100
- 调料 + 干货可以**囤几个月**
- **新鲜蔬菜 + 肉类**当周买完
- 距离远的话——**Weee! / Yamibuy 在线送货**，亚超在你家门口

# 3. 美国主流亚洲超市品牌地图

美国的“亚洲超市”实际是个**大杂烩**——韩国超市卖中餐食材、华人超市卖韩餐 / 越南餐食材，**多数时候你不需要纠结“这是哪国的超市”**。但每家有自己的强项：

## 3.1 H Mart（韩国系，最普及）

- **门店**：全美 ~100 家（NJ 总部，扩张最快）
- **覆盖**：东北部 + 加州 + 德州 + 中部主要城市
- **强项**：
    - **最干净 / 最现代**——装修接近 Whole Foods
    - **韩餐食材最全**（泡菜种类多、韩式调料、韩国冷冻食品）
    - **海鲜柜台**——活鱼 / 活虾 / 切片三文鱼 / 鳕鱼
    - **熟食吧台**——韩式饭团 / 紫菜包饭 / 烤肉 / 韩式炸鸡
    - **烘焙区**——Paris Baguette 或 Tous Les Jours 进店
    - **二楼经常是 food court**——韩餐 / 越南餐 / 日本餐 / 中餐都有
- **弱项**：
    - 中餐食材覆盖不如华系超市深
    - 部分品牌只有韩国货（缺老干妈 / 王老吉 / 香飘飘 等中国零食）
- **氛围**：现代、明亮、装修好——**带朋友 / 约会去也不丢人**

## 3.2 99 Ranch Market（大华系，西海岸 + 德州）

- **门店**：西海岸 + 德州 + 东北部分（NY / NJ / MA），全美 ~60 家
- **强项**：
    - **中餐食材覆盖最全**——华人系超市（Tawa Supermarket Inc. 旗下）
    - **海鲜活的多**——活螃蟹 / 活鱼 / 活蟹 / 活贝（特别是周末早上）
    - **猪肉部位齐全**：五花、排骨、肘子、猪头肉、肥肠等
    - **中式调料的“长尾”**：豆瓣酱、四川泡椒、广式蚝油、福建鱼丸、潮汕牛肉丸
    - **中国零食的进货量大**：旺仔、康师傅、麻辣花生、维他柠檬茶
- **弱项**：
    - 装修比 H Mart 老旧
    - 部分门店人多 + 停车位紧
    - 东岸覆盖不全（NY 有几家但 PA / FL 几乎没有）

## 3.3 大华超市 / 99 Ranch 的“姐妹品牌”

- **大华超市（Tawa Supermarket）**：99 Ranch 的同集团品牌，**主要在西海岸**——和 99 Ranch 几乎是同一家
- **新加坡超市 / Asian Mart**：小型独立亚超，**每个大学城附近都有 1-2 家**
- 这些“小亚超”——**SKU 比 99 Ranch 少**，但**价格不一定贵**，离学校近，方便快进快出

## 3.4 Great Wall Supermarket（东北 + 中部）

- **门店**：~10 家，主要在 DC / VA / MD / GA / IL / TX
- **强项**：
    - **东岸最大的华人超市**之一
    - 中餐食材深度仅次于 99 Ranch
- **弱项**：覆盖区域窄（你不在东岸 metropolitan 区域可能没有）

## 3.5 Fubonn（西北 Portland）

- **门店**：Portland 1 家旗舰店——美国西北最大亚超
- **强项**：海鲜 + 中餐食材
- **弱项**：覆盖窄（仅 Portland）

## 3.6 区域性 / 独立亚超（每个大学城都有）

不一定是大牌，但你所在小镇 / 大学城**几乎都有 1-2 家独立亚超**——可能叫 “Asian Market”、“Oriental Mart”、“China Town Market”。SKU 比连锁少，但**通常离学校近 + 老板说中文**。

**找当地亚超**：[Yelp 搜 “Asian supermarket”](https://www.yelp.com/) 或 [Google Maps 搜 “Asian grocery”]。

## 3.7 在线亚超：Weee! / Yamibuy

**Weee!（鲜) 和 Yamibuy（亚米) 是在线亚超**：

- **Weee!**：核心是**鲜活的亚洲食材 + 冷链配送**——主打蔬菜 / 鲜肉 / 海鲜 / 冷冻食品。覆盖**全美主要都市**，多数地区**次日达 + 满 $35 免运费**
- **Yamibuy（亚米)**：核心是**中国零食 + 调料 + 美妆 + 礼物**——多是干货和包装食品，**全美 1-3 天 USPS / FedEx 送货**

**留学生使用建议**：

- 你所在地**有线下亚超**：以线下为主 + Weee! 偶尔补
- 你所在地**没有线下亚超**（小镇 / 偏远）：**Weee! + Yamibuy 是救命**——把“想吃中餐但没原料”的难题解决了

# 5. 亚超的“特色品类”——美超完全没有的东西

下面这些**只有亚超有，美超基本不卖**：

## 5.1 中式肉类的“长尾”

- **猪肘 / 猪头 / 猪心 / 猪肝 / 猪舌**——美超切肉只切常规（chops、loin、tenderloin），中式肉类细分**只有亚超 / 韩超 / specialty butcher 切**
- **整鸡（whole chicken with head + feet + entrails）**——美超的 whole chicken 是**去内脏 + 去头去脚**的。亚超有“完整鸡”，可炖鸡汤
- **鸭子 / 整鸭**——美超几乎没有；亚超有冷冻 whole duck 和北京烤鸭半成品

## 5.2 活的水产

- **活鱼 / 活虾 / 活蟹 / 活蛤** —— 美式超市基本只卖冰冻 / 切片；亚超有鱼缸现捞
- **活的螃蟹（蓝蟹 / 大闸蟹）**——只在亚超

## 5.3 特殊豆制品

- **板豆腐 / 嫩豆腐 / 豆腐皮 / 豆腐丝 / 千张** —— 美超的 tofu 只有“firm / extra firm / silken”——细分品类只有亚超
- **豆腐脑 / 豆花**——只有亚超
- **腐竹 / 油豆皮**——只有亚超

## 5.4 各种“奇奇怪怪的”中式食材

- **莲藕 / 山药（fresh）**——多数美超没有
- **百合 / 莲子 / 银耳**——美超不卖
- **粽叶（zongzi leaves）**——只有亚超（端午前最多）
- **紫米 / 黑米 / 糙米的中国版**——美超只有“普通糙米”

## 5.5 调料的“长尾”

- **腐乳 / 红方腐乳**——美超不卖
- **酱油（各种特殊酱油）**——白酱油 / 蘑菇老抽 / 海鲜老抽
- **泡椒 / 泡菜（中式）**——美超不卖（韩式 kimchi 美超有）
- **腊肠 / 香肠 / 腊肉**——美超不卖

# 6. 亚超购物的反直觉提示

## 6.1 周末早上去最好

- **新鲜活鱼 / 活蟹**早上刚到货
- 周末蔬菜补货勤
- 中午前去人少
- 周日下午许多亚超关得早（**部分店周日 18:00 关门**）

## 6.2 价格不太好讨价还价，但常有 weekly ad

- 多数亚超有**纸质 / app 版 weekly ad**
- 周二 / 周三特价最多
- **整箱买大米 / 整桶买调料常有 5-10% 折扣**

## 6.3 收银员中文 OK

- **大多数亚超收银员讲中文**——你可以中文沟通
- 找特殊食材时（“哪里有山药？”）直接问中文
- **但礼貌很重要**：中文圈口碑传得快

## 6.4 现金 / 微信支付偶尔可用

- **多数大型亚超**（H Mart / 99 Ranch）：信用卡 / debit / Apple Pay 都收
- **部分小型独立亚超**：现金优先，部分接受微信 / 支付宝（微信扫码付）
- **小型店收信用卡有时有 $5-$10 起付限制**——带点现金保险

## 6.5 “本地化品牌”VS“中国进口”

- 不少调料 / 干货是 **“美国本地华人工厂”** 而非中国直接进口——比如 Three Ladies 大米（加州米）是“中式品牌但美国种”
- **Pearl River Bridge / Lee Kum Kee / 海天 / 老干妈**——这些是中国进口，**最贴近国内口味**
- 对挑剔国内口味的人，**看清是否“Made in China”** 标签

## 6.6 大袋装的隐藏成本：储存

- 25 lb 茉莉米——**米虫问题严重**——必须**密封桶 / 大塑料盒 + 几片月桂叶 + 干燥剂**
- 5 lb 红枣 / 杂粮 / 干香菇——**密封罐 + 冷冻保存**
- 这是为什么有些人宁愿买小包装多次跑亚超

## 6.7 周边小镇有“农庄”卖中国蔬菜

- 部分有华人聚居的城市周边，有**Asian farmers**——周末 farmer's market 有现采的小白菜 / 茼蒿 / 苦瓜
- **远比亚超新鲜 + 价格便宜**
- 中文 / 英文搜 “[你城市] Asian farmers market”
