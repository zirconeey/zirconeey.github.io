# 1. 问题

电热水壶用一段时间，壶底就出现一层白色或黄白色硬壳——水垢。倒水的时候有时还会跟着掉碎片到杯子里。北方水更明显，烧两三周就一层。这玩意喝下去会不会吃出肾结石？要不要每次烧水都先把壶底刷干净？长在底部影响烧水吗？除水垢有没有不伤壶的办法？

# 2. 结论先行

**水垢的主要成分是碳酸钙（CaCO₃）+ 碳酸镁（MgCO₃）**——本质和**蛋壳、贝壳、石灰岩**是同一种东西。

- **能喝**：吃下去到胃里被胃酸（pH 1-2）重新溶解为钙离子和镁离子，**和喝高钙牛奶或吃钙片差不多**
- **不致结石**：膳食钙摄入量和肾结石**反相关**——这是流行病学共识，水垢锅这一口实在背不动
- **不致癌**：没有任何流行病学证据；硬水地区心血管疾病死亡率反而略低
- **真正的问题在壶**：水垢导热系数比金属低 100 倍，会让加热效率下降、温控失灵、底盘过热缩短壶的寿命

去除方法：**白醋或柠檬酸 + 水煮沸 + 浸泡 30 分钟** 就能除干净，原理是酸 + 碳酸盐生成可溶性盐 + CO₂。**不要用钢丝球硬刮**（伤涂层）；**不要用小苏打**（碱性，对碳酸盐无效）。

# 3. 科学原理

## 3.1 水垢是怎么来的

自来水里含**碳酸氢钙**（Ca(HCO₃)₂）和**碳酸氢镁**（Mg(HCO₃)₂）——这两种盐溶解度不错，所以平时水是清的。但加热到 60 ℃ 以上时，它们会**热分解**：

> Ca(HCO₃)₂ → CaCO₃↓ + H₂O + CO₂↑

碳酸钙难溶，析出后沉淀在壶底。这就是水垢的来源。这种因为加热就能去除的硬度叫 **暂时硬度**（碳酸盐硬度）；另一类靠加热去不掉的（硫酸钙、氯化钙）叫 **永久硬度**。

> 顺便：烧开水时听到的 「轰隆」 声其实是碳酸氢钙分解产生的 CO₂ 气泡和水沸腾混在一起的声音，并不是水真的快沸腾了。

水垢的颜色由水源杂质决定：

- **白色**：纯碳酸钙 / 镁，最常见
- **黄褐色**：含铁锈或腐殖质（老旧水管或地下水水源）
- **绿色**：少见，可能含铜（铜管腐蚀）

## 3.3 喝水垢到底有没有事？

把这件事拆开三个问题：

**问题 1：喝下去会怎样？**

水垢碎片进胃后，胃酸（pH 1-2，主要是盐酸）会让它**完全重新溶解**：

> CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

也就是说，水垢在胃里 **= 钙离子 + 镁离子 + 微量打嗝**。这和你吃钙片（碳酸钙是最常见的钙片成分，比如 Caltrate / 钙尔奇）是**完全一样的化学反应**。一片 600 mg 钙的钙片，含碳酸钙约 1500 mg，比一壶水里的水垢多得多。

**问题 2：钙镁本身是好是坏？**

钙、镁是人体必需的常量元素。每日推荐摄入量（成人）：

- **钙**：1000-1200 mg
- **镁**：320-420 mg

一杯 250 ml 硬水（300 mg/L 硬度）大约提供 **75 mg 钙 + 18 mg 镁**——是有意义的补充，远没到任何“过量”标准（钙的 UL 上限是 2000-2500 mg/天）。

**问题 3：水垢和肾结石有关系吗？**

这是流传最广的误解。事实正好相反：

- 哈佛大学护士健康研究（NHS）等几个大型流行病学研究都发现：**膳食钙摄入量越高，肾结石发病率越低**（Curhan 1993, 1997）
- 机制：肠道里的钙会和食物里的草酸结合形成不溶的草酸钙，**直接随大便排出**，反而**减少了草酸吸收**进入血液 / 尿液
- 真正引起肾结石的是：**草酸 / 钠 / 动物蛋白摄入过量 + 水分摄入不足**——核心是水喝得太少

> **唯一例外**：医学诊断为「高钙尿症」或正在服用某些药物（如维生素 D 大剂量补充）的人，需要遵医嘱限钙；**普通健康人不需要担心水垢**。

**问题 4：硬水和健康长期相关性？**

WHO 1980 年代以来多次综述表明：

- 硬水地区**心血管病死亡率略低**（镁的保护作用）
- 没有证据显示硬水致癌、致结石、致老年痴呆
- 唯一关联较稳的是：**硬水可能加重湿疹 / 皮炎**（外用，洗澡水接触皮肤）；不影响饮用安全

**真正要警惕的是水源本身的问题**：

- 如果水源含**铅、镉、砷、镍**，水垢会**富集这些重金属**——这就是为什么老房子（< 1990 年的铅管）或某些工业污染区域要警惕。但这种情况下问题不在水垢，在源水
- 如果**反复烧同一壶水**——并不会“产生千滚水致癌物”（这是网传谣言），但水分蒸发后矿物浓缩、可能浓度升高。建议每次烧之前换新水

## 3.4 水垢对烧水设备的影响

这才是真正应该关心的问题。

**导热效率塌方**：

| 材料 | 导热系数 W/(m·K) |
|---|---|
| 不锈钢 | ~16 |
| 铜 | ~400 |
| **碳酸钙（水垢）** | **~2.2** |

水垢的导热是金属的 1/8 到 1/200。在发热盘上结一层水垢 = **隔热层**。后果：

- 发热盘温度需要更高才能把热量传给水 → 局部过热 → 加速元件老化
- 烧开同样多水的能耗增加（实测数据：1 mm 水垢 ≈ 多耗 5-10% 能量；3 mm 水垢可能 25%）
- 温控传感器（很多在底盘附近）失灵——明明水还没沸腾，传感器读取的是水垢隔热后那一面的温度，可能提前断电

**安全风险**：

- 严重时发热盘可能烧穿（壶底冒烟、跳闸）
- 水垢下面的不锈钢可能被腐蚀（碳酸根 + 高温 → 微小腐蚀坑）

**水垢碎片喝起来不舒服**：

- 牙齿咬到硬颗粒
- 倒水时漂浮颗粒——视觉不悦
- 不会有害但会给你“水变脏”的心理感受

# 4. 实践建议

## 4.1 日常使用习惯

- **每次烧之前换新水**：不要反复烧一壶水到见底再加水（会让矿物质浓缩在底部）
- **每次用完倒掉残水**：长期存水加速底部沉积 + 滋生水垢-生物膜复合层
- **不要“半壶水频繁烧”**：水位刻度都标了下限上限，半壶以下烧水会让发热盘暴露在边缘水线，更容易结垢
- **按硬度调整除垢频率**：
  - 硬水地区（北京、天津等）：**每月 1 次**
  - 中等硬水（上海、武汉等）：**2-3 个月 1 次**
  - 软水地区（杭州、广州、重庆等）：**半年 1 次**
  - 用纯净水 / 滤水壶过的水：可能 1-2 年才需要除一次

## 4.2 怎么除水垢（按推荐顺序）

**方法 1：柠檬酸（最推荐）**

食品级柠檬酸超市 / 网购都有，几块钱一袋够用半年。

1. 倒水到壶里到 80% 满
2. 加 30-50 g 柠檬酸（一大勺到一小袋的量，看水垢厚度）
3. **烧开**
4. 关电源后**浸泡 30 分钟到 2 小时**（厚水垢需要更久）
5. 倒掉，用清水冲 2-3 次，再烧一壶水倒掉

化学反应：

> 2 C₆H₈O₇ + 3 CaCO₃ → Ca₃(C₆H₅O₇)₂ + 3 CO₂↑ + 3 H₂O

柠檬酸钙可溶且无害（食品添加剂常见成分）。

优点：**没气味、不刺激、适合金属壶 / 玻璃壶 / 塑料件**、效果彻底
缺点：要专门买（但很便宜）

**方法 2：白醋（家里就有）**

白醋（5-9% 醋酸）兑水 1:2 或 1:1，烧开浸泡 30 分钟，效果好但有酸味。烧完冲洗 3-4 次去味，再煮一壶白水倒掉。

化学反应：

> 2 CH₃COOH + CaCO₃ → Ca(CH₃COO)₂ + CO₂↑ + H₂O

**方法 3：可乐 / 雪碧**

含磷酸 + 柠檬酸，效果可以但比柠檬酸贵且糖分会让壶后续有黏腻感。**应急用**，不推荐常用。

**方法 4：除垢专用产品**

商超有“水垢清洁剂”（主要成分 = 柠檬酸 / 氨基磺酸 / 草酸）。买的话挑 **柠檬酸基** 的最稳；含 **氨基磺酸** 的更强但要按说明严格冲洗；**含草酸** 的不推荐家用（草酸有毒，残留风险高）。

## 4.3 千万不要做的事

- ❌ **钢丝球 / 砂纸刮**：会划伤壶内壁涂层，让以后结垢更快、可能溶出金属
- ❌ **小苏打 / 苏打粉**：碱性物质对碳酸盐**无效**——很多视频教程错的，碳酸钙 + 碳酸氢钠 = 没反应。小苏打只对油污 / 中和酸有效
- ❌ **盐酸 / 工业除垢剂**：腐蚀性太强、残留风险高、塑料配件可能受损；**饮水器具不要用**
- ❌ **洗洁精 / 洗涤剂**：表面活性剂对水垢无效；只能洗外壳
- ❌ **冷水泡白醋几小时不加热**：醋酸冷态下反应极慢；**必须加热**才有用

## 4.4 怎么减少结垢

**最有效**：

- **滤水壶**（Brita、3M、海尔等）：用离子交换树脂把 Ca²⁺ / Mg²⁺ 替换成 Na⁺ → 软水。**真正显著减少结垢**，但水里钠会增加（高血压人群少喝软化水）
- **反渗透（RO）净水器**：滤掉所有矿物质，水变成“纯净水”。彻底无垢，但喝起来“空”，且 RO 出水带钠风险低
- **直饮纯净水 / 蒸馏水**：完全不结垢

**作用有限**：

- “磁化水”、“量子能量水”：**几乎都是伪科学**，没有可靠证据显示磁场能改变水的硬度
- “电解水”：复杂，且改变 pH 而非硬度

## 4.5 几个常见误区

**误区 1：「水垢喝下去会得肾结石」**

错。膳食钙摄入和肾结石**反相关**。结石主因是水喝少 + 草酸 / 钠 / 蛋白过量。

**误区 2：「千滚水有亚硝酸盐 / 致癌」**

夸大。反复烧水亚硝酸盐确实会**轻微升高**（来自原水中硝酸盐还原），但即使烧 20 次，含量也远低于一根新鲜蔬菜里的亚硝酸盐。**不需要为此焦虑，但确实建议每次换新水**——理由不是健康，是水垢浓度问题（见 4.1）。

**误区 3：「水垢能补钙，多喝水垢水」**

也错。水垢提供的钙占膳食钙的比例很小（一杯硬水 ≈ 75 mg 钙，相比一杯牛奶 250 mg、一片钙片 500 mg）。能补但有限，不是“补钙神水”。

**误区 4：「磁化水能除垢」**

证据非常薄弱。家用磁化装置卖了几十年，独立科学测试基本都显示没有显著效果。
