# 1. 这篇文章给谁看

写给**刚到美国一两个月、对各种“证件”有点晕**的留学生 / 学者。常见的几个困惑——

- **SSN、护照、驾照、I-20——这些到底分别是什么？哪个是“身份证”，哪个是“身份”？**
- **店员说 “Show me your ID”，我应该出示哪一个？护照？驾照？学生卡？**
- **登机要带护照还是只要驾照？银行开户要 I-20 吗？租车要不要签证页？**
- **SSN 卡和 I-20 平时放钱包里安不安全？**

这篇文章不讲怎么申请这些证件（散在我之前几篇文章里：[签证](/life/us-visa-types) / [驾照](/life/pa-drivers-license)），而是把它们**摊开放在桌上一次性讲清楚**——每张证件**是什么 / 证明什么 / 什么场景被认为是 ID / 平时该放哪里**。文章核心想解决的是“什么是 ID、什么不是 ID”这个看似简单但留学生反复栽跟头的问题。

# 2. 结论先行

**美国的“身份”由三类东西构成，分工明确：**

| 类别 | 举例 | 在美国官方眼里它代表什么 |
|---|---|---|
| **SSN（数字）** | 9 位社安号 | “我在美国税务和信用系统里的唯一标识”——**不是 ID**，不能“出示” |
| **Photo ID** | 驾照 / 美国 ID / 护照 / EAD | “我是站在你面前的这个人”——**这才是被 “show ID” 的对象** |
| **签证身份文件** | I-20 / DS-2019 / 签证页 / I-94 | “我合法在美国”——**通常不算 ID**，但移民局 / 海关认 |

**店员说 “Show me your ID” 时，意思是 photo ID** —— 你递护照或驾照都行，**不要递 I-20**（递了店员也认不出来）。

**反过来：** 移民局 / 海关 / 雇主 I-9 / 学校 ISSO 检查时——**只看护照 + I-20 + 签证页 + I-94**，光出示驾照不够。

这两套逻辑互不替代，**钱包里要常备 photo ID，保险柜里要好好保管签证文件**。

## 3.1 SSN：数字而非证件

**SSN（Social Security Number，社会安全号）是一个 9 位数字**——核心载体是数字本身，不是那张纸卡。

- **领到的“卡”**是一张纸（不是塑料），上面就是数字 + 你的名字 + “Not for Identification” 字样
- **正式场合很少 “show” 这张卡**——通常是在网站 / 表格里**填写数字**
- **平时不要带在钱包里**——丢了被盗用 = 信用记录 / 报税身份被盗
- 给雇主 / 银行 / 租房时，**口头或填表给数字即可**，对方一般不要求看卡

**留学生何时拿 SSN？** F-1 拿到 on-campus job offer / OPT 期间 → 去 SSA 办公室申请，2-4 周到手。F-2 / J-2 不工作的没有 SSN，办驾照时需要 SSA 出具的 “**Ineligibility Letter**”（拒绝信）替代。

## 3.2 Photo ID：日常生活的“硬通货”

**Photo ID = 带照片 + 政府发 + 全美互认**的身份卡。这是“Show ID”时几乎唯一的合理回答。

**全美硬通货级**（任何场景都认）：

| Photo ID | 发证机构 | 留学生能否拿 |
|---|---|---|
| **美国州驾照**（含 REAL ID 版） | 各州 DMV / DOT | ✅ 详见 [驾照指南](/life/pa-drivers-license) |
| **美国 State ID Card** | 各州 DMV | ✅ 不会开车 / 不想开车的可以办 |
| **美国护照 / 护照卡** | U.S. Department of State | ❌ 仅美国公民 |
| **EAD 卡**（Employment Authorization Document）| USCIS | ✅ OPT / STEM OPT / J-2 EAD / H-4 EAD |
| **军人 ID / 联邦雇员 ID** | 各联邦机构 | ❌ 几乎不适用 |

**部分场景算 ID**（要看具体情况）：

- **外国护照（中国护照）**：在大多数场景被接受，但**有些州的酒类店明确只认 state-issued ID**——所以最稳的还是带美国驾照
- **Student ID Card**（学校发的）：进图书馆 / 实验室 / 校园活动算；**买酒 / 进酒吧 / 银行开户 / 飞机登机不算**
- **Work badge**：进公司大楼算；其他场景一般不算

## 3.3 签证身份文件：证明你合法在美

这一类**不是 ID**，但移民局、海关、学校国际学生办、I-9 雇主合规检查会要：

| 文件 | 形态 | 谁发 | 证明什么 |
|---|---|---|---|
| **I-20** | 纸（多页 PDF 打印件） | 你录取的学校（DSO 签字） | F-1 / F-2 学生身份 + SEVIS 状态 |
| **DS-2019** | 纸（一页 PDF 打印件） | J-1 sponsor | J-1 / J-2 学者 / 学生身份 |
| **签证 stamp** | 贴在护照页上的彩色签证 | 美国驻外大使馆 / 领事馆 | 你**入境**的资格（不证明你在美状态）|
| **I-94** | 电子记录（[i94.cbp.dhs.gov](https://i94.cbp.dhs.gov/) 可查打印）| 美国 CBP | 你的最近一次入境时间 + 允许停留时间 |
| **I-797** | 纸（USCIS 批准信）| USCIS | H-1B 批准、I-20 续签批准、绿卡阶段批准等 |
| **EAD 卡** | 塑料卡 + 照片 | USCIS | 工作授权（**这是唯一一个同时算 Photo ID 的签证文件**）|

**关键认识**：签证文件**不是 ID**——

- I-20 / DS-2019 没有照片
- 签证 stamp 是贴在护照里，但只证明“入境资格”，不证明“目前合法”
- 没有任何人会拿着 I-20 去酒吧买酒——它**不被酒类店、TSA、银行（作为主 ID）认可**

# 4. “Show your ID” 实际指什么

## 4.1 默认含义：政府发的 photo ID

当美国人说 “show me your ID”、“can I see your ID”、“ID please”，**默认指**——

> 政府发的 + 带你照片 + 当下未过期 + 上面有出生日期 + 你的全名

**最稳妥的递法**：递驾照（如果有）。**第二稳妥**：递护照。**几乎从不递**：I-20、SSN 卡、Student ID（除了校内）。

## 4.2 不同场景的“ID 标准”差异

不是所有“ID”都一样严。同一张卡在不同场景被接受度不同：

| 场景 | 接受标准的严格度 | 最稳妥的 ID |
|---|---|---|
| 校内活动 / 图书馆 | 宽松 | Student ID |
| 普通商店 / 餐厅 | 中等 | 驾照 / 护照 |
| 酒类销售 / 进酒吧 | **严**——必须**州发的 photo ID** + 当下未过期 + 容易看到出生日期 | 美国驾照（部分酒吧拒收外国护照）|
| 银行开户 / 大额取现 | **严**——通常要 1 primary（驾照 / 护照）+ 1 secondary（信用卡 / SSN 卡 / 工作 ID） | 驾照 + 护照 |
| 租车 | **严**——必须 driver license（同时证明驾驶资格） | 美国驾照 |
| 国内航班 TSA | **极严**——必须 [TSA 接受清单](https://www.tsa.gov/travel/security-screening/identification) 上的 | REAL ID 驾照 / 护照 / EAD |
| 国际航班登机 | 必须护照（出 / 入境都要） | 护照 |
| 警察查车 | 必须驾照 | 驾照 |
| I-9（雇主入职合规） | 按 USCIS Form I-9 List A / B+C 来 | 详见 §5.9 |

## 4.3 联邦认可 vs 州认可

少数场景区分这两个层级：

- **联邦认可的 ID**：进入军事基地、某些联邦法院、TSA 国内航班——必须 **REAL ID + 金星标 / 护照 / 联邦颁发卡**
- **州认可的 ID**：买酒、进酒吧、租车、银行开户——州颁发的 photo ID（带不带星都行）

**REAL ID Act 自 2025-05-07 起强制执行**意味着：登国内航班，**普通驾照不够，要么 REAL ID 要么护照**。这是过去一年最大的政策变动，[驾照那篇](/life/pa-drivers-license)的 §5.2 详细讲了。

# 5. 主要场景到底要什么

下面这张大表是这篇文章的“工具书部分”——遇到具体场景直接查。

| 场景 | 必带 | 推荐多带 | 不需要 |
|---|---|---|---|
| **国内航班登机（TSA）** | REAL ID 驾照 / 护照 / 护照卡 / **EAD 卡** | — | I-20、签证页、SSN |
| **国际航班登机（出美国）** | 护照 + 美签（视目的地） | I-20（部分中转地查）| 驾照、SSN |
| **国际航班入境美国** | 护照 + 美签 + I-20 / DS-2019 | EAD（如有）| 驾照 |
| **买酒 / 进酒吧（21+）** | Photo ID（驾照最稳）| 护照（酒吧可能不熟悉）| I-20、SSN、Student ID |
| **银行开户** | Primary photo ID（驾照 / 护照）+ Secondary（信用卡 / SSN 卡 / 工作 ID）+ 地址证明 | I-20 偶尔被要 | — |
| **银行大额现金交易（≥ $10K）** | Photo ID + SSN | — | — |
| **租车** | **美国驾照**（中国驾照短期可，租车公司有自己门槛）+ 信用卡 | 护照 | I-20 |
| **租房 / 看公寓** | Photo ID + 收入证明 + 信用记录 + SSN | I-20（房东偶尔要看，证明在美合法） | — |
| **入住酒店 / Airbnb** | Photo ID + 信用卡 | — | I-20 |
| **警察查车（Traffic stop）** | 美国驾照 + 保险卡 + 车辆注册卡 | — | 护照、I-20 |
| **入职 I-9（雇主合规）** | List A 或 (List B + C)：详见 §5.1 | — | — |
| **校园活动 / 实验室门禁 / 校内餐厅** | Student ID | — | — |
| **学校国际学生办（ISSO）** | Student ID + I-20 / DS-2019 + 护照 | I-94 打印件 | — |
| **跨州自驾穿越** | 驾照 + 保险 + 注册 | — | 护照 |
| **邮局取邮件 / 包裹** | Photo ID（驾照 / 护照都行）| — | — |
| **签合同**（手机 / 健身房 / 网络）| Photo ID | SSN（部分服务要查信用） | — |
| **大学考试 / 标准化考试**（GRE / TOEFL） | 护照（最权威）| 驾照（部分接受）| Student ID（不接受）|
| **办理 SSN / IRS 报税** | 护照 + I-20 / DS-2019 + I-94 | — | 驾照（不必但可以辅助）|
| **新冠疫苗 / 处方药取药** | Photo ID | 保险卡 | — |

## 5.1 I-9 雇主入职合规（这一栏特别复杂）

入职新工作时（包括 OPT / STEM OPT / H-1B），HR 会让你填 [Form I-9](https://www.uscis.gov/i-9)，并要求出示文件证明“你能在美国合法工作”。规则是：

**List A（一份证明身份 + 工作授权）：**
- 美国护照 / 护照卡
- 永久绿卡（I-551）
- **外国护照 + I-94 + EAD 卡组合**
- **外国护照 + I-94 + I-20**（F-1 学生）
- **外国护照 + I-94 + DS-2019**（J-1）

**List B（证明身份）+ List C（证明工作授权）：**
- List B：驾照 / State ID / Student ID + 学校文件 / 等等
- List C：SSN 卡（必须不带 work restriction）/ 出生证（仅美籍）/ EAD（部分类型）

**留学生 OPT / STEM OPT 通常用 List A 组合**：外国护照 + I-94 + EAD 卡，一次性涵盖。

**关键：HR 可以让你选择出示哪个组合，但不能指定具体某张**。如果 HR 说“必须给驾照”——他们违法了，可以拒绝。

# 6. 留学生几个常见混淆

## 6.1 I-20 不是 ID，但平时还是要带着

**I-20 不是 ID** —— 你不能拿它去买酒、租车、登机。但它确实是你“合法在美国”的核心文件，所以：

- **平时**：放在**保险柜 / 抽屉里**，不要随身带（怕丢）
- **国际旅行**：**必须带**（出 / 入美国海关查）
- **入职 I-9**：必须带（雇主复印）
- **学校 ISSO 签到**：必须带
- **回中国办美国签证 renewal**：必须带

## 6.2 中国护照在美国到底能不能算 ID？

**绝大多数场景认**：银行开户、酒店入住、签合同、邮局取件、租房、校园活动——护照（任何国家的）都被接受。

**少数场景不认 / 不太认**：

- **某些州的酒类店**：店员习惯看驾照，看到外国护照可能不会读出生日期，导致“看不清”——技术上拒绝你。换张美国驾照立刻解决
- **TSA 登国内航班**：**接受外国护照**——但每次都要查得详细，过 TSA 队伍慢一些
- **租车公司**：理论上接受外国护照（你的证件证明身份），但**驾驶资格**还要看外国驾照 + 翻译件 / IDP，这一段中国护照不够用

**实战建议**：来美国第一年带护照走天下没问题，但**拿到美国驾照之后，钱包里日常带驾照不带护照**——驾照丢失 / 损耗代价远低于护照。

## 6.3 EAD 是个例外（同时算签证文件 + Photo ID）

**EAD 卡**（Employment Authorization Document）是 OPT / STEM OPT / J-2 / H-4 配偶等工作授权的塑料卡：

- ✅ 是签证身份文件（证明你能合法工作）
- ✅ **同时是 photo ID**（带照片，USCIS 发，TSA 接受）
- ✅ I-9 时可以单独作为 List A（不需要再叠加护照）

**但 EAD 卡价值高（重新申请要 90+ 天 + $410）**，所以：
- 入职 / TSA 登机时**带着**
- 平时**放保险柜**，不要每天揣钱包里

## 6.4 SSN 卡 ≠ SSN

**SSN 是数字。SSN 卡只是这个数字的第一次“凭证”。**

- 银行开户、入职等场合：**填数字 / 输入数字** 就够了，**不一定要看实体卡**（但有些场合要看）
- 实体 SSN 卡**不应该常带在钱包里**——丢了被盗用 = 信用 / 报税身份被盗，恢复起来非常麻烦
- 第一次办 SSN 给雇主复印或 HR 备份后，**就锁进抽屉 / 保险柜**

# 7. 实践建议：钱包 vs 保险柜分别放什么

```
┌─ 钱包（每天带 ）──────┐    ┌─ 保险柜 / 抽屉（不带）──┐
│                        │    │                          │
│  🟢 美国驾照 / State ID │    │  🔴 SSN 卡（最高优先）  │
│  🟢 信用卡 / 借记卡     │    │  🔴 美国护照（出门借出） │
│  🟢 健康保险卡          │    │  🔴 I-20 / DS-2019 原件 │
│  🟢 车辆保险卡（车上）  │    │  🔴 EAD 卡（除非当天用）│
│  🟢 Student ID          │    │  🔴 I-797 / I-94 备份   │
│  🟢 一点现金 + 信用卡   │    │  🔴 出生证 / 学位证     │
│                        │    │                          │
└────────────────────────┘    └──────────────────────────┘
```

**几条实战原则**：

- **不要把 SSN 卡放钱包**：SSN 是被盗用代价最高的证件
- **护照仅出国 / 必要时携带**：丢一次护照恢复 6-12 周 + $130 + 麻烦的紧急 visa（如果在国外）
- **I-20 复印件可以放钱包**（备用）；**原件锁起来**
- **驾照才是日常 ID 主力**：办了之后 95% 场景用驾照即可
- **手机里存所有证件 PDF**：iPhone 钱包 / Files app 里加密存：护照 + 签证 + I-20 + I-94 + EAD + 驾照 + 健康保险 + 车辆保险 + 学位证。**手机不算法律意义上的 ID，但万一丢实体卡，PDF 是补办时的救命数据**
- **每次国际入境后立刻查 [i94.cbp.dhs.gov](https://i94.cbp.dhs.gov/)**：海关偶尔录入错误，30 天内可以申诉
